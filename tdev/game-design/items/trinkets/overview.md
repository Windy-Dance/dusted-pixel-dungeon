# 饰品概述

饰品是提供各种被动效果的独特物品。它们可以在炼金锅中使用能量升级 (最高等级3)。与戒指和神器不同，饰品没有诅咒版本。

## 详细饰品机制

### 十三叶三叶草
- **效果**: 改变伤害掷骰以偏向极端 (最大或最小)
- **激活几率**: 25% + 25%*等级
- **最大/最小分割**: 60%几率造成最大伤害，40%几率造成最小伤害
- **+0时**: 25%激活率
- **+3时**: 100%激活率
- **策略**: 高风险/高回报 - 可能大幅增加或减少伤害

### 蟾蜍眼
- **效果**: 减少视野范围但授予心灵视觉
- **视野减少**: 12.5% + 12.5%*等级 (+3时: 50%减少)
- **心灵视觉范围**: 2 + 等级格
- **+0时**: -12.5%视野，2格心灵视觉
- **+3时**: -50%视野，5格心灵视觉
- **协同**: 在范围内透视墙壁显示敌人

### 异域水晶
- **效果**: 消耗品有几率成为异域版本
- **异域几率**: 12.5% + 12.5%*等级
- **+0时**: 12.5%几率
- **+3时**: 50%几率
- **影响**: 掉落和商店中的药剂、卷轴
- **财富协同**: 增加高价值掉落中的异域几率

### 羊皮纸碎片
- **效果**: 修改附魔和诅咒几率
- **附魔乘数**: 0/1/2/3级时分别为2/4/7/10x
- **诅咒乘数**: 0/1/2/3级时分别为1.5x/2x/1x/0x
- **+2时**: 7x附魔几率，正常诅咒几率
- **+3时**: 10x附魔几率，无诅咒
- **策略**: 确保好附魔而无诅咒的最佳选择

### 碎裂遗忘
- **效果**: 手动物品识别和战利品加成
- **动作**: `识别` - 识别一个就绪物品
- **战利品加成**: 每件未识别装备+20% (最多等级+1件物品)
- **被动识别禁用**: 物品不会从经验自动识别
- **追踪器**: 就绪识别法杖/投掷武器时显示增益
- **+3时**: 最多可计算4件物品获得战利品加成

### 苔藓团块
- **效果**: 增加草地/水域楼层感应的几率
- **覆盖几率**: 25% + 25%*等级
- **楼层分布**: 每个循环2个草地，4个水域感应
- **+0时**: 25%覆盖几率
- **+3时**: 100%覆盖几率
- **策略**: 一致的有益地形生成

### 雪貂毛簇
- **效果**: 增加闪避
- **闪避乘数**: 1 + 12.5%*(等级+1)
- **+0时**: +12.5%闪避
- **+3时**: +50%闪避
- **简单**: 纯闪避提升

### 陷阱机制
- **效果**: 增加陷阱/深渊楼层并揭示隐藏陷阱
- **覆盖几率**: 25% + 25%*等级
- **揭示几率**: 10% + 10%*等级用于隐藏陷阱
- **楼层分布**: 每个循环3个陷阱，3个深渊感觉
- **+3时**: 100%覆盖，40%陷阱揭示
- **协同**: 与陷阱专注构建配合使用

### 维度日晷
- **效果**: 根据现实世界时间修改敌人生成
- **白天 (8:00-20:00)**: 5% + 5%*等级更少生成
- **夜晚 (20:00-8:00)**: 25% + 25%*等级更多生成
- **+0白天**: -5%生成
- **+3白天**: -20%生成
- **+0夜晚**: +25%生成
- **+3夜晚**: +100%生成
- **警告**: 夜晚开始时提醒玩家

### 混沌香炉
- **效果**: 周期性在目标敌人附近生成气体云
- **平均回合**: 300/(等级+1)
- **+0时**: 每~300回合
- **+3时**: 每~75回合
- **气体类型**:
  - 常见 (40-70%): 毒气、混乱、再生
  - 不常见 (25-40%): 风暴云、烟雾屏幕、恶臭
  - 稀有 (5-20%): 地狱、暴风雪、腐蚀气体
- **目标**: 优先选择目标敌人附近的格子

## 饰品机制

### 升级
- 饰品可以在炼金锅中升级
- 最高等级为3
- 成本随等级和饰品类别缩放
- 两种成本等级: 6/8/10/12 或 10/15/20

### 饰品催化剂
- 生成随机饰品的特殊物品
- 在地下城中找到或购买

### 独特性
- 饰品是独特物品
- 在一次运行中不能重复找到
- 等级在运行中保持

### 等级加成
- 效果随饰品等级 (0-3) 缩放
- 等级-1表示未拥有饰品
- 强化等级考虑任何增强效果
