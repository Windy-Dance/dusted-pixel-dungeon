# 武器附魔

## 概述

附魔是可以应用到武器上的魔法效果。它们提供额外的战斗效果，在击中时触发。附魔可以是有益的（常规附魔）或有害的（诅咒）。

### 触发几率公式

大多数附魔使用一个递增的触发几率：

```java
// 常见公式（烈焰、寒霜、繁茂）
procChance = (level + 1) / (level + 3)

// 等级示例：
// 0级：33%
// 1级：50%
// 2级：60%
// 5级：75%
```

### 触发几率倍数

某些天赋和增益可以增加触发率：
- 基础倍数：1.0倍
- 可以超过1.0倍以获得增强效果

## 火焰附魔

### 烈焰
| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 橙色（0xFF4400） |
| 触发几率 | (lvl+1)/(lvl+3) |
| 效果 | 火焰伤害 + 燃烧 |

**效果详情：**
- 如果目标未燃烧：施加8回合燃烧
- 额外伤害：1到3 + 深度/4
- 伤害倍数：每点触发强度0.67倍
- 视觉效果：火焰粒子

| 等级 | 触发几率 | 备注 |
|-------|-------------|-------|
| 0 | 33% | 基础火焰应用 |
| 1 | 50% | 可靠的燃烧 |
| 2 | 60% | 持续伤害 |
| 5 | 75% | 非常频繁 |

**协同效果：**
- 对火焰免疫敌人有效（仅造成伤害）
- 燃烧会蔓延到相邻的草丛
- 与火焰相关天赋结合

|-------|-------------|----------|

| 1 | 40% | 3-6回合 |
| 2 | 50% | 3-6回合 |

**协同效果：**
- 寒冷可以与其他减速效果叠加
- 冻结的敌人受到额外伤害
- 与冰霜相关天赋结合

## 闪电附魔

### 电击
| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 白色（0xFFFFFF，50%透明度） |
| 触发几率 | 33%（固定） |
| 效果 | 连锁闪电 |

**效果详情：**
- 在2格范围内向附近敌人释放电流
- 连锁伤害：原始伤害的50%
- 水会延长电流距离
- 视觉效果：闪电效果

**连锁机制：**
- 从被击中的目标开始
- 连锁到2格半径内
- 水格子增加范围
- 不对原始目标造成伤害
- 可以击中多个敌人

| 情况 | 电流距离 |
|-----------|--------------|
| 正常 | 2格 |
| 目标在水中 | 2格（在水中电流更远） |

## 治疗附魔

### 血饮
| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 深红色（0x660022） |
| 触发几率 | 5-30%（基于缺失生命值） |
| 效果 | 击中时治疗 |

**效果详情：**
- 治疗量：造成伤害的50%
- 触发几率随缺失生命值比例递增
- 对友方或中立单位无效（除了拟态怪）
- 视觉效果：绿色治疗文字

**触发几率公式：**
```java
healChance = 0.05 + 0.25 * (missingHP / maxHP)
```

| 缺失生命值比例 | 触发几率 |
|--------------|-------------|
| 0% | 5% |
| 25% | 11% |
| 50% | 18% |
| 75% | 24% |
| 100% | 30% |

**协同效果：**
- 适合持续战斗
- 与高伤害武器配合良好
- 从增加攻击速度中受益

## 动量附魔

### 恒动
| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黄色（0xFFFF00） |
| 触发几率 | 总是 |
| 效果 | 保存溢出伤害 |

**效果详情：**
- 当敌人死亡时储存"浪费"的伤害
- 储存的伤害应用于下一次攻击
- 每回合衰减2.5%
- 视觉效果：带有伤害数字的武器增益图标

**机制：**
1. 对敌人造成的伤害超过其剩余生命值
2. 超出的伤害被储存
3. 下次攻击添加储存的伤害
4. 储存的伤害随时间衰减

**衰减率：**
- 最小衰减：每回合0.1点伤害
- 百分比衰减：每回合2.5%

## 即死附魔

### 死神
| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 0-55%（基于敌人生命值） |
| 效果 | 即死 |

**效果详情：**
- 几率随敌人生命值降低而增加
- 基础几率：基于敌人生命值百分比的0-50%
- 等级奖励：每级+5%
- 对首领或免疫敌人无效

**触发几率公式：**
```java
maxChance = 0.5 + 0.05 * level
actualChance = maxChance * (1 - enemyHPPercent)
```

| 等级 | 最大几率（在1点生命值时） |
|-------|---------------------|
| 0 | 50% |
| 1 | 55% |
| 2 | 60% |
| 5 | 75% |

**免疫：**
- 首领敌人
- 具有死神免疫的敌人
- 特定特殊敌人

## 附魔稀有度

当武器生成附魔时：
- 10%几率获得有益附魔
- 30%几率获得诅咒（如果没有有益附魔）
- 附魔卷轴保证获得有益附魔
- 诅咒注入转化为诅咒

## 硬化

移除附魔后武器可以硬化：
- 移除附魔但保留升级等级
- 无法获得新附魔
- 无特殊发光效果
- 保留净化后的诅咒免疫
