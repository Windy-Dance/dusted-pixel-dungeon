# 武器诅咒

## 概述

诅咒是有害的附魔，装备时会产生负面效果。与常规附魔不同，诅咒对玩家有害，可能使战斗变得更加困难或危险。

### 诅咒属性

- 所有诅咒都有黑色发光（0x000000）
- 被诅咒的武器在移除诅咒前无法卸下
- 诅咒效果与正常武器伤害同时触发
- 某些诅咒在特定情况下可能有用

### 诅咒获取

| 来源 | 几率 |
|--------|--------|
| 随机武器生成 | 30% |
| 诅咒注入 | 100% |
| 诅咒升级 | 可变 |

---

## 喧闹

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 5%（1/20） |
| 效果 | 警告所有敌人 |

**效果详情：**
- 向楼层上所有敌人警告你的位置
- 播放响亮的拟态怪声音
- 显示烦人的消息
- 打破隐形效果

**消息池：**
- 常见消息（1-10）：一般嘲讽
- 稀有消息（11-13）：每条约3%几率

**影响：**
- 战斗中效果轻微
- 潜行时效果严重
- 可以唤醒睡觉的敌人
- 破坏伏击机会

**特定情况用途：**
- 可用于聚集敌人进行区域攻击
- 在敌人伏击你之前警告他们

## 炫目

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 10%（1/10） |
| 效果 | 致盲附近角色 |

**效果详情：**
- 使攻击者完全致盲
- 使其他观察者致盲一半持续时间
- 对所有人都可见的闪光效果
- 爆炸音效

**致盲效果：**
- 攻击者致盲：完整致盲持续时间
- 视野内其他人：一半持续时间
- 玩家可见屏幕闪光

**影响：**
- 自我致盲很危险
- 也可以致盲相邻的敌人
- 破坏潜行潜力
- 使后续攻击更加困难

**特定情况用途：**
- 可以致盲攻击你的敌人
- 在拥挤的战斗中有用
- 致盲自己，降低准确度

## 转移

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 8.3%（1/12） |
| 效果 | 传送敌人 |

**效果详情：**
- 将敌人传送到随机位置
- 对不可移动的敌人无效
- 创建光线粒子效果
- 敌人切换到游荡状态

**传送机制：**
- 使用传送卷轴逻辑
- 对大多数普通敌人有效
- 排除：首领、静态敌人
- 将敌人AI状态更改为游荡

**影响：**
- 将敌人从即时战斗中移除
- 敌人可能传送到更糟的位置
- 可以分散成群的敌人
- 敌人可能会稍后返回

**特定情况用途：**
- 从危险的近战范围逃脱
- 打破敌人阵型
- 延迟强力敌人
- 暂时移除威胁

## 易爆

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色/温暖/炽热（变化） |
| 触发几率 | 随时间累积 |
| 效果 | 约20次攻击后爆炸 |

**效果详情：**
- 每次攻击都会累积热量
- 50%耐久度时警告："温暖"
- 10%耐久度时警告："炽热"
- 0%耐久度时爆炸
- 爆炸会损坏武器

**热量阶段：**

| 耐久度 | 发光 | 警告 |
|------------|------|---------|
| 51-100 | 黑色 | 无 |
| 11-50 | 温暖（暗淡） | "温暖" |
| 1-10 | 炽热（非常暗淡） | "炽热！" |
| 0 | 爆炸 | - |

**爆炸机制：**
- 爆炸位置：目标最近的相邻格子
- 使用炸弹.召唤炸弹伤害
- 显著损坏投掷武器
- 爆炸后重置热量

**平均爆炸攻击次数：**
- 20次攻击（平均）
- 随机耐久度减少：每次攻击0-10点

**影响：**
- 可能伤害你和附近的敌人
- 破坏附近的地形
- 可能破坏被诅咒的武器
- 警告系统提供准备时间

**特定情况用途：**
- 故意造成爆炸伤害
- 被包围时的区域伤害
- 破坏墙壁或障碍物
- 高风险高回报的游戏风格

## 友善

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 10%（1/10） |
| 效果 | 迷惑攻击者和目标 |

**效果详情：**
- 同时迷惑攻击者和防御者
- 攻击者迷惑：完整持续时间
- 防御者迷惑：一半持续时间
- 心形粒子效果

**迷惑机制：**
- 被迷惑的角色不能互相攻击
- 攻击者被防御者迷惑
- 防御者被攻击者迷惑（不能反击）
- 迷惑忽略来自迷惑源的下一次攻击

**特殊互动：**
- 如果已被目标迷惑：造成0伤害
- 迷惑效果对任何敌人都有效
- 可以创造临时休战

**影响：**
- 阻止你攻击那个敌人
- 敌人也不能攻击你
- 浪费战斗中的回合
- 可能中断重要战斗

**特定情况用途：**
- 在艰难战斗中的紧急和平
- 阻止危险的敌人攻击
- 延迟敌人攻击性
- 创造逃脱机会

## 极化

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 50% |
| 效果 | 全有或全无伤害 |

**效果详情：**
- 50%几率：150%伤害
- 50%几率：0伤害（完全未命中）
- 触发时无视觉效果
- 二元结果

**伤害计算：**
```java
if (random < 0.5) {
    damage = round(1.5 * damage);  // 强化命中
} else {
    damage = 0;  // 完全未命中
}
```

**影响：**
- 更高的平均伤害（正常值的75%）
- 伤害极不稳定
- 可能错过关键击杀
- 可能意外造成额外伤害

**数学分析：**
- 期望伤害：0.5 * 1.5 = 0.75倍正常值
- 实际上是净伤害损失
- 但方差非常高

**特定情况用途：**
- 当任何伤害都可以接受时
- 与保证伤害来源结合
- 高风险爆发伤害
- 不推荐用于可靠战斗

---

## 血祭

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 10%（1/10） |
| 效果 | 使自己流血 |

**效果详情：**
- 对攻击者造成流血
- 流血几率随当前生命值百分比递增
- 生命值越高 = 流血几率越高
- 流血伤害基于最大生命值

**流血机制：**
```java
missingPercent = HP / HT
bleedChance = pow(missingPercent, 2) * HT / 8
```

**流血量计算：**
- 基于当前生命值百分比
- 平方值使其在高生命值时更可能发生
- 除以8进行缩放

**影响：**
- 随时间自我伤害
- 如果不小心可能会杀死自己
- 浪费治疗资源
- 在持久战中危险

**流血行为：**
- 最小流血：每回合1点伤害
- 可以与其他流血叠加
- 通过治疗或绷带治愈
- 随时间自然衰减

**特定情况用途：**
- 几乎完全是负面的
- 有用的用途极少
- 尽可能避免使用
- 尽快移除诅咒

## 妄为

| 属性 | 数值 |
|----------|-------|
| 发光颜色 | 黑色（0x000000） |
| 触发几率 | 25%（1/4） |
| 效果 | 降低准确度 |

**效果详情：**
- 施加妄为增益10回合
- 妄为增益可通过击中移除
- 如果当前未激活，在未命中时重新应用
- 激活期间准确度减益

**增益机制：**
- 持续时间：10回合
- 类型：负面（可被净化）
- 成功击中时移除增益
- 再次触发时重新应用

**准确度惩罚：**
- 在weapon.accuracyFactor()中应用
- 显著降低命中几率
- 与其他准确度惩罚叠加
- 使战斗不可靠

**影响：**
- 经常未命中攻击
- 浪费战斗中的回合
- 如果继续未命中可能会连锁
- 使武器不可靠

**行为：**
- 未命中时应用增益
- 击中时移除增益
- 增益激活时未命中：不重新应用
- 创造未命中连击

**特定情况用途：**
- 总体上非常负面
- 可使用增益移除来净化
- 高准确度构建减轻效果
- 对可靠战斗来说是最糟糕的诅咒

## 诅咒移除

### 移除诅咒的方法

| 方法 | 效果 |
|--------|--------|
| 移除诅咒卷轴 | 移除诅咒，保留武器 |
| 升级卷轴 | 小几率移除诅咒 |
| 诅咒注入 | 转换为诅咒形式 |
| 变异之井 | 可以随机化附魔 |

### 诅咒注入特殊属性

当通过诅咒注入应用诅咒时：
- 武器变为被诅咒
- 保留升级等级
- 稍后可以净化
- 诅咒效果与自然诅咒相同

---

## 被诅咒的武器

### 被诅咒状态

当装备被诅咒的武器时：
- 无法在移除诅咒前卸下
- 诅咒效果始终激活
- 武器在其他方面正常运作
- 仍可升级

### 识别诅咒

| 状态 | 可见 |
|-------|---------|
| 未知 | 无指示 |
| 已知被诅咒 | 显示诅咒状态 |
| 装备被诅咒 | 无法移除，显示诅咒 |

---

## 诅咒互动

### 多重诅咒

- 武器只能有一个诅咒
- 诅咒注入替换现有诅咒
- 无法叠加诅咒效果

### 诅咒 + 附魔

- 诅咒是一种附魔
- 无法同时拥有有益附魔和诅咒
- 紊乱偶尔可以触发诅咒

### 触发顺序

1. 正常武器伤害
2. 附魔/诅咒触发
3. 特殊武器效果

---

## 战略考量

### 最不有害的诅咒

1. **转移** - 实际上可能有帮助
2. **喧闹** - 轻微不便
3. **炫目** - 特定情况下有用

### 最有害的诅咒

1. **妄为** - 使武器不可靠
2. **血祭** - 自我伤害
3. **易爆** - 不可预测的伤害

### 使用诅咒游戏

- 转移可以战略性地使用
- 易爆可用于故意爆炸
- 友善可以创造临时休战
- 极化适用于高伤害构建

### 诅咒避免

- 装备前识别武器
- 早期使用移除诅咒卷轴
- 保持备用武器可用
- 检查黑色发光指示器
