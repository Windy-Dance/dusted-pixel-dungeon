# 近战武器

## 概述

近战武器是《破碎像素地牢》中的主要战斗工具。每种武器都有一个等级（1-5），决定了其基础伤害和力量需求。

### 核心伤害公式

```java
// 每次攻击的最小伤害
min(lvl) = tier + lvl

// 每次攻击的最大伤害（标准）
max(lvl) = 5 * (tier + 1) + lvl * (tier + 1)
```

### 力量需求公式

```java
STRReq(tier, lvl) = 8 + tier * 2 - (int)(Math.sqrt(8 * lvl + 1) - 1) / 2
```

### 各等级基础力量需求

| 等级 | 基础力量需求 |
|------|--------------|
| 1    | 10           |
| 2    | 12           |
| 3    | 14           |
| 4    | 16           |
| 5    | 18           |

### 武器属性

| 属性 | 描述 |
|----------|-------------|
| **ACC**  | 准确度倍数（1.0 = 正常） |
| **DLY**  | 攻击延迟倍数（1.0 = 正常，0.5 = 2倍速度） |
| **RCH**  | 攻击范围（以格子计算，1 = 仅相邻） |

## 武器强化

武器可以被强化以修改其属性：

| 强化 | 效果 |
|---------|--------|
| **伤害** | +20%伤害，更慢的耐久损耗 |
| **速度** | +25%攻击速度，更快的耐久损耗 |
| **无** | 标准属性 |

---

## 力量过剩奖励

当英雄力量超过武器需求时：
- 随机额外伤害：0到（过剩力量）
- 每点过剩力量增加潜在额外伤害

## 力量不足惩罚

当英雄力量低于武器需求时：
- 准确度降低
- 攻击速度减慢
- 移动速度惩罚
