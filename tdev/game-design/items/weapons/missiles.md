# 远程武器

## 概述

远程武器是投掷的投射物，可以从远处造成伤害。与近战武器不同，远程武器具有耐久度，在重复使用后会损坏。

### 核心伤害公式

```java
// 每次投掷的最小伤害
min(lvl) = 2 * tier + lvl

// 每次投掷的最大伤害
max(lvl) = 5 * tier + tier * lvl
```

### 力量需求

远程武器的力量需求比同等级近战武器少1点：

```java
STRReq(lvl) = STRReq(tier, lvl) - 1
```

### 耐久度系统

- 基础耐久度：100（MAX_DURABILITY）
- 基础使用次数：8次投掷后损坏
- 每次升级：+50%耐久度
- 魔法枪套：+40%耐久度
- 耐用投射物天赋：+50-75%耐久度

### 准确度修正

| 情况 | 准确度系数 |
|-----------|-----------------|
| 目标相邻 | 0.5倍（50%） |
| 目标在射程内 | 1.5倍（150%） |
| 贴身射击天赋 | 相邻时0.75倍 |

## 特殊机制

### 刺猬效应

远程武器可以插在敌人身上并被收回：
- 武器会插在非友方敌人身上
- 敌人死亡时可收回
- 受损武器可能在收回时损坏

### 每次使用的耐久度

```java
durabilityPerUse(lvl) = MAX_DURABILITY / (baseUses * 1.5^lvl)
```

当有效使用次数达到100+时，武器具有无限耐久度（返回0）。

### 射程准确度

```java
// 相邻目标
accuracyFactor = 0.5f + 0.25f * PointBlankTalent

// 远程目标
accuracyFactor = 1.5f
```

### 神射手戒指奖励

- 伤害奖励添加到有效等级
- 应用耐久度倍数
- 与投掷武器构建协同作用

## 影响远程武器的天赋

### 耐用投射物（女猎手）
- 1级：+50%耐久度
- 2级：+75%耐久度

### 贴身射击（女猎手）
- 1级：+25%相邻准确度
- 2级：+50%相邻准确度

### 共享附魔（女猎手）
- 每点天赋有33%几率应用灵弓附魔

### 预知射击（女猎手）
- 投掷未命中时在20回合冷却时间内揭示周围区域

### 投射动量（自由奔跑者）
- 自由奔跑时每点天赋+15%伤害

## 远程武器上的附魔

远程武器可以获得附魔：
- 30%几率生成诅咒
- 10%几率生成附魔
- 索敌附魔将射程延长4格
- 易爆诅咒在爆炸时损坏投掷的武器

### 特殊互动

| 附魔 | 对远程武器的效果 |
|-------------|-------------------|
| 索敌 | +4格射程（无视墙壁） |
| 弹性 | 击中时击退 |
| 易爆 | 爆炸时损坏武器 |
| 血饮 | 造成伤害的50%用于治疗攻击者 |
