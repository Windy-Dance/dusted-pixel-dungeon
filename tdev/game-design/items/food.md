# 食物物品

食物物品满足饥饿感并可能提供额外效果。饥饿以点数衡量，其中 `HUNGRY = 100` 和 `STARVING = 300`。

## 概要表

| 食物 | 饥饿值 | 说明 |
|------|-------------|-------|
| 小份口粮 | 50 | 半份口粮 |
| 口粮 (食物) | 100 | 标准口粮 |
| 浆果 | 33 | 快速食用 (1回合)，每2个浆果生成一颗种子 |
| 肉馅饼 | 300 | 完整餐食，节日变体 |
| 肉派 | 600 | 双倍饥饿值，饱食增益 |
| 淡味水果 | 300 | 与种子一起烹饪时，获得药剂效果 |
| 烤肉 | 100 | 烹饪后的神秘肉，无副作用 |
| 冻肉片 | 100 | 冻结后的神秘肉，有益效果 |
| 神秘肉 | 50 | 随机负面效果 |
| 炖肉 | 100 | 安全食用 |
| 幽灵肉 | 300 | 具有幽灵特性的特殊肉类 |
| 补给口粮 | 50 | 快速零食 |

## 标准食物物品

### 小份口粮
- **饥饿**: 50 (HUNGRY/2)
- **价值**: 10金币
- **说明**: 基础食物物品，通常在商店中找到

### 口粮 (食物类)
- **饥饿**: 100 (HUNGRY)
- **价值**: 10金币
- **食用时间**: 3回合 (有食物天赋时1回合)
- **说明**: 标准地下城食物

### 肉馅饼
- **饥饿**: 300 (STARVING)
- **价值**: 20金币
- **节日变体**: 基于现实日期的不同效果
- **特殊效果**: 参见节日食物部分

### 肉派
- **饥饿**: 600 (2x STARVING)
- **价值**: 40金币
- **奖励效果**: 应用饱食增益
- **配方**: 肉馅饼 + 口粮 + 神秘肉变体
- **饱食**: 增加再生和伤害

### 浆果
- **饥饿**: 33 (HUNGRY/3)
- **价值**: 5金币
- **食用时间**: 1回合 (有天赋时0回合)
- **特殊**: 每吃2个浆果掉落一颗随机种子
- **遗骨**: 不出现在英雄遗骨中

---

## 神秘肉变体

### 神秘肉
- **饥饿**: 50 (HUNGRY/2)
- **价值**: 5金币
- **风险**: 随机负面效果 (1/5几率无事发生)
- **效果**:
  1. **燃烧**: 英雄着火
  2. **根植**: 英雄被根植2倍持续时间
  3. **中毒**: 英雄中毒 (HP/5伤害)
  4. **缓慢**: 英雄被减速
- **策略**: 烹饪或冷冻使其安全

### 烤肉
- **饥饿**: 100 (HUNGRY)
- **价值**: 5金币
- **制作**: 将神秘肉扔进火焰
- **安全**: 无负面效果
- **说明**: 使用神秘肉的最佳方式

### 冻肉片
- **饥饿**: 100 (HUNGRY)
- **价值**: 5金币
- **制作**: 将神秘肉扔进冻结陷阱/气体
- **安全**: 代替负面效果的有益效果
- **可能收益**:
  - 治疗少量HP
  - 治愈中毒
  - 获得隐形
  - 减缓附近敌人

### 炖肉
- **饥饿**: 100 (HUNGRY)
- **价值**: 5金币
- **制作**: 在锅中烹饪
- **安全**: 无负面效果

## 饥饿机制

### 饥饿值
- **饱足**: 0-50饥饿点
- **饥饿**: 100饥饿点
- **饥饿致死**: 300饥饿点 (最大)

### 饥饿致死
- 英雄在饥饿致死时每回合失去1 HP
- 饥饿每回合减少约1点
- 食物天赋可以修改效果

### 挑战: "节食" (NO_FOOD)
- 所有食物提供1/3正常饥饿值
- 丰饶号角也提供1/3价值每充能

### 诅咒丰饶号角
- 食用时将食物价值减少33%

## 食用时间

| 条件 | 时间 |
|-----------|------|
| 正常 | 3回合 |
| 有食物天赋 | 1回合 |
| 浆果 (正常) | 1回合 |
| 浆果 (有天赋) | 0回合 |

### 食物天赋
减少食用时间的天赋:
- 铁胃
- 充能餐
- 神秘餐
- 提神餐
- 专注餐
- 启迪餐

## 食物配方

### 肉派配方
- **成分**: 肉馅饼/幽灵肉 + 口粮 + 神秘肉变体
- **炼金成本**: 6能量
- **结果**: 1个肉派

### 淡味水果烹饪
- **成分**: 淡味水果 + 种子
- **炼金成本**: 2能量
- **结果**: 带有药剂效果的熟淡味水果
