# 异域药剂

本文档涵盖了Shattered Pixel Dungeon中的13种异域药剂。异域药剂是标准药剂的升级版本，通过将标准药剂与特定种子组合而成。

## 概述

异域药剂是标准药剂的更强大变体。它们无法正常找到，必须在炼金锅中通过将标准药剂与符石结合来制作，或通过嬗变标准药剂来制作。

## 详细描述

### 净化药剂
一种强大的防御药剂，可移除所有负面减益并赋予5回合的减益免疫。可以通过向盟友投掷来使用。饮用时还会满足饥饿感。

**移除的关键减益:** 所有负面效果，包括中毒、燃烧、腐化等。

### 腐蚀气体药剂
创建一种强大的腐蚀气体，根据地下城深度造成递增伤害。这种气体比毒气更致命，并摧毁地形。伤害强度: 2 + (深度/5)。

### 神圣启示药剂
最有价值的异域药剂之一。为任何所选等级授予一个额外的天赋点(1-4级)。每个角色每个等级只能获得一个额外点数。

**用途:**
- 更早解锁强大的4级天赋
- 通过额外效用来增强构建
- 最多限制4个额外点数 (每个等级一个)

### 龙息药剂
一种目标药剂，发射一个最多6格远的60度火焰锥形。使敌人着火并使其瘸腿5回合。还会点燃地形和打开门。

**战斗效用:**
- 无自伤 (不像某些替代品)
- 良好的区域覆盖
- 燃烧易燃障碍物

### 大地护甲药剂
赋予树皮皮肤增益，提供随英雄等级增加的额外护甲:
- 基础: 2 + (英雄等级/3) 护甲
- 持续时间: 50回合
- 与其他护甲来源叠加

### 魔法视觉药剂
赋予魔法视觉增益，暂时允许英雄透视所有墙壁和障碍物。揭示整个地下城布局，包括秘密。

### 掌握药剂
永久降低一件武器或护甲的力量需求，使其"更容易使用"。不能用于灵能弓或已经具有掌握加成的物品。

**战略价值:**
- 早期使用高级装备
- 允许更灵活的构建

### 护盾药剂
创建一个吸收伤害的屏障:
- 护盾量: ~75%的最大HP (0.6 * HT + 10)
- 即时保护 (无治疗延迟)
- 在药物恐惧症挑战下有效 (反而造成中毒伤害)

### 迷雾药剂
创建一个阻挡敌人视野的烟雾屏幕:
- 9格半径效果
- 敌人失去对英雄的视野
- 优秀的逃跑工具

### 急冻药剂
立即冻结3x3区域内的所有敌人，并施加根植效果，持续时间是正常情况的两倍。与普通冻结不同，冻结效果是即时的。

### 耐力药剂
赋予耐力增益，这是一种长时间的移动速度提升。虽然不如极速强烈，但持续时间长得多，更适合探索。

### 风暴云药剂
创建一个风暴云:
- 将敌人推开
- 使受影响的角色眩晕
- 提供区域控制
- 可以战略性地重新定位敌人

## 制作配方

异域药剂是通过使用嬗变卷轴嬗变标准药剂而创建的。每种标准药剂都会转化为其对应的异域变体。

## 战略说明

1. **神圣启示** - 对天赋构建最有价值
2. **净化** - 应对带有重型减益的Boss战必不可少
3. **护盾** - 紧急保护比治疗更好
4. **龙息** - 不需要法杖的优秀远程选择
5. **大地护甲** - 脆弱构建的强大防御选择
