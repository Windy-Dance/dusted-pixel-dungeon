# 酿剂

本文档涵盖了Shattered Pixel Dungeon中的7种酿剂物品。酿剂是通过炼金术制作的可投掷区域效果物品。

## 概述

酿剂是特殊的消耗品，只能投掷。它们产生强大的区域效果，并使用标准或异域药剂与其他物品组合制作而成。

## 酿剂表

| 酿剂 | 成分 | 炼金成本 | 效果 |
|------|-------------|--------------|--------|
| 水之酿剂 | 风暴云药剂 | 8 | 喷泉陷阱效果 |
| 暴风雪酿剂 | 冰霜药剂 | 8 | 创建暴风雪气体 |
| 腐蚀酿剂 | 毒气 + 粘液团 | 1 | 在半径3内施加粘液 |
| 地狱酿剂 | 液火药剂 | 12 | 创建地狱气体 |
| 震荡酿剂 | 麻痹气体药剂 | 10 | 在半径3内创建电力 |
| 不稳定酿剂 | 任意药剂 + 任意种子 | 1 | 随机药剂效果 |

## 详细描述

### 水之酿剂
创建类似喷泉陷阱的效果:
- 击退区域内的敌人
- 造成水伤害
- 击退方向基于投掷轨迹

**配方:** 1瓶风暴云药剂 = 8瓶水之酿剂
**数量:** 以8瓶为一批创建
**战略用途:** 将敌人定位到环境击杀点或将其击入危险区域

### 暴风雪酿剂
创建覆盖大面积的暴风雪气体云:
- 随时间冻结敌人
- 造成寒冷伤害
- 比冰霜药剂更大的区域
- 气体持续多个回合

**配方:** 1瓶冰霜药剂 = 1瓶暴风雪酿剂
**最佳用途:** 对抗敌人群体、火焰系敌人

### 腐蚀酿剂
在半径3内的所有角色身上施加粘液减益:
- 粘液造成持续伤害
- 粘液在相邻角色之间传播
- 使用任务物品粘液团制作

**配方:** 1瓶毒气药剂 + 1个粘液团 = 1瓶腐蚀酿剂
**特别注意:** 在制作中使用粘液团计入目录完成度

### 地狱酿剂
创建地狱气体云:
- 比普通火焰更强大
- 比液火药剂更大的区域
- 灼烧敌人和地形
- 摧毁易燃障碍物

**配方:** 1瓶液火药剂 = 1瓶地狱酿剂
**最佳用途:** 对抗植物敌人、区域控制

### 震荡酿剂
在半径3区域内创建电力:
- 电击范围内的所有角色
- 使目标眩晕
- 闪电链效果
- 适合打断敌人行动

**配方:** 1瓶麻痹气体药剂 = 1瓶震荡酿剂
**最佳用途:** 群体控制、打断施法者

### 不稳定酿剂
一种独特的酿剂，随机触发任何标准药剂的效果:
- 可以饮用 (用于有益药剂效果)
- 可以投掷 (用于有害药剂效果)
- 自动识别
- 由于随机性，售价较低

**配方:** 任意药剂 + 任意种子 = 1瓶不稳定酿剂

**药剂几率权重:**
| 药剂 | 权重 |
|--------|--------|
| 治疗 | 3 |
| 灵视 | 2 |
| 冰霜 | 2 |
| 液火 | 2 |
| 毒气 | 2 |
| 极速 | 2 |
| 隐形 | 2 |
| 浮空 | 2 |
| 麻痹气体 | 2 |
| 净化 | 2 |
| 经验 | 1 |

**饮用时:** 如果结果是仅投掷药剂则重新掷骰
**投掷时:** 如果结果是仅饮用药剂则重新掷骰

## 制作指南

酿剂在炼金锅中制作。炼金成本决定了从炼金锅消耗多少能量。

| 成本 | 描述 |
|------|-------------|
| 1 | 非常便宜 (使用任务物品) |
| 8 | 中等成本 |
| 10 | 较高成本 |
| 12 | 昂贵 |

## 战斗策略

1. **战前准备:** 在交战前投掷酿剂以伤害/削弱敌人
2. **瓶颈控制:** 在走廊中使用区域控制酿剂
3. **敌人聚集:** 与聚集敌人的能力结合使用
4. **逃生工具:** 使用震荡或暴风雪酿剂进行群体控制

## 注意事项

- 酿剂总是被投掷，从不饮用 (除了不稳定酿剂)
- 所有酿剂都受益于投掷武器天赋
- 如果英雄陷入区域，酿剂效果可能会伤害英雄
- 不稳定酿剂最多才多艺但最不可预测
