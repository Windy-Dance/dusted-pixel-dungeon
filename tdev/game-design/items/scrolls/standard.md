# 标准卷轴

本文档涵盖了Shattered Pixel Dungeon中的12种标准卷轴。

## 概述

卷轴是可消耗物品，在阅读时提供魔法效果。与药剂不同，所有卷轴都是通过阅读使用，而不是投掷。一些卷轴需要选择目标或物品。

## 详细描述

### 升级卷轴
最有价值的标准卷轴。将任何可装备物品升级+1级:
- 增加武器伤害
- 增加护甲防御
- 改善法杖充能
- 可能移除诅咒
- 有几率削弱/移除诅咒附魔

**特殊属性:**
- 独特 (更高生成率)
- 2倍天赋经验
- 计入升级统计数据

### 鉴定卷轴
鉴定单个物品的所有属性:
- 显示确切等级
- 显示诅咒
- 显示附魔/刻印
- 不能用于已鉴定物品

**使用碎裂遗忘:** 设置物品为被动鉴定就绪状态

### 传送卷轴
将英雄传送到随机位置:
- 偏好未探索区域
- 可以发现秘密房间
- 移除根植减益
- 在特殊区域可能失败

**特殊交互:**
- 可以传送到秘密/特殊房间入口
- 如果被Boss机制根植则无法传送

### 祛邪卷轴
从装备中移除诅咒:
- 对已装备物品有效
- 可以净化受折磨的灵魂
- 移除降解减益
- 揭示未识别物品的诅咒状态

**移除的诅咒类型:**
- 诅咒装备
- 诅咒附魔
- 诅咒刻印
- 降解减益

### 充能卷轴
为法杖使用者赋予充能增益:
- 快速恢复法杖充能
- 持续时间: 固定量
- 视觉充能效果
- 与法师构建协同

### 探地卷轴
显示整个楼层布局:
- 显示所有地形
- 发现所有秘密
- 更新战争迷雾
- 可能揭示隐藏门

**奖励:** 如果发现秘密会播放声音

### 盛怒卷轴
激怒所有可见敌人:
- 施加狂乱5回合
- 敌人攻击最近目标 (包括彼此)
- 吸引所有敌人朝向英雄
- 有效制造混乱

**策略:** 在被许多敌人包围时使用

### 恐惧卷轴
使所有可见敌人逃跑:
- 施加恐惧减益
- 持续时间: 完整恐惧持续时间
- 敌人远离英雄逃跑
- 有效的逃跑工具

**效果:** 因敌人而异；有些免疫

### 催眠卷轴
使所有可见敌人 (和英雄) 入睡:
- 施加困倦，导致魔法睡眠
- 也影响英雄
- 用于潜行方法
- 可用于逃跑

**注意:** 英雄受到伤害可以醒来

### 镜像卷轴
创建2个英雄镜像:
- 镜像与英雄并肩作战
- 伤害基于装备武器
- HP有限
- 可以触发天赋

**战斗效用:**
- 额外伤害输出
- 分散敌人注意力
- 继承英雄武器

### 复仇卷轴
根据英雄缺失的HP伤害所有可见敌人:
- 基础伤害: 敌人最大HP的10%
- 额外伤害随英雄受伤程度增加
- 致盲敌人
- 使英雄虚弱和致盲

**伤害公式:**
- 力量随HP缺失从0x到1x缩放 (HP约10%时最大)
- 最大力量: 4x乘数

### 嬗变卷轴
将一个物品转变为同一类别中的不同物品:
- 武器变成不同武器
- 戒指变成不同戒指
- 法杖变成不同法杖
- 药剂变成其异域变体 (反之亦然)
- 神器变成不同神器

**特殊规则:**
- 保留升级等级
- 保留诅咒状态
- 独特物品转换方式不同
- 种子变成不同种子
- 符石变成不同符石

## 识别

卷轴必须被识别才能知道其效果。方法包括:

1. **使用时未识别:** 使用时识别
2. **鉴定卷轴:** 直接识别
3. **经验药剂:** 可能授予识别
4. **商店购买:** 已识别物品

## 注意事项

- 所有卷轴阅读需1回合
- 阅读可以被打断
- 诅咒卷轴可能有负面效果
- 卷轴在沉默时有效 (不像法术)
- 库存卷轴可以直接对物品使用
