# 异域卷轴

本文档涵盖了Shattered Pixel Dungeon中的12种异域卷轴。异域卷轴是标准卷轴的升级版本，具有更强大的效果。

## 概述

异域卷轴通过嬗变标准卷轴创建。它们提供基础卷轴效果的增强版本或全新的能力。

## 异域卷轴表

| 异域卷轴 | 基础卷轴 | 效果 |
|---------------|-------------|--------|
| 反魔法卷轴 | 充能 | 赋予魔法免疫 |
| 挑战卷轴 | 盛怒 | 创建战斗竞技场 |
| 占卜卷轴 | 鉴定 | 鉴定4个随机物品 |
| 恐怖卷轴 | 恐惧 | 造成永久恐惧或击杀 |
| 附魔卷轴 | 升级 | 添加附魔/刻印 |
| 远见卷轴 | 探地 | 随时间揭示秘密 |
| 变形卷轴 | 嬗变 | 改变天赋 |
| 神秘能量卷轴 | 充能 | 给神器充能 |
| 通道卷轴 | 传送 | 返回上一楼层 |
| 棱镜影像卷轴 | 镜像 | 创建棱镜守护者 |
| 心灵爆破卷轴 | 复仇 | 对所有敌人造成重伤害 |
| 塞壬之歌卷轴 | 催眠 | 魅惑 + 转化一个敌人 |

## 详细描述

### 反魔法卷轴
赋予魔法免疫增益:
- 对所有魔法效果免疫
- 持续时间: 固定量
- 无法被法术瞄准
- 阻挡有害和有益的魔法

**战略用途:** 安全对抗魔法重型Boss

### 挑战卷轴
创建战斗竞技场效果:
- 所有敌人被吸引到英雄身边
- 竞技场区域在内部提供护甲增益
- 竞技场持续100回合或直到英雄离开
- 竞技场大小适应房间大小

**竞技场属性:**
- 视觉红色粒子效果
- 在竞技场内获得护甲加成
- 计时器在增益图标上可见
- 适合控制战斗

### 占卜卷轴
一次性鉴定多个物品:
- 鉴定4个随机未知物品
- 可以是药剂、卷轴或戒指
- 优先不同类别
- 结果在窗口中显示

**价值:** 比4个鉴定卷轴更高效

### 恐怖卷轴
一种强大的恐惧卷轴:
- 施加恐怖 (永久恐惧) 到敌人
- 免疫敌人改为普通恐惧
- 恐怖使敌人逃跑直到死亡或治愈
- 可能使敌人立即死亡

**对抗Boss:** 改为施加恐惧而非恐怖

### 附魔卷轴
为装备添加附魔或刻印:
- 从3个随机选项中选择
- 可以附魔武器
- 可以为护甲铭刻刻印
- 替换现有附魔/刻印

**特殊属性:**
- 独特 (更高价值)
- 在选择前可以预览附魔效果
- 提供每个选项的信息

### 远见卷轴
赋予远见增益:
- 警告隐藏陷阱和秘密
- 随时间揭示秘密
- 持续时间: 固定量
- 优秀的探索工具

**探索价值:** 防止陷阱死亡，发现秘密

### 变形卷轴
将一个天赋变为不同职业的天赋:
- 选择要替换的天赋
- 从3个随机替换选项中选择
- 必须是相同等级
- 保留投入的天赋点

**要求:**
- 不能替换没有的天赋
- 替换必须是相同等级
- 打开天赋选择窗口

### 神秘能量卷轴
赋予神器充能增益:
- 快速给神器充能
- 持续时间: 30回合
- 对所有已装备神器有效
- 可以排除特定神器 (如丰饶号角)

**最适合:** 神器重型构建

### 通道卷轴
将英雄传送到安全位置:
- 返回当前区域的起始点
- 返回到楼层: max(1, 深度 - 1 - (深度-2)%5)
- 示例: 第6层返回第5层，第10层返回第9层
- 如果楼层间传送被阻止则无法使用

**安全用途:** 紧急逃跑到已清理楼层

### 棱镜影像卷轴
创建或治疗棱镜影像:
- 如果不存在影像: 创建守护者
- 如果存在影像: 治疗至满HP
- 影像是强大的盟友
- 可以置于停滞状态

**棱镜影像属性:**
- 基于英雄的高HP
- 使用英雄武器
- 在楼层间持久存在

### 心灵爆破卷轴
毁灭性的区域攻击:
- 对所有可见敌人造成50% HT + 50% HP伤害
- 英雄受到伤害 (每击中目标减少)
- 致盲并虚弱英雄延长持续时间
- 如果不小心可能杀死英雄

**英雄伤害:** 50% HT * (0.9^击中目标数)

**警告:** 可能因友好魔法导致死亡

### 塞壬之歌卷轴
针对一个敌人进行转化:
- 目标敌人成为永久盟友
- 所有其他可见敌人被魅惑
- 无法转化免疫敌人 (他们会受到魅惑)
- 转化敌人有心形视觉效果

**策略:** 转化群体中最强大的敌人

## 嬗变配方

所有异域卷轴都是通过在对应标准卷轴上使用嬗变卷轴创建的:

| 标准 | → | 异域 |
|----------|---|--------|
| 充能 | → | 反魔法 |
| 盛怒 | → | 挑战 |
| 鉴定 | → | 占卜 |
| 恐惧 | → | 恐怖 |
| 升级 | → | 附魔 |
| 探地 | → | 远见 |
| 嬗变 | → | 变形 |
| 充能 | → | 神秘能量 |
| 传送 | → | 通道 |
| 镜像 | → | 棱镜影像 |
| 复仇 | → | 心灵爆破 |
| 催眠 | → | 塞壬之歌 |

## 价值比较

所有异域卷轴都有:
- 金币价值: 50
- 能量价值: 10
- 比标准卷轴价值更高

## 战略说明

1. **附魔** - 装备增强最有价值
2. **变形** - 启用跨职业天赋构建
3. **塞壬之歌** - 转化危险敌人
4. **心灵爆破** - 伤害最高但危险
5. **挑战** - 适合控制Boss战

## 注意事项

- 所有异域卷轴都算作独特
- 许多在使用前提供目标选项
- 有些可以在不消耗卷轴的情况下取消
- 附魔和变形有详细的选择窗口
