# 护甲刻印 - Shattered Pixel Dungeon

刻印是可以铭刻在护甲上的有益附魔。当穿戴者被攻击时，它们会提供被动效果。

## 刻印稀有度等级

| 稀有度 | 刻印 | 几率 |
|--------|--------|--------|
| 常见 (50%) | 晦暗, 迅捷, 粘稠, 电势 | 各约12.5% |
| 不常见 (40%) | 狱火, 磐岩, 缠绕, 反斥, 迷彩, 涌流 | 各约6.67% |
| 稀有 (10%) | 魅惑, 敌法, 荆棘 | 各约3.33% |

## 常见刻印

### 晦暗
| 属性 | 数值 |
|----------|-------|
| 颜色 | 灰色 (0x888888) |
| 效果 | 增加潜行 |
| 公式 | `(1 + 等级/3) * 乘数` |

**描述**: 隐藏穿戴者，使其更难以被发现。潜行加成随护甲等级提升而增加。

### 迅捷
| 属性 | 数值 |
|----------|-------|
| 颜色 | 黄色 (0xFFFF00) |
| 效果 | 附近无敌人时提升速度 |
| 公式 | `1.2 + 0.04 * 等级 * 乘数` |
| 触发条件 | 3格路径内无敌人 |

**描述**: 在不靠近敌人时增强移动速度。激活时显示粒子效果。

### 粘稠
| 属性 | 数值 |
|----------|-------|
| 颜色 | 紫色 (0x8844CC) |
| 效果 | 将伤害延迟分摊 |
| 公式 | `(等级+1)/(等级+6) * 乘数` 伤害延迟 |

**描述**: 存储受到的伤害，并随着时间缓慢造成，而不是一次性全部造成。显示"延迟X"状态。

### 电势
| 属性 | 数值 |
|----------|-------|
| 颜色 | 白色 (0xFFFFFF, 60%不透明度) |
| 效果 | 为手持法杖充能 |
| 公式 | `(等级+1)/(等级+6)` 触发几率 |
| 0级 | ~16.7% 几率 |
| 1级 | ~28.6% 几率 |
| 2级 | ~37.5% 几率 |

**描述**: 被击中时积累能量，激活时为穿戴者的法杖提供充能。

## 不常见刻印

### 狱火
| 属性 | 数值 |
|----------|-------|
| 颜色 | 橙色 (0xFF4400) |
| 效果 | 火焰免疫 |
| 触发条件 | 被动 (在 Char.isImmune 中检查) |

**描述**: 保护穿戴者及其物品免受火焰伤害，完全消除其伤害和负面效果。

### 磐岩
| 属性 | 数值 |
|----------|-------|
| 颜色 | 深灰色 (0x222222) |
| 效果 | 将闪避转换为伤害减免 |
| 权衡 | 闪避设为0 |

**描述**: 用沉重的魔法岩石包围护甲。使闪避成为不可能，但按闪避属性的比例阻挡伤害。

### 缠绕
| 属性 | 数值 |
|----------|-------|
| 颜色 | 棕色 (0x663300) |
| 效果 | 赋予大地之根护甲 |
| 公式 | 25% 几率, `(5 + 2*等级) * 乘数` 护甲 |

**描述**: 在穿戴者周围生长出泥土护甲以吸收伤害。如果穿戴者移动，护甲会破裂。

### 反斥
| 属性 | 数值 |
|----------|-------|
| 颜色 | 白色 (0xFFFFFF) |
| 效果 | 击退相邻攻击者 |
| 公式 | `(等级+1)/(等级+5)` 几率, 2格击退 |
| 0级 | 20% 几率 |
| 1级 | 33% 几率 |
| 2级 | 43% 几率 |

**描述**: 对攻击者反弹力量，使其向后飞去。

### 迷彩
| 属性 | 数值 |
|----------|-------|
| 颜色 | 绿色 (0x448822) |
| 效果 | 踩踏草丛时隐形 |
| 公式 | `(3 + 等级/2) * 乘数` 回合 |

**描述**: 允许穿戴者融入高草丛中，在穿过时获得临时隐形效果。

### 涌流
| 属性 | 数值 |
|----------|-------|
| 颜色 | 蓝色 (0x0000FF) |
| 效果 | 在水中提升速度 |
| 公式 | `(2 + 0.5*等级) * 乘数` |
| 触发条件 | 站在水格上 |

**描述**: 操纵周围的水流，使穿戴者在水中移动时更快。

## 稀有刻印

### 魅惑
| 属性 | 数值 |
|----------|-------|
| 颜色 | 粉色 (0xFF4488) |
| 效果 | 魅惑攻击者 |
| 公式 | `(等级+3)/(等级+20)` 几率 |
| 0级 | ~15% 几率 |
| 1级 | ~19% 几率 |
| 2级 | ~23% 几率 |

**描述**: 操控攻击者的心智，使其暂时被魅惑。被魅惑的敌人不会攻击。

### 敌法
| 属性 | 数值 |
|----------|-------|
| 颜色 | 青色 (0x88EEFF) |
| 效果 | 魔法抗性 + 对魔法的额外伤害减免 |
| DR公式 | `等级` 到 `3 + 等级*1.5` |

**描述**: 使护甲能够防御魔法攻击以及物理攻击。对魔法提供额外的伤害减免。

**抗性包括**:
- 魔法减益 (魅惑, 虚弱, 易伤, 诅咒, 降解)
- 法杖伤害 (所有法杖)
- 魔法陷阱 (解离, 致命)
- 魔法能力 (元素爆破, 各种Boss攻击)

### 荆棘
| 属性 | 数值 |
|----------|-------|
| 颜色 | 深红色 (0x660022) |
| 效果 | 对攻击者造成流血伤害 |
| 公式 | `(等级+2)/(等级+12)` 几率, `(4+等级)*乘数` 伤害 |
| 0级 | ~16.7% 几率 |
| 1级 | ~23.1% 几率 |
| 2级 | ~28.5% 几率 |

**描述**: 伤害攻击者，使其在攻击穿戴者时缓慢流血。

## 触发几率乘数

所有刻印效果都受触发几率乘数影响:
- 基础: 1.0x
- 奥术戒指: 增加乘数
- 防护光环: +25% + 每天赋点+25%
- 神圣守卫: 为圣骑士提供伤害阻挡

## 升级时的刻印丢失

升级时可能会丢失刻印:
- +4 到 +5: 10-20% 几率
- +6 到 +7: 40-80% 几率
- +8+: 100% 几率

**战士的符文转移天赋** 将刻印丢失延迟1-2个升级等级。

## 强化刻印

刻印可以被"强化"以防止丢失:
- 通过特定游戏机制应用
- +6级以上有10-20%几率失去强化
- +10级以上有100%几率失去强化
