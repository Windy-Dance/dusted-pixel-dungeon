# 护甲诅咒 - Shattered Pixel Dungeon

诅咒是可以出现在护甲上的有害刻印变体。与普通刻印不同，诅咒具有负面效果，并被标记为诅咒物品。

## 诅咒列表

| 诅咒 | 触发几率 | 效果 |
|-------|-------------|--------|
| 反熵 | 12.5% | 灼烧穿戴者，冻结周围环境 |
| 臃肿 | 被动 | 穿过门时减慢移动速度 |
| 腐蚀 | 10% | 扩散酸性粘液 |
| 定相 | 5% | 随机传送 |
| 代谢 | 16.7% | 将饥饿转换为治疗 |
| 分身 | 5% | 创建镜像或敌人分身 |
| 丛生 | 5% | 激活随机植物 |
| 恶臭 | 12.5% | 释放有毒气体 |

## 详细诅咒效果

### 反熵
| 属性 | 数值 |
|----------|-------|
| 颜色 | 黑色 (0x000000) |
| 触发几率 | 1/8 (12.5%) |
| 效果 | 灼烧穿戴者，冻结周围8格 |

**描述**: 违背宇宙力量，将能量拉入穿戴者并从周围环境中抽走。短暂地点燃穿戴者，同时冻结周围的一切。

```
效果: Freezing.affect(defender.pos + NEIGHBOURS8)
       火焰状态在穿戴者身上 (如果不在水中则持续4回合)
```

### 臃肿

|----------|-------|
| 颜色 | 黑色 (0x000000) |

**描述**: 护甲看起来更大更威严，但没有增强防御。增加的尺寸使进入门口变得非常困难。

```

```

### 代谢

|----------|-------|

```
饥饿消耗 = 治疗量 * 10
```


## 诅咒机制

### 诅咒识别
- 诅咒护甲一旦装备就无法卸下
- 物品信息中显示"诅咒"状态
- 牧师有15-25%的几率在装备前检测到诅咒 (神圣直觉天赋)

### 移除诅咒
| 方法 | 成功率 |
|--------|--------------|
| 升级 | 每次升级33%几率 |
| 祛邪卷轴 | 100% |

### 诅咒注入
当通过诅咒注入应用诅咒时:
- 获得+1等级加成
- 每6基础等级额外+1
- 清除诅咒时移除加成

## 诅咒与刻印比较

| 方面 | 刻印 | 诅咒 |
|--------|-------|-------|
| 效果 | 有益 | 有害 |
| 移除 | 升级时可能丢失 | 升级时33%移除几率 |
| 物品状态 | 正常 | 诅咒 (无法卸下) |
| 发光颜色 | 各种 | 黑色 (0x000000) |
| 随机生成 | 15%几率 | 30%几率 |

## 触发几率乘数

诅咒使用与刻印相同的触发几率乘数:
- 奥术戒指影响诅咒触发率
- 更高的乘数 = 更频繁的诅咒效果 (对玩家不利)

## 相关文件

- `Armor.Glyph.curses` 数组在 `Armor.java` 中 - 诅咒类列表
- `curses/*.java` - 各个诅咒实现

## 实现说明

所有诅咒:
1. 继承 `Armor.Glyph` 类
2. 重写 `curse()` 返回 `true`
3. 使用黑色发光颜色 (0x000000)
4. 对穿戴者有负面效果
5. 无法在护甲掉落中自然获得 (仅通过诅咒生成)
