# 戒指概述

戒指是提供被动属性加成的可装备物品。它们可以戴在戒指槽中，当装备两个相同类型的戒指时会叠加效果。

## 详细戒指机制

### 精准戒指
- **效果**: 增加攻击准确性
- **公式**: `accuracyMultiplier = 1.3^等级`
- **+1时**: +30%准确性
- **+10时**: +1278%准确性 (13.78x)

### 奥术戒指
- **效果**: 增加武器附魔和护甲刻印威力
- **公式**: `enchantPowerMultiplier = 1.175^等级`
- **+1时**: +17.5%附魔威力
- **+10时**: +410%附魔威力 (5.1x)
- **影响**: 所有附魔、刻印、诅咒效果

### 元素戒指
- **效果**: 减少来自元素和魔法来源的伤害
- **公式**: `damageTaken = damage * 0.825^等级`
- **+1时**: 17.5%伤害减免
- **+10时**: 79%伤害减免
- **抗性**: 火焰、寒冷、电击、中毒、气体、麻痹、腐蚀、粘液

### 能量戒指
- **效果**: 增加法杖、神器和护甲能力的充能率
- **公式**: `chargeMultiplier = 1.175^等级`
- **+1时**: +17.5%充能速度
- **+10时**: +410%充能速度 (5.1x)
- **天赋协同**: 
  - 光明阅读 (牧师): 每点额外+20%法杖充能
  - 轻斗篷 (盗贼): 每点额外+20%神器充能

### 闪避戒指
- **效果**: 增加闪避几率
- **公式**: `evasionMultiplier = 1.125^等级`
- **+1时**: +12.5%闪避
- **+10时**: +224%闪避 (3.24x)

### 武力戒指
- **效果**: 增加徒手攻击伤害
- **公式**: 基于STR的有效阶级造成的伤害
- **阶级计算**: `tier = max(1, (STR-8)/2)`, 5级以上递减
- **伤害**: `min = tier + level`, `max = 5*(tier+1) + level*(tier+1)`
- **决斗者能力**: 拳手姿态 - 激活时进一步加成
- **STR 10, +1时**: 2-12伤害
- **STR 20, +10时**: 12-80伤害 (无姿态)

### 狂怒戒指
- **效果**: 增加攻击速度
- **公式**: `attackSpeedMultiplier = 1.09051^等级`
- **+1时**: +9%攻击速度
- **+10时**: +136%攻击速度 (2.36x)

### 疾速戒指
- **效果**: 增加移动速度
- **公式**: `speedMultiplier = 1.175^等级`
- **+1时**: +17.5%移动速度
- **+10时**: +410%移动速度 (5.1x)

### 根骨戒指
- **效果**: 增加STR和最大HP
- **公式**: 
  - `STRBonus = level + 1`
  - `HPMultiplier = 1.035^level`
- **+1时**: +1 STR, +3.5% HP
- **+10时**: +10 STR, +41% HP
- **注意**: 装备/卸下时更新最大HP

### 神射戒指
- **效果**: 增加远程武器伤害和耐久度
- **公式**:
  - `damageBonus = level` (添加到伤害掷骰)
  - `durabilityMultiplier = 1.2^level`
- **+1时**: +1伤害, +20%耐久度
- **+10时**: +10伤害, +519%耐久度 (6.2x)

### 韧性戒指
- **效果**: 根据缺失HP减少受到的伤害
- **公式**: `damageTaken = damage * 0.85^(level * HP_missing%)`
- **+1, 50% HP时**: 7.5%减少
- **+10, 50% HP时**: 81%减少
- **接近死亡, +10时**: 最多80%减少
- **缩放**: 受伤时更有效

### 财富戒指
- **效果**: 增加物品掉落几率和质量
- **公式**: `dropChanceMultiplier = 1.2^level`
- **+1时**: +20%掉落几率
- **+10时**: +519%掉落几率 (6.2x)
- **额外掉落**: 定期额外消耗品/装备掉落
- **掉落类型**: 金币、药剂、卷轴、炸弹、装备

## 戒指机制

### 识别
- 戒指在装备时获得1级经验后识别
- 诅咒戒指佩戴时有负面效果
- 碎裂遗忘可以强制识别

### 等级加成
- **单独加成**: `level + 1` (或诅咒时`min(0, level - 2)`)
- **强化加成**: 包括增强效果 (强化戒指增益)

### 叠加
- 两个相同类型的戒指以等级加成叠加
- 使用强化加成的效果是乘法的
- 示例: 两个+5精准戒指 = +10等级 = 13.79x准确性

### 诅咒戒指
- 诅咒戒指没有祛邪无法卸下
- 提供负面加成: `min(0, level - 2)`
- +0诅咒时: -2有效等级

### 宝石类型 (仅视觉)
- 石榴石、红宝石、黄玉、祖母绿、黑曜石
- 欧泊、碧玺、蓝宝石、紫水晶
- 水晶、玛瑙、钻石

## 财富掉落机制

财富戒指有特殊的掉落系统:
- **计数器系统**: 追踪回合和掉落
- **消耗品掉落**: 低/中/高级分别为60%/30%/10%
- **装备掉落**: 每5-10个消耗品掉落
- **最低等级**: 装备最低+等级/2
- **额外附魔**: 每等级+10%几率获得好附魔
