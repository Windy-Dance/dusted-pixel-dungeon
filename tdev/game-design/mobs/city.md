# 城市区（深度16-20）

城市包含游戏中最复杂的普通敌人，其能力可以对抗常见策略，需要玩家适应。

## 普通怪物

### 矮人尸群

亡灵战士成对生成并互相复活。

| 属性 | 数值 |
|------|-------|
| HP | 45 |
| 防御技能 | 20 |
| 伤害 | 16-22 |
| 攻击技能 | 24 |
| DR | 0-4 |
| EXP | 5 |
| 最高等级 | 20 |

**属性：**
- UNDEAD

**战利品：**
- 金币（20%几率）

**特殊机制：**
- **伙伴系统**：总是与伙伴矮人尸群一起生成
- **生命链接**：死亡时，倒下的矮人尸群转移到附近存活的矮人尸群
- **复活**：倒下的矮人尸群在回合数基础上复活（倒下次数 × 5回合）
- **连锁复活**：多个矮人尸群可以互相持续复活

**倒下状态：**
- 外观皱缩但可见
- 如果距离足够远，可以转移到另一个矮人尸群
- 在深渊中死亡可防止复活

**策略：**
- 在复活前快速击杀两个矮人尸群
- 分离矮人尸群以防止生命链接
- 深渊击杀可防止复活
- 高爆发伤害至关重要

### 矮人术士

亡灵法师，会施加减益并造成远程伤害。

| 属性 | 数值 |
|------|-------|
| HP | 70 |
| 防御技能 | 18 |
| 伤害 | 12-18（近战和远程）
| 攻击技能 | 25 |
| DR | 0-8 |
| EXP | 11 |
| 最高等级 | 21 |

**属性：**
- UNDEAD

**战利品：**
- 药水（50%几率，治疗药水掉落有限）

**特殊机制：**
- **暗影飞弹**：远程魔法攻击，1回合施法时间
- **降级**：50%几率命中时施加降级减益
- **魔法飞弹射程**：可以使用Ballistica进行远程攻击

**降级效果：**
- 降低装备效果
- 持续Degrade.DURATION回合
- 持续时间叠加，不叠加强度

**策略：**
- 由于降级为高优先级目标
- 快速接近距离
- 降级使装备效果降低
- 可被魔法免疫抵抗

### 矮人武僧

亡灵武术家，拥有完美闪避。

| 属性 | 数值 |
|------|-------|
| HP | 70 |
| 防御技能 | 30 |
| 伤害 | 12-25 |
| 攻击技能 | 30 |
| DR | 0-2 |
| EXP | 11 |
| 最高等级 | 21 |

**属性：**
- UNDEAD

**战利品：**
- 食物（8.3%几率）

**特殊机制：**
- **专注**：狩猎时获得完美闪避增益
- **招架**：闪避攻击并失去专注，触发6-7回合冷却
- **双倍攻击速度**：0.5x攻击延迟
- **冷却减少**：移动更快减少专注冷却

**专注行为：**
- INFINITE_EVASION专注时
- 招架触发冷却（6-7回合）
- 移动额外减少冷却0.67

**策略：**
- 攻击一次触发招架，然后在冷却期间攻击
- 放风筝实际上帮助他们更快恢复专注
- 低DR使持续伤害有效
- 快速攻击意味着他们输出高DPS

## Boss：矮人国王

亡灵矮人城市的国王，位于深度20。

| 属性 | 正常 | 更强Boss |
|------|--------|-----------------|
| HP | 300 | 450 |
| 防御技能 | 22 | 22 |
| EXP | 40 | 40 |

**属性：**
- BOSS
- UNDEAD

### 战斗属性

| 攻击类型 | 伤害 | 攻击技能 |
|-------------|--------|--------------|
| 近战 | 15-25 | 26 |
| DR | 0-10 | - |

### 阶段结构

**第一阶段（100-17% HP / 100-22% 更强Boss）：**
- 积极战斗
- 定期召唤手下
- 使用生命链接和传送能力
- 伤害减少冷却

**第二阶段（17% HP / 22% 更强Boss - 护盾阶段）：**
- 传送到王座
- 变得IMMOVABLE且无敌
- 拥有完整HP屏障护盾
- 召唤手下波次

**第三阶段（护盾摧毁后）：**
- 重新开始积极战斗
- 持续召唤手下
- 延迟伤害系统

### 第一阶段能力

**召唤：**
- 正常：每10-14回合（更强Boss为8-10回合）
- DKGhoul（75%），DKMonk/DKWarlock（25%）

**生命链接：**
- 与最远的手下链接
- 与链接手下共享伤害
- 冷却时间：10-14回合（更强Boss为8-10回合）

**传送：**
- 将手下传送到英雄旁边
- 自己重新定位到对面
- 冷却时间：10-14回合（更强Boss为8-10回合）

### 第二阶段手下波次

**正常：**
| 波次 | 触发条件 | 手下 |
|------|---------|---------|
| 1 | 开始 | 1个矮人尸群 |
| 2 | 护盾 ≤200 | 3个矮人尸群 + 1个武僧/术士 |
| 3 | 护盾 ≤100 | 2个矮人尸群 + 1个武僧 + 1个术士 |

**更强Boss：**
| 波次 | 触发条件 | 手下 |
|------|---------|---------|
| 1 | 开始 | 2个矮人尸群 |
| 2 | 护盾 ≤300 | 2个矮人尸群 + 1个武僧 |
| 3 | 护盾 ≤150 | 2个矮人尸群 + 1个术士 + 1个武僧 或 2个魔像 |

### 第三阶段机制

**延迟伤害：**
- 所有伤害被延迟而不是立即生效
- 伤害显示为延迟减益
- 随时间造成伤害而不是立即造成

**持续召唤：**
- 每3回合召唤手下（更强Boss为2回合）
- 最多同时存在4个活跃召唤物

### Boss手下变种

| 类型 | 特殊属性 |
|------|-------------------|
| DKGhoul | 无伙伴，总是狩猎 |
| DKMonk | 标准属性，总是狩猎 |
| DKWarlock | 法杖释放损失400 Boss分数 |
| DKGolem | 仅在更强Boss中，总是狩猎 |

### 策略

1. **第一阶段**：快速击杀手下，避免生命链接
2. **第二阶段**：专注于破坏护盾，管理波次生成
3. **第三阶段**：在造成延迟伤害的同时击杀手下
4. **位置**：控制手下生成位置
5. **爆发伤害**：所有阶段都至关重要

### 掉落

- 国王王冠（保证掉落）
- 劳埃德信标升级（如果拥有）
- 死亡时移除降级减益
