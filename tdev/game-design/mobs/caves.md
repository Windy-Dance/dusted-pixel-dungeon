# 洞穴区域（深度11-15）

洞穴引入了明显更强的敌人和更复杂的能力。这个区域测试玩家的构筑强度和战术意识。

## 普通怪物

### 吸血蝙蝠

快速的飞行生物，攻击时会治疗自己。

| 属性 | 数值 |
|------|-------|
| HP | 30 |
| 防御技能 | 15 |
| 伤害 | 5-18 |
| 攻击技能 | 16 |
| DR | 0-4 |
| EXP | 7 |
| 最高等级 | 15 |
| 基础速度 | 2.0x |

**属性：**
- 飞行

**战利品：**
- 治疗药水（16.7%基础几率，限制7次掉落）

**特殊机制：**
- **生命偷取**：成功攻击时治疗（伤害 - 4），上限为缺失HP
- 双倍移动速度
- 飞行时忽略地形障碍

**策略：**
- 高爆发伤害防止治疗
- 快速且高伤害 - 群体中很危险
- 快速击杀以最小化治疗

### 魔像

巨大的构造体，可以传送自己和敌人。

| 属性 | 数值 |
|------|-------|
| HP | 120 |
| 防御技能 | 15 |
| 伤害 | 25-30 |
| 攻击技能 | 28 |
| DR | 0-12 |
| EXP | 12 |
| 最高等级 | 22 |

**属性：**
- INORGANIC
- LARGE

**战利品：**
- 武器或护甲（20%基础几率，掉落递减）
- 使用T5（恶魔大厅）概率表

**特殊机制：**
- **传送敌人**：释放以将敌人传送到魔像旁边
- **传送自己**：游荡时可传送到随机目的地
- **冷却时间**：30回合（自己），20回合（敌人传送）

**策略：**
- 极高伤害 - 尽可能避免近战
- 高DR使持续伤害很重要
- 传送可能打乱位置
- 对许多减益免疫（见抗性）

| 属性 | 数值 |
|------|-------|

- 冰霜：避开水域格子

- 混沌：不可预测，高优先级

## Boss：DM-300

在深处苏醒的古代采矿机器。位于深度15。

| 属性 | 正常 | 更强Boss |
|------|--------|-----------------|
| HP | 300 | 400 |
| 防御技能 | 15 | 15 |
| EXP | 30 | 30 |

**属性：**
- BOSS
- INORGANIC
- LARGE

### 战斗属性

| 攻击类型 | 伤害 | 攻击技能 |
|-------------|--------|--------------|
| 近战 | 15-25 | 20 |
| DR | 0-10 | - |

**抗性：**
- 恐怖
- 魅惑
- 眩晕
- 残废
- 冻伤
- 寒冷
- 根系束缚
- 缓慢

**免疫：**
- 睡眠

### 特殊机制

**气体排放：**
- 向目标发射毒气线
- 正常：250总气体体积
- 更强Boss：2倍气体体积
- 冷却时间：5-9回合（更强Boss为5-7回合）

**落石：**
- 在目标区域掉落岩石
- 6-12伤害 + 3回合麻痹（更强Boss为10-20 + 5回合）
- 击退：相邻时2格，2格远时1格
- 在模式中创建单个安全格子

**超级充能：**
- 当HP达到阈值时激活
- HP阈值：66%（正常）或75%/50%/25%（更强Boss）
- 变得无敌
- 双倍移动速度
- 破墙追击

**能量塔互动：**
- 超级充能期间必须摧毁能量塔
- 正常：2座能量塔
- 更强Boss：3座能量塔
- 摧毁所有能量塔结束超级充能

**能量护盾：**
- 踩踏非激活陷阱获得屏障
- 护盾数量：30 + (HT-HP)/10
- 不从充能格子获得

### 阶段表

| HP范围 | 行为 |
|----------|----------|
| 100-67% | 正常战斗，使用能力 |
| 66-0%（正常） | 66%时超级充能，33%时重复 |
| 75-0%（更强） | 75%、50%、25%时超级充能 |

### 策略

1. **避开气体** - 垂直于气体线移动
2. **注意落石预示** - 红色格子显示冲击区域
3. **超级充能期间的能量塔** - 必须摧毁才能继续
4. **不要穿陷阱放风筝** - DM-300获得护盾
5. **破墙** - 超级充能的DM-300破坏墙壁
6. **更强Boss** - 3座能量塔，更快的能力，更多伤害

### 掉落

- 金属碎片（2-4个，平均2.5个）
- 劳埃德信标升级（如果拥有）
