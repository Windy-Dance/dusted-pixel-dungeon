# 监狱区域（深度6-10）

监狱引入了更复杂的敌人机制和更大的威胁。这里的敌人拥有需要战略性思考才能克服的特殊能力。

## 普通怪物

### 骷髅

亡灵战士，死亡时会爆炸。

| 属性 | 数值 |
|------|-------|
| HP | 25 |
| 防御技能 | 9 |
| 伤害 | 2-10 |
| 攻击技能 | 12 |
| DR | 0-5 |
| EXP | 5 |
| 最高等级 | 10 |

**属性：**
- UNDEAD
- INORGANIC

**战利品：**
- 武器（16.7%基础几率，每次掉落递减）

**特殊机制：**
- **骨爆**：死亡时，对所有相邻角色造成6-12伤害
- 骨爆的DR加倍
- 如果被深渊击杀则不会爆炸

**策略：**
- 尽可能使用远程击杀
- 拥有高DR以承受爆炸
- 完成击杀时不要站在相邻位置
- 深渊击杀可防止爆炸

### 疯狂小偷

亡灵盗贼，会偷取物品并逃跑。

| 属性 | 数值 |
|------|-------|
| HP | 20 |
| 防御技能 | 12 |
| 伤害 | 1-10 |
| 攻击技能 | 12 |
| DR | 0-3 |
| EXP | 5 |
| 最高等级 | 11 |

**属性：**
- UNDEAD

**战利品：**
- 戒指或神器（3%基础几率，每次掉落递减）

**特殊机制：**
- **偷窃**：命中时，从英雄处偷取随机非唯一物品
- **逃跑**：偷窃后，进入FLEEING状态
- **逃脱**：如果距离6格以上且不在视野内，传送出地图并带走偷取的物品
- **携带时减速**：携带偷取物品时速度为5/6
- **双倍攻击速度**：0.5x攻击延迟

**策略：**
- 在他们逃脱前快速击杀
- 不要携带有价值的未装备物品
- 如果他们逃脱，物品将永久丢失
- 死亡时会掉落偷取的物品和金币

### 监狱守卫

披甲的亡灵监狱守卫，拥有连锁攻击。

| 属性 | 数值 |
|------|-------|
| HP | 40 |
| 防御技能 | 10 |
| 伤害 | 4-12 |
| 攻击技能 | 12 |
| DR | 0-7 |
| EXP | 7 |
| 最高等级 | 14 |

**属性：**
- UNDEAD

**战利品：**
- 护甲（20%基础几率，每次掉落递减）

**特殊机制：**
- **连锁拉拽**：可以使用一次链条将远处的敌人（5格内）拉至相邻位置
- 链条施加4回合残废效果
- 无法对不可移动目标使用链条

**策略：**
- 尽可能保持最大距离
- 链条使用后，它们的危险性降低
- 高DR使它们很耐打
- 利用只能使用一次链条的限制

亡灵法师，会召唤骷髅手下。

| 属性 | 数值 |
|------|-------|
| HP | 40 |
| 防御技能 | 14 |
| 伤害 | 0（无近战）
| 攻击技能 | N/A |
| DR | 0-5 |
| EXP | 7 |
| 最高等级 | 14 |

**属性：**
- UNDEAD

**战利品：**
- 治疗药水（20%基础几率，有限掉落）

**特殊机制：**
- **召唤骷髅**：召唤NecroSkeleton手下（首次召唤1回合，后续2回合）
- **治疗骷髅**：对骷髅释放以治疗20%最大HP
- **强化骷髅**：如果骷髅满HP，给予Adrenaline
- **传送骷髅**：如果骷髅远离目标，重新定位骷髅

**NecroSkeleton属性：**
| 属性 | 数值 |
|------|-------|
| HP | 20 |
| HT | 25 |
| 伤害 | 2-10 |
| 防御 | 同骷髅相同 |
| EXP | 0 |

**策略：**
- 优先击杀死灵法师以摧毁关联的骷髅
- 死灵法师存活时骷髅会重生
- 对死灵法师使用爆发伤害
- 范围攻击可同时打击两者

## Boss：天狗

一位被囚禁在监狱深处的传奇刺客。位于深度10。

| 属性 | 正常 | 更强Boss |
|------|--------|-----------------|
| HP | 200 | 250 |
| 防御技能 | 15 | 15 |
| EXP | 20 | 20 |

**属性：**
- BOSS
- 视野距离：12

### 战斗属性

| 攻击类型 | 伤害 | 攻击技能 |
|-------------|--------|--------------|
| 近战 | 6-12 | 10（相邻）或20（远程） |
| DR | 0-5 | - |

**免疫：**
- 根系束缚
- 失明
- 恐惧
- 恐怖

### 阶段结构

**第一阶段（100-50% HP）：**
- 在小房间内战斗
- 使用房间周围的陷阱
- 受到伤害时跳跃（8个HP区间）

**第二阶段（50-0% HP）：**
- 传送到更大的竞技场
- 使用能力配合攻击
- 更具攻击性

### 特殊机制

**跳跃：**
- 传送到新位置
- 留下烟雾效果
- 第一阶段：留在房间内，设置陷阱
- 第二阶段：跳到战略位置

**炸弹能力：**
- 投掷带有3回合倒计时的炸弹
- 在2格半径内造成5+深度到10+深度*2伤害
- 显示倒计时："3... 2... 1..."

**火焰能力：**
- 在某个方向创建蔓延的火焰模式
- 火焰烧灼地形和角色
- 更强Boss挑战中无法使用（被额外炸弹替代）

**电击能力：**
- 放置电击装置，发射闪电
- 在基本方向和对角线模式之间交替
- 命中时造成2+深度伤害

### 能力使用

| 条件 | 能力频率 |
|-----------|-------------------|
| 正常 | 冷却后每1-4回合 |
| 进度落后 | 更快的能力使用 |
| 更强Boss | 火焰被替换，能力加倍 |

### 策略

1. **第一阶段**：避开陷阱，造成稳定伤害
2. **第二阶段**：注意能力预示
3. **炸弹**：远离倒计时区域
4. **电击器**：跟踪闪电模式
5. **火焰**：避开蔓延的火焰
6. **陷阱**：学习陷阱生成模式

### 掉落

- 天狗面具（如果未选择子职业）
- 劳埃德信标升级（如果拥有）
