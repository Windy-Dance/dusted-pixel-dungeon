# 怪物系统概述

本文档全面概述了《破碎像素地牢》中的怪物系统，包括AI状态、属性和属性公式。

## 核心怪物属性

所有怪物都共享`Mob`类中定义的一组通用基础属性：

| 属性 | 类型 | 描述 |
|----------|------|-------------|
| `HP` | int | 当前生命值 |
| `HT` | int | 最大生命值 |
| `defenseSkill` | int | 基础防御/闪避等级 |
| `EXP` | int | 击杀后获得的经验值 |
| `maxLvl` | int | 获得完整经验值的英雄最高等级 |
| `baseSpeed` | float | 移动速度倍数（默认1.0） |
| `viewDistance` | int | 视野范围（格数，默认8） |
| `flying` | boolean | 可以穿越陷阱和水域 |

## AI状态

怪物使用有限状态机进行操作，具有以下状态：

### 睡眠状态（SLEEPING）
- **行为**：怪物等待被惊扰
- **触发条件**：怪物生成，或对目标失去兴趣
- **动作**：在被噪音或视野惊扰前无任何动作
- **状态转换**：被惊扰时变为`游荡`状态

### 游荡状态（WANDERING）
- **行为**：在关卡内随机移动
- **触发条件**：默认生成状态，或失去目标后
- **动作**：随机移动，偶尔改变方向
- **状态转换**：发现敌人时变为`狩猎`状态

### 狩猎状态（HUNTING）
- **行为**：主动追击目标
- **触发条件**：敌人进入视野范围
- **动作**：向敌人移动，在射程内进行攻击
- **状态转换**：生命值危急时可切换为`逃跑`状态

### 逃跑状态（FLEEING）
- **行为**：逃离危险
- **触发条件**：生命值危急，特定怪物行为
- **动作**：远离威胁
- **状态转换**：安全时返回`游荡`状态

### 被动状态（PASSIVE）
- **行为**：无自动动作
- **触发条件**：特殊条件（被魅惑等）
- **动作**：无
- **状态转换**：取决于条件持续时间

## 怪物属性

怪物可以拥有影响游戏玩法的特殊属性：

| 属性 | 效果 |
|----------|--------|
| `BOSS` | 特殊生命条，经验值不随等级缩放 |
| `BOSS_MINION` | 由Boss召唤，特殊击杀规则 |
| `UNDEAD` | 受到反亡灵效果影响 |
| `DEMONIC` | 恶魔生物，特殊互动 |
| `INORGANIC` | 无血液，免疫流血 |
| `LARGE` | 无法通过狭窄通道 |
| `IMMOVABLE` | 无法被位移效果移动 |
| `FLYING` | 可以飞越障碍物和缺口 |
| `FIERY` | 免疫火焰，接触时造成燃烧伤害 |
| `ICY` | 免疫寒冷，接触时造成冻结效果 |
| `ELECTRIC` | 免疫闪电，接触时造成电击效果 |
| `ACIDIC` | 击中时施加腐蚀淤泥 |
| `MINIBOSS` | 具有增强属性的特殊敌人 |
| `STATIC` | 不会移动 |

## 战斗公式

### 伤害计算
```java
int damage = Random.NormalIntRange(minDamage, maxDamage);
```

### 防御计算（伤害减免）
```java
int dr = Random.NormalIntRange(minDR, maxDR);
int actualDamage = Math.max(0, damage - dr);
```

### 命中判定
```java
// 攻击命中条件：Random(attackSkill) >= Random(defenseSkill)
boolean hit = Random.Int(attackSkill) >= Random.Int(defenseSkill);
```

### 攻击技能计算
- 基础攻击技能决定命中几率
- 受距离、增益和减益影响
- 远程攻击可能使用不同的准确性

### 闪避计算
```java
int evasion = defenseSkill * (受各种效果增益);
if (paralysed) evasion = 0;
```

## 属性缩放

### 经验值公式
- 当英雄等级 <= 怪物maxLvl时，获得完整经验值
- 当英雄等级 > 怪物maxLvl时，经验值减少
- 后期区域的经验值随深度缩放

### 生命值缩放（更强Boss挑战）
某些Boss在"更强Boss"挑战下拥有更高的生命值：
- **粘咕**：100 → 120 HP
- **天狗**：200 → 250 HP
- **DM-300**：300 → 400 HP
- **矮人国王**：300 → 450 HP

### 速度修正
- `baseSpeed` 乘以移动速度
- 飞行怪物忽略地形惩罚
- 水域会减慢非飞行、非水生怪物的速度

## 特殊能力

许多怪物拥有超越基础攻击的独特能力：

### 远程攻击
- 使用`Ballistica`进行视线判定
- 可被墙壁和实体地形阻挡
- 可能有使用冷却时间

### 召唤
- 死灵法师召唤骷髅
- 矮人国王分波次召唤手下
- Yog-Dzewa召唤各种恶魔类型

### 状态效果
- **腐蚀淤泥**：随时间造成酸性伤害（粘咕）
- **燃烧**：随时间造成火焰伤害（火焰元素）
- **残废**：减少移动速度（巨蝎）
- **魅惑**：攻击盟友而非敌人（魅魔）
- **降级**：降低装备效果（矮人术士）

## 阵营系统

| 阵营 | 行为 |
|-----------|----------|
| `ENEMY` | 对英雄和盟友敌对 |
| `NEUTRAL` | 非敌对，忽略战斗 |
| `ALLY` | 与英雄并肩作战 |

## 战利品系统

### 掉落几率
- `lootChance`：基础概率（0.0到1.0）
- `loot`：掉落的物品类别
- 有限掉落会降低未来掉落率

### 掉落表
- 武器和护甲随深度缩放
- 药水排除某些类型（治疗、力量）
- 金币数量随深度缩放

## 实现说明

### 状态机模式
每个AI状态都实现为继承`Mob.State`的嵌套类。`act()`方法根据当前状态和环境确定行为。

### 回合系统
- 怪物在游戏回合中行动
- 速度影响行动频率
- 冷却时间每回合递减

### 目标选择
- `chooseEnemy()`选择有效目标
- 通过`heroFOV`检查视线
- 距离影响目标优先级
