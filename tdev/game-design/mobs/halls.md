# 大厅区域（深度21-25）

大厅是最终区域，包含最危险的普通敌人和终极Boss Yog-Dzewa。

## 普通怪物

### 魅魔

具有魅惑和闪烁能力的诱惑恶魔。

| 属性 | 数值 |
|------|-------|
| HP | 80 |
| 防御技能 | 25 |
| 伤害 | 25-30 |
| 攻击技能 | 40 |
| DR | 0-10 |
| EXP | 12 |
| 最高等级 | 25 |
| 视野距离 | 10（Light.DISTANCE） |

**属性：**
- DEMONIC

**战利品：**
- 卷轴（33%几率，排除鉴定和升级）

**特殊机制：**
- **魅惑**：33%几率命中时施加魅惑减益，持续时间为一半
- **魅惑伤害加成**：攻击被魅惑目标时，获得治疗或护盾
- **闪烁**：如果距离3格以上，可以传送到目标方向（4-6回合冷却）

**魅惑效果：**
- 被魅惑的目标可能攻击盟友
- 魅魔攻击被魅惑目标时获得5+伤害HP或护盾
- 自身对魅惑免疫

**策略：**
- 极高准确性（40）- 难以闪避
- 魅惑可以使盟友倒戈相向
- 闪烁使它们难以放风筝
- 在魅惑叠加前快速击杀

### 邪眼

漂浮的恶魔之眼，发射毁灭性的死亡射线。

| 属性 | 数值 |
|------|-------|
| HP | 100 |
| 防御技能 | 20 |
| 伤害 | 20-30（近战），30-50（死亡凝视）
| 攻击技能 | 30 |
| DR | 0-10 |
| EXP | 13 |
| 最高等级 | 26 |
| 视野距离 | 10（Light.DISTANCE） |

**属性：**
- DEMONIC
- 飞行

**战利品：**
- 总是掉落某些东西：
  - 50%露滴
  - 25%种子
  - 25%符石

**特殊机制：**
- **充能射线**：2回合充能，然后发射死亡凝视
- **死亡凝视**：穿透射线造成30-50伤害
- **伤害减免**：射线充能时受到75%更少伤害
- **冷却时间**：射线之间4-6回合

**死亡凝视属性：**
- 击中线路中的所有目标
- 可以被闪避
- 摧毁可燃地形
- 抗性：法杖解体、死亡凝视、解体陷阱

**策略：**
- 充能期间打破视线
- 射线期间的高爆发伤害降低其效果
- 射线可以击中多个目标
- 低DR使持续伤害有效

### 巨蝎

披甲恶魔，使用致残射击进行远程攻击。

| 属性 | 数值 |
|------|-------|
| HP | 110 |
| 防御技能 | 24 |
| 伤害 | 30-40 |
| 攻击技能 | 36 |
| DR | 0-16 |
| EXP | 14 |
| 最高等级 | 27 |
| 视野距离 | 10（Light.DISTANCE） |

**属性：**
- DEMONIC

**战利品：**
- 药水（50%几率，排除治疗和力量）

**特殊机制：**
- **仅远程**：无法攻击相邻目标
- **残废**：50%几率命中时施加残废
- **放风筝**：狩猎时跑开以保持距离
- **仇恨限制**：只能被它能看到的目标激怒

**策略：**
- 接近距离以防止攻击
- 极高DR（0-16）使它们很耐打
- 残废使接近距离更困难
- 利用角落和门来接近

## Boss：Yog-Dzewa

最终Boss，一个古老的恶魔之神。位于深度25。

| 属性 | 数值 |
|------|-------|
| HP | 1000 |
| 防御技能 | N/A（INFINITE_ACCURACY） |
| EXP | 50 |

**属性：**
- BOSS
- IMMOVABLE
- DEMONIC
- STATIC
- 视野距离：12

### 阶段结构

**第0阶段（休眠）：**
- 等待英雄注意到
- 不采取任何行动

**第1-4阶段（拳头阶段）：**
- 每个阶段在300 HP间隔
- 每个阶段召唤一个拳头
- 更强Boss：每个阶段召唤2个拳头

**第5阶段（最终）：**
- 所有拳头被摧毁
- 快速召唤和能力
- 增加攻击性

### Yog-Dzewa能力

**黑暗射线：**
- 目标显示红色格子（可被根系束缚打断）
- 每束射线造成20-30伤害（更强Boss为30-50）
- 满HP时1束射线，每损失400 HP增加1束射线
- 冷却时间：10-15回合，受受到伤害减少

**召唤手下：**
- 生成幼虫、YogRipper、YogEye、YogScorpio
- 冷却时间：10-15回合，受受到伤害减少
- 生成率受存活的恶魔生成器影响

### 拳头系统

**拳头类型：**

| 拳头 | 配对 | 能力 |
|------|------|---------|
| 燃烧之拳 | 大地之拳 | 火焰攻击 |
| 大地之拳 | 燃烧之拳 | 根系/植物攻击 |
| 腐烂之拳 | 锈蚀之拳 | 腐蚀/减益 |
| 锈蚀之拳 | 腐烂之拳 | 重甲/流血 |
| 辉耀之拳 | 暗影之拳 | 光明/失明 |
| 暗影之拳 | 辉耀之拳 | 黑暗/阴影 |

**拳头规则：**
- 正常：每个阶段1个拳头（总共3个）
- 更强Boss：每个阶段2个拳头（总共6个）
- 配对拳头在更强Boss中不能一起生成
- 拳头存活时Yog无敌

### 手下类型

**幼虫：**
| 属性 | 数值 |
|------|-------|
| HP | 20 |
| 防御 | 12 |
| 伤害 | 15-25 |
| 攻击技能 | 30 |
| DR | 0-4 |

**YogRipper（基于撕裂恶魔）：**
- 高伤害，快速移动
- 向目标跳跃

**YogEye（基于邪眼）：**
- 死亡凝视能力
- 比普通邪眼属性更低

**YogScorpio（基于巨蝎）：**
- 远程攻击带残废
- 比普通巨蝎属性更低

### 阶段表

| 阶段 | HP范围 | 行为 |
|-------|----------|----------|
| 0 | 1000 | 休眠，等待 |
| 1 | 700-999 | 活跃，生成第1个拳头 |
| 2 | 400-699 | 生成第2个拳头 |
| 3 | 100-399 | 生成第3个拳头 |
| 4 | 100 | 被最后的拳头保护 |
| 5 | 0-99 | 所有拳头死亡，最终决战 |

### 可见度系统

Yog-Dzewa减少关卡的视野距离：
- 第1阶段：4格
- 第2阶段：3格
- 第3阶段：2格
- 第4阶段：1格
- 第5阶段：重置为正常

黑暗挑战进一步减少此距离。

### 策略

1. **先击杀拳头** - 拳头存活时Yog无敌
2. **打破视线** - 避免射线攻击
3. **管理手下** - 它们会持续生成
4. **恶魔生成器** - 存活越多 = 更好的分数
5. **爆发伤害** - 减少冷却更快
6. **更强Boss** - 6个拳头，更多手下，更难的战斗

### 掉落

- 分数：5000 + 1250 × 存活的恶魔生成器数量
- 徽章：如果4个生成器存活则获得Boss挑战徽章（仅更强Boss）
