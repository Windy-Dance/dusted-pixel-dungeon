# HeroClass - 英雄职业类

## 概述
`HeroClass` 枚举类定义了 Shattered Pixel Dungeon 中的六个基础英雄职业。每个职业都有独特的起始装备、能力特点和游戏风格，为玩家提供多样化的游戏体验。

职业系统是游戏核心玩法的重要组成部分，影响英雄的成长路径、天赋选择和战斗策略。

## 枚举常量

### WARRIOR (战士)
- **子职业**: `Berserker` (狂战士), `Gladiator` (角斗士)
- **特点**: 高生命值、高防御、近战专家
- **起始装备**: 磨损的短剑、布甲（带破碎封印）、投掷石、食物、治疗药水、愤怒卷轴

### MAGE (法师)  
- **子职业**: `Battlemage` (战斗法师), `Warlock` (术士)
- **特点**: 魔法专精、远程攻击、法杖依赖
- **起始装备**: 魔法飞弹法杖、布甲、食物、升级卷轴、液态火焰药水

### ROGUE (盗贼)
- **子职业**: `Assassin` (刺客), `Freerunner` (自由奔跑者)  
- **特点**: 隐身能力、高暴击、敏捷灵活
- **起始装备**: 匕首、阴影披风、投掷刀、食物、魔法地图卷轴、隐身药水

### HUNTRESS (猎人)
- **子职业**: `Sniper` (狙击手), `Warden` (守护者)
- **特点**: 远程专精、自然亲和、投掷武器专家
- **起始装备**: 手套、灵能弓、食物、心灵视野药水、催眠曲卷轴

### DUELIST (决斗者)
- **子职业**: `Champion` (冠军), `Monk` (武僧)
- **特点**: 武器大师、连击系统、高机动性
- **起始装备**: 细剑、投掷钉刺、食物、力量药水、镜像卷轴

### CLERIC (牧师)
- **子职业**: `Priest` (祭司), `Paladin` (圣骑士)
- **特点**: 神圣魔法、治疗辅助、防御支援
- **起始装备**: 权杖、神圣典籍、食物、净化药水、移除诅咒卷轴

## 字段

### 核心字段
- **subClasses**: `HeroSubClass[]` - 该职业可用的子职业数组

## 构造函数

### HeroClass(HeroSubClass... subClasses)
初始化职业及其可用的子职业。

## 核心方法

### initHero(Hero hero)
初始化英雄的起始状态：
- 设置职业类型
- 初始化天赋系统
- 分配起始装备
- 设置快捷栏物品
- 根据设置自动将水袋添加到快捷栏

### masteryBadge()
返回该职业的大师成就徽章。

### title()
获取职业的本地化标题名称。

### desc()
获取职业的详细描述文本。

### shortDesc()
获取职业的简短描述文本。

### subClasses()
返回该职业可用的子职业数组。

### armorAbilities()
返回该职业可用的职业盔甲能力数组。

### spritesheet()
返回该职业的角色精灵图集路径。

### splashArt()
返回该职业的启动画面艺术图路径。

### isUnlocked()
检查该职业是否已解锁（基于成就系统）。

### unlockMsg()
获取该职业的解锁消息文本。

## 职业初始化方法

每个职业都有专门的初始化方法来设置其独特的起始装备：

### initWarrior(Hero hero)
- 装备磨损的短剑
- 添加投掷石到背包和快捷栏
- 为布甲附加破碎封印
- 识别治疗药水和愤怒卷轴

### initMage(Hero hero)
- 创建并装备魔法飞弹法杖
- 添加法杖到快捷栏
- 识别升级卷轴和液态火焰药水

### initRogue(Hero hero)
- 装备匕首
- 装备并激活阴影披风
- 添加投掷刀到背包和快捷栏
- 识别魔法地图卷轴和隐身药水

### initHuntress(Hero hero)
- 装备手套
- 添加灵能弓到背包和快捷栏
- 识别心灵视野药水和催眠曲卷轴

### initDuelist(Hero hero)
- 装备细剑并激活
- 添加2个投掷钉刺到背包和快捷栏
- 识别力量药水和镜像卷轴

### initCleric(Hero hero)
- 装备权杖并激活
- 装备并激活神圣典籍
- 添加典籍到快捷栏
- 识别净化药水和移除诅咒卷轴

## 使用示例

```java
// 创建战士英雄
Hero hero = new Hero();
HeroClass.WARRIOR.initHero(hero);

// 检查职业是否已解锁
if (HeroClass.MAGE.isUnlocked()) {
    // 允许选择法师职业
}

// 获取职业描述
String description = HeroClass.ROGUE.desc();
```

## 注意事项

1. **成就依赖**：除战士外，其他职业需要通过特定成就解锁
2. **装备差异**：每个职业的起始装备都经过精心设计以体现其特色
3. **快捷栏设置**：职业会自动配置最有用的物品到快捷栏
4. **本地化支持**：所有文本都通过 Messages 系统进行本地化
5. **扩展性**：职业系统设计为可扩展，支持自定义职业模组
6. **平衡性**：各职业在设计时考虑了游戏平衡性，各有优势和劣势
