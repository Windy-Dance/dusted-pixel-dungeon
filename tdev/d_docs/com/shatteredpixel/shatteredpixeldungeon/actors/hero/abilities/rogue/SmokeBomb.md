# 烟雾弹 (Smoke Bomb)

**类型**: 盗贼  
**基础充能消耗**: 50点  
**冷却时间**: 无固定冷却时间，依赖盔甲充能系统

## 技能效果

烟雾弹是一项逃生和战术位移技能，允许英雄传送到可见位置并产生烟雾效果。

1. **传送能力**: 将英雄传送到6格范围内的任意可见位置
2. **致盲效果**: 对相邻的敌人施加短暂致盲效果
3. **替身术**: 配合天赋可以留下一个替身木桩吸引敌人注意
4. **快速撤退**: 配合天赋可以获得加速和隐身效果

## 使用方式

- 选择地图上任意可见且可通行的位置（必须在6格范围内）
- 英雄会瞬间传送到目标位置并产生烟雾特效
- 如果处于隐身状态并拥有相应天赋，可以不消耗行动时间

## 相关天赋

- **急速撤退 (Hasty Retreat)**: 获得持续1-4回合的加速和隐身效果
- **替身术 (Body Replacement)**: 在原位置留下一个具有防御力的替身木桩（生命值和防御随天赋等级提升）
- **暗影步 (Shadow Step)**: 当处于隐身状态时，大幅减少技能充能消耗（最多50%）并保持隐身
- **英雄能量 (Heroic Energy)**: 减少技能的充能消耗（12%/23%/32%/40%）

## 注意事项

- 目标位置必须是可见且未被其他角色占据的单元格
- 如果英雄被定身，则只能传送到自身位置（主要用于触发天赋效果）
- 替身木桩对多种控制效果免疫（恐惧、恐怖、混乱、魅惑、睡眠等）
- 处于隐身状态使用技能时不会消耗行动时间
- 是盗贼重要的逃生和战术机动技能
