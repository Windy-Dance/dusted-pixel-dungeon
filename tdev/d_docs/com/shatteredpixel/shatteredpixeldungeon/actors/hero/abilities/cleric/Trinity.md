# 三位一体 (Trinity)

**类型**: 牧师  
**基础充能消耗**: 25点  
**冷却时间**: 无固定冷却时间，依赖盔甲充能系统

## 技能效果

三位一体允许牧师临时获得三种不同类型的能力：身之形态（武器/护甲附魔）、心之形态（法杖/投掷武器）和灵之形态（戒指/神器）。

1. **身之形态 (Body Form)**: 临时为当前武器或护甲添加附魔或铭文效果，持续20回合
2. **心之形态 (Mind Form)**: 临时使用已发现的法杖或投掷武器的效果，需要选择目标
3. **灵之形态 (Spirit Form)**: 临时使用已发现的戒指或神器的效果，持续20回合或激活一次

## 使用方式

- 首先通过相应的天赋（身之形态、心之形态、灵之形态）将想要的能力注入三位一体中
- 使用技能时会弹出窗口，显示所有已注入的能力选项
- 选择想要使用的能力类型，消耗相应的充能值即可激活

## 相关天赋

- **身之形态 (Body Form)**: 可以将已发现的武器附魔或护甲铭文注入三位一体
- **心之形态 (Mind Form)**: 可以将已发现的法杖或投掷武器注入三位一体  
- **灵之形态 (Spirit Form)**: 可以将已发现的戒指或神器注入三位一体
- **英雄能量 (Heroic Energy)**: 减少技能的充能消耗（12%/23%/32%/40%）

## 注意事项

- 稀有附魔/铭文和特殊物品（如火焰法杖、重生法杖等）消耗50点充能
- 普通能力消耗25点充能
- 身之形态不会覆盖已有相同类型的附魔/铭文
- 心之形态使用法杖时受魔法免疫影响
- 灵之形态中的戒指和圣杯提供被动效果，其他神器提供主动效果
- 是牧师最灵活的技能，可以适应各种战斗情况
