# 影子分身 (Shadow Clone)

**类型**: 盗贼  
**基础充能消耗**: 35点  
**冷却时间**: 无固定冷却时间，依赖盔甲充能系统

## 技能效果

影子分身召唤一个英雄的影子分身作为盟友协助战斗。

1. **分身召唤**: 在英雄周围召唤一个影子分身盟友
2. **分身能力**: 分身具有基于英雄等级的基础属性，并可通过天赋增强
3. **远程控制**: 可以指挥分身进行攻击、防御或跟随
4. **位置交换**: 配合高级天赋可以与分身交换位置

## 使用方式

- 第一次使用：在英雄周围召唤影子分身（消耗充能）
- 后续使用：可以选择目标位置指挥分身移动或执行命令（不消耗充能）
- 分身会自动跟随英雄，也可以手动指定行动

## 相关天赋

- **影刃 (Shadow Blade)**: 增加分身的攻击力（最多80%的英雄标准化伤害）并使其能够触发武器附魔效果
- **克隆护甲 (Cloned Armor)**: 增加分身的防御力（最多120%的英雄防御值）并使其能够触发护甲附魔效果
- **完美复制 (Perfect Copy)**: 增加分身的生命值（最多150点额外生命），提升交互距离，并允许与分身交换位置
- **英雄能量 (Heroic Energy)**: 减少技能的充能消耗（12%/23%/32%/40%）

## 注意事项

- 只能存在一个影子分身
- 分身对盟友增益效果免疫
- 分身移动速度较快，返回英雄身边时会以双倍速度移动
- 分身死亡后需要重新召唤
- 与分身交换位置时会打破隐身状态
