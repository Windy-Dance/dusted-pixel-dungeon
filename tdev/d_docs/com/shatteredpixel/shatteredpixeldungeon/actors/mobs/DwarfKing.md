# 矮人王 (Dwarf King)

## 概述
矮人王是《破碎的像素地牢》中的城市层Boss，是一个强大的不死族敌人。他拥有三个不同的战斗阶段，能够召唤部下、建立生命链接，并使用传送能力。矮人王战是游戏中最复杂的Boss战之一，需要玩家在不同阶段采用不同的策略。

## AI阶段

### 第一阶段 (HP > 50%)
- **基础属性**: 
  - 生命值: 普通模式300点，挑战模式450点
  - 防御技能: 22
  - 经验值: 40点
  - 属性: Boss、不死族
- **召唤机制**: 
  - 召唤冷却时间: 10-14回合(普通模式)，8-10回合(挑战模式)
  - 召唤顺序: 食尸鬼 → 僧侣/术士 → 食尸鬼 → 食尸鬼 → 僧侣/术士 → 食尸鬼...
  - 挑战模式: 每9次召唤会出现魔像(Golem)
- **特殊能力**: 
  - **生命链接**: 与最远的部下建立生命链接，共享伤害
  - **传送**: 将最远的部下传送到英雄附近
  - 能力冷却时间: 与召唤相同的冷却时间范围

### 第二阶段 (HP ≤ 50%)
- **触发条件**: 
  - 生命值降至50%以下(普通模式150点，挑战模式225点)
  - 自动传送回王座位置
- **状态变化**: 
  - 获得不可移动(IMMOVABLE)属性
  - 获得完全伤害免疫(除了特定攻击类型)
  - 获得强大的护盾(等于最大生命值)
  - 所有现有部下被清除
- **召唤模式**: 
  - **普通模式**:
    - 波次1: 连续召唤3个食尸鬼
    - 波次2: 护盾≤200时，召唤食尸鬼和僧侣/术士
    - 波次3: 护盾≤100时，大规模召唤(4个术士、4个僧侣、4个食尸鬼×2)
  - **挑战模式**:
    - 波次1: 连续召唤6个食尸鬼(分3轮，每轮2个)
    - 波次2: 护盾≤300时，召唤食尸鬼和僧侣/术士
    - 波次3: 护盾≤150时，召唤食尸鬼、僧侣/术士和魔像

### 第三阶段 (护盾耗尽)
- **触发条件**: 第二阶段的护盾完全耗尽
- **状态变化**: 
  - 失去不可移动属性
  - 失去伤害免疫
  - Boss血条开始流血(显示红色)
  - 播放最终战斗音乐
- **召唤恢复**: 恢复第一阶段的召唤机制，但召唤冷却时间更短

## 特殊能力

### 生命链接
- **机制**: 
  - 与最远的部下建立双向生命链接
  - 对任一方造成的伤害会同时影响另一方
  - 使用HealthRay视觉效果显示链接
- **战术影响**: 
  - 玩家需要同时攻击两个目标才能有效造成伤害
  - 链接可以被技能或法杖打断

### 传送能力
- **机制**: 
  - 将最远的部下传送到英雄附近
  - 矮人王自身也会轻微移动以优化位置
  - 使用Pushing动画显示传送过程
- **使用条件**: 
  - 必须有可传送的部下
  - 英雄必须是有效的目标

### 召唤系统
- **召唤位置**: 使用CityBossLevel.getSummoningPos()获取召唤位置
- **召唤延迟**: 不同类型的部下有不同的召唤延迟(3-4回合)
- **视觉效果**: 
  - 食尸鬼: 骨头粒子效果
  - 僧侣: 火焰粒子效果
  - 术士: 黑暗粒子效果
  - 魔像: 闪电粒子效果
- **失败处理**: 如果召唤位置被占用，会对占用者造成伤害并可能对矮人王造成反伤

### 王座互动
- **物品投掷**: 在王座位置时会周期性投掷周围的物品
- **死亡掉落**: 击败后会在王座附近掉落王冠

## 掉落物品

### 国王王冠
- **必然掉落**: 击败矮人王后必定掉落国王王冠(KingsCrown)
- **掉落位置**: 
  - 如果死在王座上，掉落在王座下方
  - 否则掉落在死亡位置
- **用途**: 关键任务物品，用于后续游戏进度

### 成就奖励
- **击败Boss成就**: 击败矮人王后获得
- **Boss挑战完成成就**: 如果满足徒手挑战条件获得
- **劳埃德信标升级**: 自动升级英雄持有的劳埃德信标

### 分数奖励
- **基础Boss分数**: 4000分
- **挑战分数惩罚**: 术士的攻击每次扣除400分

## 战斗策略

### 玩家策略
- **第一阶段**: 
  - 优先清理被生命链接的部下
  - 注意躲避被传送的部下
  - 利用远程攻击安全输出
- **第二阶段**: 
  - 专注于摧毁护盾
  - 利用狭窄通道限制部下数量
  - 准备应对大规模召唤
- **第三阶段**: 
  - 快速击杀，避免过多部下积累
  - 注意最后100点生命值时的绝望喊话

### Boss挑战条件
- **徒手挑战要求**: 
  - 必须徒手攻击(不使用武器)
  - 或者使用力量戒指(RingOfForce)的加成
  - 不能使用法杖、法术或其他远程攻击
- **挑战失败条件**: 
  - 使用任何非徒手攻击方式
  - 让部下对矮人王造成伤害(通过KingDamager机制)

## 技术细节

### 特殊部下类
- **DKGhoul**: 基础食尸鬼，无伙伴系统
- **DKMonk**: 僧侣变种，具有火焰抗性
- **DKWarlock**: 术士变种，攻击会扣除挑战分数
- **DKGolem**: 魔像变种，只在挑战模式出现

### 伤害处理机制
- **KingDamager**: 特殊伤害源，用于部下对矮人王的伤害
- **Viscosity延迟伤害**: 第三阶段将直接伤害转换为延迟伤害
- **挑战伤害判定**: 详细检查各种攻击类型是否符合徒手要求

### 免疫和抗性
- **免疫效果**: 通过Doom施加的伤害增益(第二阶段后)
- **抗性效果**: 标准的不死族抗性
- **特殊免疫**: 第二阶段对几乎所有伤害类型免疫

### 音频和视觉
- **挑战音效**: 每波召唤开始时播放特殊的挑战音效
- **屏幕特效**: 使用Speck.SCREAM粒子效果显示愤怒状态
- **音乐变化**: 第三阶段切换到最终战斗音乐
