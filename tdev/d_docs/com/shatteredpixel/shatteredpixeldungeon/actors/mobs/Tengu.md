# 天狗 (Tengu)

## 概述
天狗是《破碎的像素地牢》中的第二个Boss，出现在监狱层。他是一个戴着面具的神秘武士，具有远程攻击能力和多种特殊技能。天狗战分为两个主要阶段，每个阶段都有不同的战斗机制。

## AI阶段

### 第一阶段 (HP > 50%)
- **基础行为**: 
  - 生命值: 普通模式200点，挑战模式250点
  - 防御技能: 15
  - 经验值: 20点
- **移动机制**: 
  - 天狗会不断传送到房间内的不同位置
  - 传送后会在当前位置放置陷阱
  - 传送范围限制在初始战斗房间内
- **攻击模式**: 
  - 近距离(相邻): 攻击技能10，伤害6-12点
  - 远距离: 攻击技能20，使用远程武器进行攻击

### 第二阶段 (HP ≤ 50%)
- **过渡机制**: 
  - 当生命值降至50%时，天狗会说"有趣..."并传送到更大的竞技场
  - Boss血条开始流血(显示红色)
- **增强能力**:
  - 传送范围扩大到整个竞技场区域
  - 开始使用特殊能力(炸弹、火焰、电击器)
  - 传送次数限制为4次，每次传送后能力使用次数增加
- **能力使用逻辑**: 
  - 能力使用有冷却时间(1-4回合)
  - 落后于目标使用次数时会加速使用能力
  - 挑战模式下永远不会使用火焰能力

## 特殊能力

### 传送机制
- **第一阶段传送**: 
  - 在小房间内随机传送
  - 传送后在原位置生成可见的烟雾效果
  - 在新位置周围放置陷阱(填充率随生命值降低而增加)
- **第二阶段传送**: 
  - 在更大区域内传送，保持与玩家和敌人的距离(5-7格)
  - 每次传送后能力使用配额增加
  - 最多进行4次传送

### 炸弹能力
- **使用条件**: 通常是第一个使用的特殊能力
- **机制**: 
  - 向目标附近投掷炸弹
  - 炸弹会在3回合后爆炸，显示倒计时(3...2...1...)
  - 爆炸范围为半径2格的圆形区域
  - 造成5+深度到10+深度×2点伤害
- **视觉效果**: 
  - 爆炸前会有烟雾粒子效果
  - 爆炸时产生冲击波和闪光

### 火焰能力
- **使用条件**: 通常作为备用能力，特别是当无法使用其他能力时
- **机制**: 
  - 向目标方向喷射火焰
  - 火焰沿直线传播并扩散到相邻格子
  - 在路径上生成火焰区域，持续燃烧
- **效果**: 
  - 对角色施加燃烧状态
  - 点燃可燃物品和环境
  - 破坏植物和某些障碍物

### 电击器能力
- **使用条件**: 通常是第二个使用的特殊能力
- **机制**: 
  - 向目标附近投掷电击器
  - 电击器会向四个方向(横向和纵向)发射闪电
  - 交替发射对角线和直线闪电
- **效果**: 
  - 造成固定伤害(2+深度点)
  - 对英雄会造成额外的Boss挑战分数惩罚

### 挑战模式特殊机制
- **能力使用加速**: 在"更强Boss"挑战模式下：
  - 能力使用更加频繁
  - 永远不会使用火焰能力
  - 使用其他能力后会额外使用一次火焰能力(但不计入常规能力使用)

## 掉落物品

### 条件掉落
- **天狗面具**: 
  - 仅当英雄未选择任何副职业时掉落
  - 是解锁刺客副职业的关键物品

### 必然奖励
- **劳埃德信标升级**: 击败天狗后，英雄持有的劳埃德信标会自动升级
- **成就奖励**: 
  - "击败Boss"成就
  - 如果满足Boss挑战条件，获得"Boss挑战完成"成就

### 分数奖励
- 基础Boss分数: 2000分
- 每次受到特殊能力伤害: 扣除100分

## 战斗策略

### 玩家策略
- **第一阶段**: 
  - 尽量避免踩到陷阱
  - 利用远程武器安全输出
  - 注意天狗的传送模式，预测其位置
- **第二阶段**: 
  - 保持移动，避免被特殊能力击中
  - 注意炸弹的倒计时，及时远离
  - 利用电击器的固定攻击模式进行躲避

### Boss挑战条件
- 要完成Boss挑战成就，玩家需要:
  - 不使用任何法杖或元素攻击
  - 避免被任何特殊能力击中
  - 在合理的时间内击败天狗

## 技术细节
- **属性**: Boss
- **抗性**: 对根系、失明、恐惧、恐怖等控制效果免疫
- **特殊机制**: 
  - 战斗开始时会密封房间
  - 具有特殊的死亡逻辑，由PrisonBossLevel类控制
  - 视野距离为12格，比普通敌人更远
