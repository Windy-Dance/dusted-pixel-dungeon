# 尤格拳头 (Yog Fist)

## 概述
尤格拳头是《破碎的像素地牢》最终Boss尤格·德泽瓦的附属敌人，作为其战斗机制的重要组成部分。拳头们分为六种不同类型，总是成对召唤(燃烧拳/污秽拳、腐烂拳/锈蚀拳、明亮拳/黑暗拳)，每种类型都有独特的攻击模式和特殊能力。拳头的存在直接影响尤格的无敌状态。

## 基础属性

### 通用属性
- **生命值**: 300点
- **防御技能**: 20
- **经验值**: 25点
- **最大等级**: -2(不会获得经验)
- **视野距离**: 光之护符距离(Light.DISTANCE)
- **属性**: Boss、恶魔
- **状态**: 始终处于HUNTING状态
- **远程冷却**: 使用rangedCooldown变量跟踪远程攻击冷却时间

### 通用行为
- **AI逻辑**: 
  - 当英雄可见且处于WANDERING状态时，会立即切换到HUNTING状态
  - 远程攻击有8-12回合的冷却时间
  - 无敌警告: 当接近尤格时会显示无敌警告
- **伤害处理**: 
  - 对英雄造成的伤害会扣除Boss挑战分数
  - 受到的伤害会影响尤格的冷却时间(减少冷却)
- **死亡机制**: 
  - 任何拳头死亡都会触发YogDzewa.processFistDeath()
  - 最后一个拳头死亡会触发尤格的最终阶段

## 拳头类型详解

### 燃烧拳 (Burning Fist)
- **特殊属性**: 火焰(FIERY)
- **环境互动**: 
  - 站在水上会蒸发水域，变为普通地面
  - 平均每次行动蒸发1.67个水域格子
- **火焰传播**: 
  - 每回合在周围9格内传播火焰
  - 确保每个格子的火焰强度达到4点
- **远程攻击**: 
  - 蒸发目标位置的水域
  - 或对目标施加燃烧状态
  - 在目标周围9格内传播火焰
- **抗性和免疫**: 
  - 免疫: 冰冻(Frost)
  - 抗性: 雷暴云(StormCloud)、间歇泉陷阱(GeyserTrap)

### 污秽拳 (Soiled Fist)
- **环境互动**: 
  - 平均每次行动翻耕1.33个草地板
  - 在周围9格内传播草地(非翻耕地)
- **伤害减免**: 
  - 周围每有一个草地板或高草板，受到的伤害减少1/6
  - 完全被草包围时伤害减免至1/6
- **远程攻击**: 
  - 对目标施加3回合根系(Roots)效果
  - 在目标周围传播草地或翻耕地
- **特殊机制**: 
  - 可以被点燃但不受燃烧伤害
  - 主要通过环境控制来增强自身

### 腐烂拳 (Rotting Fist)
- **特殊属性**: 酸性(ACIDIC)
- **再生能力**: 
  - 站在水中每回合恢复最大生命值的1/50
  - 显示治疗特效和数值
- **伤害转换**: 
  - 将大部分直接伤害转换为流血(Bleeding)效果
  - 流血量为原始伤害的60%
  - 显示流血状态和数值
- **远程攻击**: 
  - 在目标位置生成100点强度的毒气
- **近战攻击**: 
  - 50%几率施加污泥(Ooze)效果
- **免疫**: 对毒气(ToxicGas)完全免疫

### 锈蚀拳 (Rusted Fist)
- **特殊属性**: 大型(LARGE)、无机物(INORGANIC)
- **增强攻击**: 
  - 伤害范围: 22-44点(比其他拳头更高)
- **伤害转换**: 
  - 将所有直接伤害转换为粘性延迟伤害(Viscosity.DeferedDamage)
  - 显示延迟伤害状态和数值
- **远程攻击**: 
  - 对目标施加4回合残废(Cripple)效果
- **战术定位**: 作为高伤害、高耐久的近战拳头

### 明亮拳 (Bright Fist)
- **特殊属性**: 电系(ELECTRIC)
- **特殊限制**: 无法在近战中使用远程攻击(canRangedInMelee = false)
- **无冷却机制**: 远程攻击没有冷却时间，可以连续使用
- **远程攻击**: 
  - 造成10-20点光束伤害
  - 施加失明(Blindness)效果，持续时间为基础时间的一半
  - 对英雄造成致命伤害时触发"死于敌方法术"成就
- **半血机制**: 
  - 生命值降至50%时，会使英雄失明并传送到随机位置
  - 死亡时会使英雄长时间失明
  - 屏幕闪烁白色特效

### 黑暗拳 (Dark Fist)
- **特殊限制**: 无法在近战中使用远程攻击(canRangedInMelee = false)
- **无冷却机制**: 远程攻击没有冷却时间，可以连续使用
- **远程攻击**: 
  - 造成10-20点暗影伤害
  - 削弱英雄的光之护符效果(减少50点强度)
- **半血机制**: 
  - 生命值降至50%时，移除英雄的光之护符并传送到随机位置
  - 死亡时会移除英雄的光之护符
  - 屏幕变暗特效

## 战斗策略

### 玩家应对策略
- **优先级**: 根据拳头类型调整击杀优先级
  - **高优先级**: 明亮拳、黑暗拳(影响视野)
  - **中优先级**: 燃烧拳、腐烂拳(持续伤害)
  - **低优先级**: 污秽拳、锈蚀拳(相对容易应对)
- **环境利用**: 
  - 利用水域对抗燃烧拳
  - 利用狭窄通道限制拳头移动
  - 避免在草地区域与污秽拳战斗
- **装备准备**: 
  - 毒气和酸性抗性对腐烂拳有效
  - 失明抗性对明亮拳有用
  - 光之护符对抗黑暗拳的视野剥夺

### 拳头协同效应
- **配对优势**: 互补的拳头组合会产生协同效应
  - 燃烧+污秽: 火焰vs草地的环境控制
  - 腐烂+锈蚀: 持续伤害+高爆发
  - 明亮+黑暗: 视野控制的双重打击
- **阶段推进**: 拳头的存在延缓尤格的阶段推进，但增加了战斗复杂度

## 技术细节

### 伤害源区分
- **LightBeam**: 明亮拳的远程攻击使用特殊伤害源
- **DarkBolt**: 黑暗拳的远程攻击使用特殊伤害源  
- **这种区分使抗性系统能够正确处理不同类型的魔法攻击**

### 状态管理
- **rangedCooldown**: 精确跟踪远程攻击冷却
- **invulnWarned**: 防止重复显示无敌警告
- **isNearYog()**: 动态检测是否在尤格附近

### 视觉和音频
- **粒子效果**: 每种拳头都有独特的粒子效果
  - 燃烧拳: 火焰粒子
  - 污秽拳: 叶子粒子  
  - 腐烂拳: 黑色粒子
  - 锈蚀拳: 闪电粒子
  - 明亮拳: 亮光粒子
  - 黑暗拳: 暗影粒子
- **屏幕特效**: 明亮拳和黑暗拳有特殊的屏幕闪烁效果

### 数据持久化
- **冷却时间保存**: rangedCooldown会在存档中正确保存
- **状态同步**: 拳头的状态与尤格的相位同步保存
