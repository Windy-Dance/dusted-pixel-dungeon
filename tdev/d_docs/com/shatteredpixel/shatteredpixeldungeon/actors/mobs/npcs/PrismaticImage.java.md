# PrismaticImage（棱镜分身）

## 角色概述
PrismaticImage 是 Shattered Pixel Dungeon 中由英雄的棱镜守卫能力召唤的强大盟友 NPC。与普通的镜像分身不同，棱镜分身拥有更高的生命值和更全面的能力复制机制，能够在战斗中提供持久的支援。

## 核心机制
### 生命与防御
- **生命值**：初始 HP 为 10 点，最大生命值基于英雄的 `PrismaticGuard.maxHP()` 计算
- **防御技能**：基础防御技能为 1
- **对齐方式**：设置为 `ALLY`（盟友），且标记为 `intelligentAlly`（智能盟友）
- **行动优先级**：设置为 `MOB_PRIO + 1`，确保在其他怪物之前行动

### 行为状态
- **默认状态**：处于 `HUNTING`（狩猎）状态，积极攻击敌人
- **游荡状态**：当没有敌人在视野范围内时，会自动返回英雄身边并消失

## 属性复制机制
### 攻击能力
- **伤害计算**：造成基于英雄等级的固定伤害范围（2 + 英雄等级/4 到 4 + 英雄等级/2）
- **攻击技能**：复制英雄的基础攻击技能，并受益于精准戒指加成
- **攻击效果**：攻击时会为敌人添加对棱镜分身的仇恨

### 防御能力
- **闪避计算**：基于英雄的基础闪避、闪避戒指和护甲效果，应用 50% 的加成
- **伤害减免**：完全复制英雄的伤害减免能力
- **护甲效果**：能够触发英雄护甲的特殊效果和符文

### 符文等级
- **等级继承**：继承英雄的符文等级，确保符文效果的正确触发
- **能力同步**：确保护甲相关的所有能力都与英雄保持同步

## 特殊行为机制
### 返回机制
- **自动返回**：当没有敌人在视野范围内时，会自动返回英雄身边
- **守护消失**：返回后会立即消失，并为英雄施加 `PrismaticGuard` 增益效果
- **视觉效果**：消失时会播放光效和传送音效

### 死亡动画
- **延迟死亡**：被击败后不会立即消失，而是有 5 回合的淡出动画
- **透明度渐变**：通过调整透明度实现平滑的消失效果
- **状态管理**：正确处理死亡状态和活动状态的转换

## 视觉表现
### 外观同步
- **精灵类**：使用 `PrismaticSprite` 作为视觉表现
- **护甲等级**：根据英雄当前的护甲等级动态更新外观
- **实时更新**：每次行动时检查英雄的护甲等级是否发生变化

### 死亡效果
- **麻痹状态**：死亡期间显示麻痹状态效果
- **颜色重置**：正确处理死亡动画期间的颜色变化

## 免疫特性
棱镜分身对以下效果免疫：
- **毒性气体**（ToxicGas）
- **腐蚀性气体**（CorrosiveGas）
- **燃烧**（Burning）
- **盟友增益**（AllyBuff）

## 生命周期管理
### 英雄绑定
- **ID关联**：通过 `heroID` 与召唤它的英雄进行绑定
- **存活检查**：如果绑定的英雄不存在，棱镜分身会自动消失
- **数据持久化**：英雄 ID 和死亡计时器会在存档和加载时正确保存

### 创建方法
- **duplicate(Hero hero, int HP)**：主要的初始化方法，绑定到指定英雄并设置初始 HP
- **HP 计算**：初始 HP 由调用者传入，最大 HP 基于英雄的棱镜守卫能力计算

## 技术实现
### 状态管理
- **deathTimer**：用于控制死亡后的淡出动画持续时间
- **isActive()**：重写活动状态判断，确保在死亡动画期间仍被视为活动状态
- **die() 方法**：重写死亡逻辑以支持延迟消失机制

### 数据存储
- **HEROID**：存储绑定英雄的 ID
- **TIMER**：存储死亡计时器的当前值
- **Bundle系统**：正确集成到游戏的存档系统中

## 游戏内应用
棱镜分身主要用于以下场景：
- **棱镜守卫能力**：英雄的棱镜守卫能力是召唤棱镜分身的唯一途径
- **持久战斗支援**：相比镜像分身，提供更持久的战斗支援能力
- **战术撤退**：在没有威胁时自动返回并为英雄提供增益效果
