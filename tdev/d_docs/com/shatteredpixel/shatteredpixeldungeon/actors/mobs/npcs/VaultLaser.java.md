# VaultLaser（金库激光）

## 角色概述
VaultLaser 是 Shattered Pixel Dungeon 中一种位于金库区域的防御性 NPC，表现为固定的激光发射装置。它会按照预设的方向和时间间隔发射致命的激光束，对路径上的英雄造成威胁。

## 核心机制
### 激光发射系统
- **方向控制**：通过 `laserDirs` 数组定义激光发射的方向序列
- **循环发射**：按顺序循环使用所有预设方向进行激光发射
- **多向支持**：支持单个激光器配置多个发射方向

### 冷却机制
- **射击冷却**：`curCooldown` 控制当前冷却时间
- **射击后冷却**：`afterShotCooldown` 定义射击完成后的基础冷却时间
- **连续射击**：`shotsAfterCooldown` 允许在一次冷却后连续发射多发激光

### 警告系统
- **视觉警告**：在下次射击前一回合显示红色警告标记
- **可配置**：`giveWarning` 标志控制是否启用警告功能
- **智能显示**：仅在玩家视野范围内时显示警告

## 技术特性
### 发射逻辑
- **弹道计算**：使用 `Ballistica` 类计算激光路径
- **实体阻挡**：激光会在遇到固体地形或角色时停止
- **范围限制**：激光会穿透路径上的所有格子直到被阻挡

### 音效管理
- **集体音效**：所有激光哨兵共享音效播放限制（最多每 80ms 播放一次）
- **多重音效**：同时播放射线音效和警告音效
- **实时同步**：使用 `ShatteredPixelDungeon.realTime` 确保音效同步

### 视觉效果
- **死亡射线**：使用 `Beam.DeathRay` 显示激光束的视觉效果
- **目标标记**：使用 `TargetedCell` 显示警告位置
- **精灵链接**：通过 `WardSprite.linkVisuals()` 正确显示外观

## 免疫特性
- **完全免疫**：对所有效果类型完全免疫
- **无敌状态**：无法被任何方式伤害或影响
- **被动存在**：纯粹作为环境威胁存在

## 数据持久化
### 存档数据
- **laserDirs**：保存激光发射方向数组
- **laserDirIdx**：保存当前使用的方向索引
- **冷却时间**：保存当前和射击后的冷却时间
- **射击配置**：保存连续射击次数和警告设置

### 兼容性处理
- **版本兼容**：包含对 3.3.X 版本存档的兼容处理
- **默认值**：为旧存档提供合理的默认冷却时间
- **Bundle系统**：正确集成到游戏的存档和加载机制中

## 游戏内应用
### 地图生成
- **金库房间**：主要出现在金库相关的特殊房间中
- **固定位置**：作为不可移动的防御装置存在
- **环境威胁**：增加特定区域的挑战性

### 战术考量
- **路径规划**：玩家需要观察激光模式并规划安全路径
- **时机把握**：利用冷却时间间隙快速通过危险区域
- **警告利用**：通过警告标记预测激光发射位置

## 行为特性
### 行动逻辑
- **持续行动**：每个回合都会尝试行动
- **物品投掷**：继承 NPC 的 `throwItems()` 方法，但通常不适用
- **优先级**：遵循标准的 NPC 行动优先级

### 交互机制
- **无交互**：`interact()` 方法直接返回 true，不提供任何交互选项
- **被动威胁**：纯粹作为环境威胁存在，不响应玩家互动

## 技术实现细节
### 精灵系统
- **WardSprite**：使用守卫精灵类作为基础
- **IMMOVABLE 属性**：确保不会被移动或位移
- **视觉链接**：通过 `linkVisuals()` 方法正确显示外观

### 状态管理
- **射击计数**：`shotsFired` 跟踪当前连续射击次数
- **方向循环**：自动循环回到第一个方向
- **冷却重置**：正确处理连续射击和冷却重置逻辑
