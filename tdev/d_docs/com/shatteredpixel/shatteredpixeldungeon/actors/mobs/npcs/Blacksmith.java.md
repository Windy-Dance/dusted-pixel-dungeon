# Blacksmith（铁匠）

## 角色概述
Blacksmith 是 Shattered Pixel Dungeon 中的一个重要 NPC，位于地下城的第 12-16 层之间随机生成的铁匠房中。作为游戏中的关键任务给予者，铁匠为玩家提供武器和防具强化服务，并通过完成其任务获得各种奖励。

## 任务机制
### 任务触发
- 铁匠任务在地下城深度大于 11 层时有概率生成（概率随深度增加而提高）
- 当前实现的任务类型有两种：水晶任务（CRYSTAL）和豺狼人任务（GNOLL），真菌任务（FUNGI）未完全实现
- 玩家首次与铁匠互动时会触发任务介绍

### 任务流程
1. **任务接收**：与铁匠对话后，玩家会获得一把镐子（Pickaxe）
2. **任务执行**：根据任务类型，玩家需要收集暗金（DarkGold）
   - 水晶任务：击败水晶守卫后收集暗金
   - 豺狼人任务：击败豺狼人首领后收集暗金
3. **任务完成**：返回铁匠处交还镐子和暗金，完成任务

### 奖励系统
#### 基础奖励
- **暗金奖励**：每单位暗金提供 50 点好感度，最多 2000 点
- **Boss奖励**：如果在击败 Boss 后完成任务，额外获得 1000 点好感度
- **免费镐子**：当好感度达到 2500 点时，可免费获得一把镐子

#### 强化服务
完成任务后，铁匠提供以下服务：
- **重铸（Reforge）**：重置武器的附魔
- **硬化（Harden）**：提升防具的基础防御值
- **升级（Upgrade）**：提升物品等级
- **铁匠奖励**：从预生成的四个物品中选择一个（两把武器、一件投掷武器、一件防具）

## 交互机制
### 对话系统
- **未接任务**：根据玩家职业显示不同的任务介绍文本
- **进行中**：提醒玩家当前任务目标
- **可领取奖励**：显示奖励选择界面
- **任务完成后**：显示"离我远点"的对话

### 特殊行为
- **不死身**：铁匠拥有无限闪避和伤害免疫
- **Ascension挑战**：在玩家进行 Ascension 挑战时会自动消失
- **地图标记**：在任务进行期间会在地图上显示为地标的"Troll"

## 游戏内影响
- **统计数据**：完成任务的分数会计入 `Statistics.questScores[2]`
- **存档数据**：所有任务状态和奖励信息都会保存在游戏存档中
- **重生机制**：每个游戏会话只会生成一个铁匠

## 技术细节
- **房间生成**：使用 `BlacksmithRoom` 类生成专用房间
- **奖励生成**：武器和防具的等级分布为：+0（30%）、+1（45%）、+2（20%）、+3（5%）
- **附魔概率**：30% 的基础概率获得附魔，可通过羊皮纸碎片提升概率
