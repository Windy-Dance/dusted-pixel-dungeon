# MirrorImage（镜像分身）

## 角色概述
MirrorImage 是 Shattered Pixel Dungeon 中由玩家英雄技能或物品召唤的临时盟友 NPC。它作为英雄的镜像分身，能够复制英雄的部分属性和能力，在战斗中提供强大的辅助支持。

## 核心机制
### 生命与防御
- **生命值**：固定为 1 点 HP（HP = HT = 1）
- **防御技能**：基础防御技能为 1
- **对齐方式**：设置为 `ALLY`（盟友），不会攻击玩家

### 行动优先级
- **高优先级**：行动优先级设置为 `MOB_PRIO + 1`，确保在其他怪物之前行动
- **状态管理**：默认处于 `HUNTING`（狩猎）状态，积极寻找敌人

## 属性复制机制
### 攻击能力
- **伤害计算**：造成英雄武器伤害的一半（向上取整）
- **无武器情况**：如果英雄未装备武器，则使用英雄的基础伤害（包含力量戒指加成）
- **攻击技能**：复制英雄的基础攻击技能，并受益于精准戒指和武器加成

### 防御能力
- **闪避计算**：基于英雄的基础闪避和闪避戒指效果，应用 50% 的加成
- **伤害减免**：如果英雄装备了武器，会获得武器防御因子一半的额外伤害减免

### 特殊能力复制
- **圣化武器**：当英雄未装备武器但拥有圣化武器能力时，镜像分身也能触发该效果
- **身体形态**：复制英雄的身体形态附魔效果
- **攻击延迟**：复制英雄的攻击延迟（受狂怒戒指影响）

## 视觉表现
### 外观同步
- **精灵类**：使用 `MirrorSprite` 作为视觉表现
- **护甲等级**：根据英雄当前的护甲等级动态更新外观
- **隐身效果**：初始状态下处于隐身状态（MirrorInvis），首次攻击后解除

### 护甲更新
- **实时同步**：每次行动时检查英雄的护甲等级是否发生变化
- **视觉反馈**：护甲外观会随着英雄护甲等级的变化而实时更新

## 战斗行为
### 攻击范围
- **扩展攻击**：除了标准的相邻格攻击外，还能使用英雄武器的特殊攻击范围
- **武器效果**：能够触发武器的特殊效果和附魔

### 敌人交互
- **仇恨生成**：攻击敌人时会为其添加对镜像分身的仇恨
- **死亡处理**：被击败时会正常触发死亡事件和相关效果

## 免疫特性
镜像分身对以下效果免疫：
- **毒性气体**（ToxicGas）
- **腐蚀性气体**（CorrosiveGas）
- **燃烧**（Burning）
- **盟友增益**（AllyBuff）

## 生命周期管理
### 英雄绑定
- **ID关联**：通过 `heroID` 与召唤它的英雄进行绑定
- **存活检查**：如果绑定的英雄不存在，镜像分身会自动消失
- **数据持久化**：英雄 ID 会在存档和加载时正确保存

### 隐身机制
- **初始隐身**：创建时立即应用 `MirrorInvis` 隐身效果
- **攻击解除**：首次攻击敌人时自动解除隐身状态
- **无图标显示**：隐身效果不会在 UI 上显示图标

## 技术实现
### 创建方法
- **duplicate(Hero hero)**：主要的初始化方法，绑定到指定英雄并应用隐身效果
- **sprite() 方法**：重写精灵创建逻辑以确保正确的护甲等级同步

### 数据存储
- **HEROID**：存储绑定英雄的 ID，用于跨回合的英雄引用
- **Bundle系统**：正确集成到游戏的存档系统中

## 游戏内应用
镜像分身主要用于以下场景：
- **战士技能**：某些战士职业的特殊技能可以召唤镜像分身
- **法术效果**：特定法术或卷轴可能产生镜像分身效果
- **临时支援**：作为战斗中的临时强力盟友提供支援
