# Imp（小恶魔）

## 角色概述
Imp 是 Shattered Pixel Dungeon 中一个狡猾的小恶魔 NPC，出现在地下城第 17-19 层的野心小恶魔房间中。作为游戏后期的重要任务给予者，小恶魔会请求玩家帮助它收集矮人代币，并提供强大的戒指作为奖励。

## 任务机制
### 任务触发
- 小恶魔任务在地下城深度 17-19 层有概率生成（概率随深度增加而提高）
- 每个游戏会话只会生成一个小恶魔
- 使用 `AmbitiousImpRoom` 生成专用的房间

### 任务类型
根据地下城深度不同，小恶魔会分配不同的收集目标：
- **第 17 层**：收集僧侣（Monk）掉落的矮人代币
- **第 18 层**：50% 概率为僧侣任务，50% 概率为魔像任务
- **第 19 层**：收集魔像（Golem）掉落的矮人代币

### 任务流程
1. **初始对话**：与小恶魔互动后接受任务，了解需要收集的目标
2. **代币收集**：击败指定的敌人（僧侣或魔像）获得矮人代币
3. **奖励兑换**：收集足够代币后返回小恶魔处兑换奖励

## 奖励系统
### 代币要求
- **标准任务**：需要收集 5 个矮人代币
- **替代任务**：需要收集 4 个矮人代币（仅适用于僧侣任务）

### 奖励物品
- **随机戒指**：从所有非诅咒戒指中随机选择一个
- **强化等级**：奖励戒指会自动升级 +2 级
- **特殊处理**：奖励戒指会被标记为诅咒状态，但在实际使用中不会产生负面效果

## 交互机制
### 对话系统
- **未接任务**：显示任务介绍和目标说明
- **进行中**：根据收集进度显示不同的提醒对话
- **可兑换**：显示奖励兑换界面（WndImp）

### 特殊行为
- **不死身**：小恶魔拥有无限闪避和伤害免疫
- **首次见面**：当玩家首次看到小恶魔时会大喊"Hey [玩家名]!"
- **逃跑机制**：完成任务后会大喊"Cya [玩家名]!"然后消失
- **Ascension挑战**：在玩家进行 Ascension 挑战时不会出现

## 游戏内影响
- **统计数据**：完成任务获得 4000 分，计入 `Statistics.questScores[3]`
- **地图标记**：在任务期间会在地图上显示为"IMP"地标
- **存档数据**：所有任务状态和奖励信息都会保存在游戏存档中

## 技术细节
### 敌人掉落
- 通过 `process(Mob mob)` 方法监听指定敌人的死亡事件
- 当符合条件的敌人被击败时，自动在敌人位置生成矮人代币
- 掉落机制确保玩家能够正常收集到所需物品

### 奖励生成
- 戒指通过 `Generator.random(Generator.Category.RING)` 随机生成
- 循环确保生成的戒指不是诅咒状态
- 生成后立即升级 +2 级并标记为诅咒（用于任务完成判断）

### 状态管理
- **alternative**：标记是否为替代任务（僧侣 vs 魔像）
- **spawned**：标记小恶魔是否已生成
- **given**：标记任务是否已接受
- **completed**：标记任务是否已完成
- **reward**：存储奖励戒指的引用
