# Ghost（幽灵）

## 角色概述
Ghost 是 Shattered Pixel Dungeon 中一个悲伤的幽灵 NPC，出现在地下城第 2-4 层的下水道区域。作为游戏中的第一个任务给予者，幽灵会请求玩家帮助它复仇，并提供丰厚的奖励作为回报。

## 任务机制
### 任务触发
- 幽灵任务在地下城深度 2-4 层有概率生成（概率随深度增加而降低）
- 每个游戏会话只会生成一个幽灵
- 幽灵会固定在房间的出口区域，无法移动到堆叠物品或关卡出口上

### 任务类型
根据地下城深度不同，幽灵会分配不同的复仇目标：
- **第 2 层**：腐臭鼠（Fetid Rat）
- **第 3 层**：豺狼人骗子（Gnoll Trickster）  
- **第 4 层**：巨蟹（Great Crab）

### 任务流程
1. **初始对话**：玩家与幽灵互动后，会在随机位置生成对应的任务 Boss
2. **Boss 战斗**：击败 Boss 后，玩家会收到"找到我"的提示
3. **奖励领取**：返回幽灵处领取武器和防具奖励

## 奖励系统
### 奖励物品
- **武器**：随机生成一件武器，品质分布为：2级（50%）、3级（30%）、4级（15%）、5级（5%）
- **防具**：随机生成一件防具，品质分布同样为：2级（50%）、3级（30%）、4级（15%）、5级（5%）
- **等级强化**：物品等级分布为：+0（50%）、+1（30%）、+2（15%）、+3（5%）

### 附魔机制
- **基础概率**：20% 的基础概率获得附魔
- **羊皮纸增益**：可通过羊皮纸碎片（ParchmentScrap）提升附魔概率
- **独立生成**：武器附魔和防具符文分别独立生成

## 交互机制
### 对话系统
- **未接任务**：显示对应的复仇请求对话（根据 Boss 类型不同）
- **任务进行中**：提醒玩家需要击败指定的 Boss
- **可领取奖励**：显示奖励选择界面（WndSadGhost）

### 特殊行为
- **不死身**：幽灵拥有无限闪避和伤害免疫
- **慢速移动**：移动速度为正常的一半（0.5f）
- **音乐效果**：与幽灵互动时会播放特殊的幽灵音效并淡出背景音乐
- **Ascension挑战**：在玩家进行 Ascension 挑战时会自动消失

## 游戏内影响
- **统计数据**：完成任务获得 1000 分，计入 `Statistics.questScores[0]`
- **地图标记**：在任务期间会在地图上显示为"GHOST"地标
- **存档数据**：所有任务状态和奖励信息都会保存在游戏存档中

## 技术细节
### 房间生成
- 使用标准的下水道房间生成机制
- 幽灵位置通过 `room.random()` 方法随机选择
- 确保生成位置不与固体地形、开放空间限制或关卡出口冲突

### 奖励生成
- 武器和防具通过 `Generator.random()` 方法生成
- 所有奖励物品都会清除起始属性（等级0、无附魔、未诅咒）
- 然后根据概率分布进行升级和附魔处理

### 状态管理
- **spawned**：标记幽灵是否已生成
- **given**：标记任务是否已接受
- **processed**：标记 Boss 是否已被击败
- **completed**：标记奖励是否已领取完毕
