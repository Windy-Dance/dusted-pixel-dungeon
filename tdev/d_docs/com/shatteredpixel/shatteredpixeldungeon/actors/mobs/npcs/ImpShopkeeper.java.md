# ImpShopkeeper（小恶魔商人）

## 角色概述
ImpShopkeeper 是 Shattered Pixel Dungeon 中的一个特殊商人 NPC，本质上是标准 Shopkeeper（商人）的变体，但使用小恶魔的外观和特定的对话文本。它出现在地下城的固定商店位置，为玩家提供商品交易服务。

## 核心特性
### 外观与行为
- **精灵类**：使用 `ImpSprite` 作为视觉表现，呈现小恶魔的形象
- **基础行为**：继承自标准 `Shopkeeper` 类，具有完整的商人功能
- **不可移动**：拥有 `IMMOVABLE` 属性，固定在商店位置

### 对话系统
- **首次见面**：当玩家首次在视野范围内看到小恶魔商人时，会主动打招呼
- **Ascension模式**：如果玩家正在进行 Ascension 挑战，会显示特殊的对话文本
- **标准交互**：提供与普通商人相同的购买、出售和对话选项

## 交互机制
### 商人功能
- **商品交易**：提供标准的商品买卖功能
- **回购系统**：支持最多 3 个物品的回购历史
- **价格计算**：使用标准的商人定价公式（物品价值 × 5 × (深度/5 + 1)）

### 特殊对话
小恶魔商人有两套不同的问候语：
- **普通模式**：`Messages.get(this, "greetings", Messages.titleCase(Dungeon.hero.name()))`
- **Ascension模式**：`Messages.get(this, "greetings_ascent", Messages.titleCase(Dungeon.hero.name()))`

### 安全机制
- **伤害处理**：如果对小恶魔商人造成伤害或施加负面状态，会触发警告机制
- **逃跑机制**：在受到持续伤害后会逃跑并清空商店物品
- **免疫能力**：对所有负面效果免疫（通过父类 Shopkeeper 实现）

## 游戏内定位
### 出现场景
- 主要出现在地下城的标准商店房间中
- 作为普通商人的视觉变体存在
- 不涉及特殊的任务或剧情

### 与普通商人的区别
- **视觉差异**：使用小恶魔的外观而非普通商人的外观
- **对话差异**：拥有独特的首次见面对话文本
- **功能相同**：所有核心商人功能与普通商人完全一致

## 技术实现
### 继承关系
- 继承自 `Shopkeeper` 基类
- 覆盖了 `spriteClass` 属性以使用小恶魔精灵
- 添加了 `seenBefore` 标志来跟踪是否已经向玩家打招呼

### 状态管理
- **seenBefore**：布尔标志，确保只在首次见面时打招呼
- **act() 方法**：重写了行动逻辑以包含打招呼功能
- **数据持久化**：继承父类的所有存档和加载机制

## 游戏体验影响
- **视觉多样性**：为商店体验增加视觉变化
- **角色个性**：通过独特的对话文本展现小恶魔的性格特征
- **功能一致性**：确保玩家获得与普通商人相同的完整交易体验
