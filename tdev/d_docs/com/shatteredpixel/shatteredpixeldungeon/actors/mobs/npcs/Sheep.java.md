# Sheep（绵羊）

## 角色概述
Sheep 是 Shattered Pixel Dungeon 中一个临时性的 NPC，主要由羊毛炸弹（Woolly Bomb）或其他相关能力召唤产生。作为游戏中最简单的 NPC 之一，绵羊主要用于提供短暂的互动体验和特殊效果。

## 核心特性
### 生命周期
- **临时存在**：绵羊具有固定的生命周期，在行动时会自动消失
- **生命值**：HP 被强制设置为 0，确保在行动回合立即死亡
- **消失机制**：在 `act()` 方法中直接调用 `destroy()` 和 `sprite.die()`

### 外观与行为
- **精灵类**：使用 `SheepSprite` 作为视觉表现
- **互动反馈**：与绵羊互动时会显示不同的"咩"声文本
- **音效播放**：互动时会播放绵羊叫声，并带有随机音高变化

## 交互机制
### 对话系统
绵羊拥有四种不同的互动文本，通过随机选择显示：
- "Baa!"（兴奋的咩叫）
- "Baa?"（疑问的咩叫）
- "Baa."（平淡的咩叫）
- "Baa..."（拖长的咩叫）

### 特殊功能
- **图鉴记录**：任何互动都会将绵羊添加到怪物图鉴中
- **羊毛炸弹互动**：由羊毛炸弹召唤的绵羊可以通过互动提前消散
- **状态显示**：互动时会在绵羊上方显示相应的对话文本

## 技术实现
### 生命周期管理
- **lifespan 属性**：存储绵羊的预期生命周期
- **initialize() 方法**：初始化生命周期并设置行动时间
- **随机偏移**：实际生命周期会有 -2 到 +2 回合的随机偏移

### 数据持久化
- **LIFESPAN**：在存档中保存生命周期信息
- **Bundle 系统**：正确集成到游戏的存档和加载机制中

### 行动逻辑
- **立即消失**：在 `act()` 方法中立即销毁自身
- **图鉴更新**：如果在英雄视野范围内，会自动添加到图鉴
- **无战斗能力**：没有攻击或防御能力，纯粹的装饰性 NPC

## 游戏内应用
### 召唤来源
- **羊毛炸弹**：主要的召唤来源，爆炸后生成绵羊
- **其他能力**：可能由其他特殊物品或能力召唤

### 游戏体验
- **趣味元素**：为游戏增添轻松有趣的元素
- **图鉴收集**：作为怪物图鉴的一部分供玩家收集
- **音效体验**：提供独特的听觉反馈和互动体验

## 特殊机制
### 羊毛炸弹绵羊
- **可消散**：由羊毛炸弹召唤的绵羊（生命周期 ≥ 20 回合）可以通过互动提前消散
- **冷却重置**：消散时会重置冷却时间，允许立即进行下一次行动

### 随机性设计
- **文本随机**：每次互动显示不同的咩叫文本
- **音高随机**：绵羊叫声的音高会有随机变化（0.91f 到 1.1f 之间）

## 免疫特性
- **伤害免疫**：对所有类型的伤害完全免疫
- **状态免疫**：无法被施加任何增益或减益效果
- **被动存在**：纯粹作为装饰性 NPC 存在，不影响战斗平衡
