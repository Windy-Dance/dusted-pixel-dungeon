# AlchemyScene 类文档

## 概述
`AlchemyScene` 是 Shattered Pixel Dungeon 的炼金场景类，继承自 `PixelScene`。它提供玩家进行物品合成、配方查看和资源管理的界面，是游戏中重要的制作系统。

## 类结构
```java
public class AlchemyScene extends PixelScene
```

## 主要功能

### 1. 配方管理系统
- **配方显示**: 显示所有已知的合成配方
- **配方解锁**: 根据游戏进度和收集情况动态解锁新配方
- **配方分类**: 按类型（药水、卷轴、装备等）对配方进行分类
- **配方详情**: 点击配方显示详细的材料需求和产出说明

### 2. 合成操作界面
- **材料选择**: 从背包中选择合成所需的材料
- **合成预览**: 显示合成结果的预览效果
- **批量合成**: 支持一次性合成多个相同物品
- **材料验证**: 自动验证是否有足够材料进行合成

### 3. 资源管理
- **能量水晶**: 显示当前拥有的能量水晶数量（合成货币）
- **液体金属**: 显示液体金属资源（用于装备升级）
- **催化剂**: 管理各种催化剂物品（三音节催化剂等）
- **库存同步**: 实时同步背包中的可用材料

### 4. 特殊物品支持
- **炼金工具包**: 支持炼金术士工具包的特殊功能
- **神秘种子**: 处理神秘种子的特殊合成选项
- **符文系统**: 集成符文和神器的制作系统
- **升级系统**: 支持武器和装备的升级合成

## UI设计特点

### 响应式布局
- **多列显示**: 横屏模式显示多列配方，竖屏显示单列
- **滚动支持**: 使用滚动面板处理大量配方的显示
- **安全区域适配**: 自动处理刘海屏和圆角屏的安全区域

### 视觉反馈
- **合成动画**: 成功合成时的粒子特效和动画
- **材料高亮**: 可用材料的视觉高亮显示
- **禁用状态**: 不可用配方的灰色显示
- **成功提示**: 合成成功后的确认提示

### 交互设计
- **拖拽操作**: 支持拖拽材料到合成槽
- **点击选择**: 点击快速选择材料和配方
- **批量操作**: 长按或右键进行批量操作
- **快捷键支持**: 键盘快捷键（桌面平台）

## 技术架构

### 配方系统
- **Recipe基类**: 所有配方继承自Recipe基类
- **动态配方**: 支持运行时生成的动态配方
- **条件配方**: 根据游戏状态条件性显示的配方
- **配方缓存**: 配方数据的高效缓存机制

### 物品系统集成
- **Item类深度集成**: 与核心物品系统紧密集成
- **Bag系统**: 支持各种背包类型的物品管理
- **Belongings**: 与英雄物品系统的同步
- **Catalog**: 与物品图鉴系统的数据共享

### 粒子效果系统
- **SparkParticle**: 合成时的火花粒子效果
- **Speck**: 各种装饰性粒子效果
- **性能优化**: 限制同时存在的粒子数量

## 生命周期方法

### create()
场景初始化：
1. 创建背景和基础UI元素
2. 加载所有可用配方
3. 初始化材料选择和合成界面
4. 设置资源显示和状态同步
5. 添加退出按钮和导航控件
6. 执行淡入动画

### update()
每帧更新逻辑：
- 处理输入事件和用户交互
- 更新材料可用性状态
- 管理粒子效果动画
- 同步背包变化

### destroy()
场景销毁时清理粒子系统和临时资源

## 数据管理

### 配方数据
- **静态配方**: 预定义的固定配方
- **动态配方**: 根据游戏状态生成的配方
- **隐藏配方**: 未解锁但存在的配方数据
- **配方进度**: 玩家配方发现进度跟踪

### 资源跟踪
- **能量消耗**: 合成消耗的能量水晶计算
- **材料消耗**: 材料使用量的精确计算
- **产出计算**: 合成结果的数量和质量计算
- **成功率**: 某些合成的成功率计算

### 持久化
- **配方发现**: 已发现配方的持久化存储
- **设置保存**: 用户偏好设置的保存
- **统计更新**: 合成相关的统计数据更新

## 用户体验设计

### 学习曲线
- **新手引导**: 初次使用时的简单引导
- **配方提示**: 未发现但可制作的配方提示
- **渐进复杂**: 从简单到复杂的配方解锁顺序

### 效率优化
- **快速合成**: 常用配方的快捷访问
- **批量操作**: 大量材料的高效处理
- **搜索功能**: 配方的快速查找

### 反馈机制
- **错误提示**: 材料不足或条件不满足的明确提示
- **成功确认**: 合成成功的视觉和音频反馈
- **进度显示**: 复杂合成的进度指示

## 特殊功能

### 炼金术士职业支持
- **职业特化**: 炼金术士职业的特殊配方和能力
- **技能加成**: 职业相关技能对合成的影响
- **专属内容**: 职业独有的合成选项

### 挑战模式兼容
- **挑战限制**: 某些挑战模式下的合成限制
- **平衡调整**: 挑战模式下的配方平衡
- **特殊规则**: 挑战特有的合成规则

### 调试和测试
- **开发者模式**: 快速测试合成系统的调试功能
- **配方编辑**: 开发阶段的配方快速修改
- **资源生成**: 测试用的资源生成工具

## 性能优化

### 内存管理
- **按需加载**: 配方详情按需加载
- **资源缓存**: 常用资源的有效缓存
- **垃圾回收**: 及时清理不再需要的对象

### 渲染优化
- **批量渲染**: 多个相似元素的批量渲染
- **视锥剔除**: 不可见元素的渲染跳过
- **LOD系统**: 根据距离调整细节级别
