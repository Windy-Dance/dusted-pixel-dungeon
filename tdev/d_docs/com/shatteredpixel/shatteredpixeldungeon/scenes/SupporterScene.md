# SupporterScene 类文档

## 概述
`SupporterScene` 是 Shattered Pixel Dungeon 的支持者场景类，继承自 `PixelScene`。它提供玩家支持游戏开发者的途径，包括捐赠、购买付费版本或成为Patreon支持者等选项。

## 类结构
```java
public class SupporterScene extends PixelScene
```

## 主要功能

### 1. 支持选项展示
- **多平台支持**: 显示不同平台的支持选项（Steam、Google Play、iOS、Patreon等）
- **平台适配**: 根据当前运行平台显示相关的支持选项
- **视觉突出**: 使用金色主题和特殊图标强调支持的重要性

### 2. 外部链接集成
- **商店链接**: 直接跳转到各平台的商店页面
- **Patreon链接**: 连接到开发者的Patreon页面
- **网站链接**: 链接到官方游戏网站
- **安全处理**: 确保外部链接的安全性和可靠性

### 3. 用户激励
- **感谢信息**: 向现有支持者显示感谢消息
- **特权说明**: 说明支持者获得的特殊权益（如果有）
- **社区贡献**: 强调支持对游戏发展的积极影响

### 4. 平台特定功能
- **移动端**: 链接到应用商店购买完整版本
- **桌面端**: 链接到Steam或GOG购买
- **网页端**: 链接到Patreon或直接捐赠

## UI设计特点

### 视觉主题
- **金色配色**: 使用金色作为主要强调色，象征价值和支持
- **特殊图标**: 使用金币图标 (`Icons.GOLD`) 作为支持标识
- **标题样式**: 使用特殊的标题颜色 (`Window.TITLE_COLOR`)

### 布局设计
- **垂直排列**: 支持选项按垂直列表排列
- **间距优化**: 使用固定的按钮高度和间距确保一致性
- **响应式设计**: 适配不同屏幕尺寸和方向

### 交互反馈
- **点击效果**: 按钮点击时的视觉反馈
- **加载状态**: 外部链接打开时的状态提示
- **成功确认**: 支持操作完成后的确认信息

## 技术实现

### 平台检测
- **自动识别**: 检测当前运行的平台（Android、iOS、Desktop等）
- **条件显示**: 只显示当前平台相关的支持选项
- **后备方案**: 提供通用的支持选项作为后备

### 外部调用
- **安全链接**: 使用平台安全的外部URL调用机制
- **错误处理**: 处理链接打开失败的情况
- **用户确认**: 在打开外部链接前可能显示确认对话框

### 本地化支持
- **多语言文本**: 所有支持选项文本支持多语言
- **文化适配**: 根据地区调整支持选项的优先级
- **货币显示**: 如果涉及价格，正确显示本地货币

## 生命周期方法

### create()
场景初始化：
1. 创建标题背景和基础UI元素
2. 根据平台创建相应的支持选项按钮
3. 设置按钮点击事件处理外部链接
4. 添加退出按钮返回主菜单
5. 执行淡入动画

### destroy()
场景销毁时清理外部链接相关资源

## 用户体验考虑

### 透明度
- **清晰说明**: 明确说明每种支持方式的具体内容
- **无压力设计**: 不强制或过度推销支持选项
- **价值体现**: 清晰传达支持对游戏开发的价值

### 便利性
- **一键访问**: 支持选项直接链接到目标页面
- **快速返回**: 容易返回到游戏主界面
- **离线可用**: 场景本身可以在离线状态下访问

### 社区建设
- **感谢文化**: 营造感谢支持者的社区氛围
- **透明开发**: 说明支持资金如何用于游戏开发
- **长期关系**: 鼓励长期的社区支持关系

## 特殊注意事项

### 法律合规
- **平台政策**: 遵守各应用商店的政策要求
- **隐私保护**: 不收集用户的支付或个人信息
- **年龄适宜**: 确保内容适合所有年龄段玩家

### 技术限制
- **网络依赖**: 外部链接需要网络连接
- **平台差异**: 不同平台的功能可能有所不同
- **维护成本**: 需要定期更新链接和选项
