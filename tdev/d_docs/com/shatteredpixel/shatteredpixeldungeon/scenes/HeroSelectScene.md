# HeroSelectScene 类文档

## 概述
`HeroSelectScene` 是 Shattered Pixel Dungeon 的英雄选择场景类，继承自 `PixelScene`。它提供玩家在开始新游戏前选择英雄职业、配置挑战模式、设置自定义种子等高级选项的界面。

## 类结构
```java
public class HeroSelectScene extends PixelScene
```

## 主要功能

### 1. 英雄职业选择
- **职业展示**: 显示所有可用的英雄职业（战士、法师、游侠、盗贼等）
- **动态预览**: 根据选择的职业实时更新界面和描述
- **随机选择**: 提供随机英雄按钮，自动选择一个职业
- **详细信息**: 点击信息按钮显示职业详细说明窗口

### 2. 挑战模式配置
- **挑战列表**: 显示所有可用的挑战模式选项
- **挑战描述**: 每个挑战都有详细的说明文本
- **难度标识**: 不同挑战有不同的难度等级标识
- **随机挑战**: 提供随机挑战按钮

### 3. 高级游戏设置
**自定义种子系统**:
- 支持输入自定义地牢种子
- 自动生成基于时间戳的种子（格式：YYYYMMDDHHMM）
- 种子验证和错误处理
- 种子历史记录查看

**每日挑战支持**:
- 专门的每日挑战模式入口
- 自动获取官方每日挑战种子
- 重播功能支持

### 4. 游戏启动流程
- **存档槽位确认**: 确认使用哪个存档槽位
- **最终确认**: 显示启动前的最终确认对话框
- **错误处理**: 处理各种启动异常情况
- **无缝过渡**: 平滑过渡到 InterlevelScene

### 5. UI设计特点
- **响应式布局**: 横竖屏完全不同的UI布局方案
- **渐变背景**: 使用深色渐变背景突出内容
- **淡入淡出效果**: 职业选择时的视觉反馈动画
- **安全区域适配**: 自动处理刘海屏和圆角屏

## 内部组件

### GameOptions (游戏选项面板)
私有内部类，管理所有游戏设置选项：
- 挑战模式复选框
- 自定义种子输入框
- 每日挑战开关
- 各种滑块和按钮控件
- 动态显示/隐藏逻辑

### ClassButton (职业按钮)
私有内部类，专门用于英雄职业选择：
- 职业图标显示
- 选中状态高亮
- 点击事件处理
- 布局管理

### ChallengesPane (挑战面板)
私有内部类，显示挑战模式选项：
- 挑战标题和描述
- 难度等级标识
- 解锁状态显示
- 复选框交互

## 生命周期方法

### create()
场景初始化的核心方法：
1. 重置英雄实例
2. 加载全局数据（徽章、日志）
3. 创建背景和视觉元素
4. 初始化标题文本
5. 创建英雄选择按钮
6. 设置选项面板和控制按钮
7. 处理初始状态（恢复之前的选择）
8. 执行淡入动画

### update()
每帧更新逻辑：
- 处理UI淡入淡出效果
- 更新各种动态元素
- 处理输入事件
- 管理选项面板状态

### onBackPressed()
返回键处理：
- 如果选项面板打开，先关闭选项面板
- 否则返回到 StartScene

## 数据管理

### 状态持久化
- `heroWasRandomized`: 记录英雄是否为随机选择
- `chalWasRandomized`: 记录挑战是否为随机选择
- 自动保存和恢复用户选择

### 种子生成
- 时间戳种子：`SimpleDateFormat("yyyyMMddHHmm")`
- 随机种子：使用 `Random.Long()`
- 种子验证：确保种子格式正确

### 挑战解锁
- 根据玩家进度自动解锁挑战
- 显示未解锁挑战的锁定状态
- 提供挑战完成条件提示

## 用户体验设计

### 信息层次
- **主要操作**: 英雄选择和开始游戏最为突出
- **次要选项**: 挑战和种子设置可折叠隐藏
- **辅助信息**: 详细说明通过单独窗口显示

### 交互反馈
- **视觉反馈**: 选择时的颜色变化和动画
- **音频反馈**: 按钮点击音效
- **错误提示**: 无效输入时的明确提示

### 可访问性
- **触摸友好**: 足够大的按钮和触摸区域
- **键盘支持**: 完整的键盘导航支持
- **控制器支持**: 游戏手柄完全兼容

## 特殊功能

### 随机系统
- 真正的随机职业选择（考虑已解锁状态）
- 随机挑战组合（避免不可能的组合）
- 智能种子生成（避免重复）

### 验证机制
- 输入验证（种子格式、挑战组合）
- 状态验证（确保游戏可以正常启动）
- 错误恢复（异常情况下的优雅降级）

## 性能优化
- **延迟加载**: 选项面板按需创建
- **内存管理**: 正确清理临时资源
- **渲染优化**: 使用高效的图像缓存
