# NewsScene 类文档

## 概述
`NewsScene` 是 Shattered Pixel Dungeon 的新闻场景类，继承自 `PixelScene`。它显示开发者博客文章、游戏更新公告和社区新闻等内容。

## 类结构
```java
public class NewsScene extends PixelScene
```

## 主要功能

### 1. 新闻文章展示
- **文章列表**: 显示所有可用的新闻文章
- **时间排序**: 按发布日期倒序排列（最新在前）
- **标题预览**: 显示文章标题和简短摘要
- **已读状态**: 标记已阅读的文章

### 2. 文章详情查看
- **完整内容**: 点击文章标题显示完整内容窗口 (`WndTitledMessage`)
- **格式支持**: 支持基本的文本格式化
- **链接处理**: 处理文章中的外部链接（如果平台支持）

### 3. 阅读状态管理
- **自动标记**: 查看文章后自动标记为已读
- **时间戳记录**: 记录最后阅读时间到设置中
- **未读提示**: 在主菜单的新闻按钮上显示未读计数

### 4. 网络服务集成
- **后台获取**: 通过 `News` 服务类获取文章数据
- **离线缓存**: 支持离线查看已下载的文章
- **自动检查**: 定期检查新文章（可配置）

## 内部组件

### ArticleButton (文章按钮)
私有内部类，用于显示单个新闻文章：
- 包含标题文本和日期信息
- 已读文章显示正常颜色，未读文章显示高亮颜色
- 处理点击事件显示文章详情

### ArticlesList (文章列表)
私有内部类，管理所有文章按钮的布局和滚动：
- 垂直滚动列表布局
- 自动计算列表高度
- 处理边界情况（空列表、长列表等）

## 生命周期方法

### create()
场景初始化：
1. 创建标题背景和UI元素
2. 加载新闻文章数据
3. 创建文章列表和按钮
4. 添加退出按钮和返回导航
5. 执行淡入动画

### update()
每帧更新（主要用于动态效果）：
- 未读文章的脉冲动画效果
- 各种UI元素的状态更新

## 数据处理

### 文章对象
使用 `NewsArticle` 对象存储文章数据：
- `title`: 文章标题
- `date`: 发布日期
- `desc`: 文章摘要或完整内容
- `body`: 完整文章内容（可能为null）

### 语言支持
- 自动适配当前游戏语言
- 英文文章作为默认后备
- 支持多语言文章（如果可用）

### 时间格式化
- 显示相对时间（"X天前"）或具体日期
- 根据文章年龄自动选择格式

## 用户体验设计

### 阅读体验
- **清晰排版**: 标题和日期清晰区分
- **重点突出**: 未读文章获得视觉强调
- **流畅导航**: 从列表到详情的平滑过渡

### 性能考虑
- **按需加载**: 文章详情按需创建
- **内存优化**: 只保留必要的文章数据
- **网络友好**: 最小化网络请求频率

### 可访问性
- **字体大小**: 使用可读的字体大小
- **颜色对比**: 确保良好的文本可读性
- **交互反馈**: 提供明确的点击反馈

## 特殊功能

### 调试支持
- 开发者模式下可能显示额外的调试信息
- 支持强制刷新文章列表

### 错误处理
- 网络错误时显示友好的错误消息
- 空列表时显示相应的提示信息
- 文章加载失败的优雅降级

## 限制和注意事项
- **网络依赖**: 需要网络连接获取新文章
- **平台差异**: 某些平台可能不支持外部链接
- **缓存策略**: 文章缓存大小和过期时间限制
