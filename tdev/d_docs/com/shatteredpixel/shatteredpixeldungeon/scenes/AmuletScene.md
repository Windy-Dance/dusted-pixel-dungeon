# AmuletScene 类文档

## 概述
`AmuletScene` 是 Shattered Pixel Dungeon 的护身符胜利场景类，继承自 `PixelScene`。它在玩家获得护身符（游戏胜利条件）后显示，提供庆祝动画、成就展示和游戏完成选项。

## 类结构
```java
public class AmuletScene extends PixelScene
```

## 主要功能

### 1. 胜利动画系统
- **护身符展示**: 中央显示闪耀的护身符物品图标
- **粒子效果**: 围绕护身符的闪光和粒子特效 (`Speck`, `Flare`)
- **背景音乐**: 播放特殊的胜利主题音乐
- **动态动画**: 护身符的旋转、缩放和闪烁效果

### 2. 成就系统集成
- **徽章展示**: 自动显示与胜利相关的徽章横幅 (`BadgeBanner`)
- **成就验证**: 验证并解锁胜利相关的各种成就
- **全球同步**: 更新全局徽章数据

### 3. 游戏完成选项
**退出选项**:
- **主菜单**: 返回到游戏主菜单开始新游戏
- **排行榜**: 查看当前游戏的完成记录
- **关闭游戏**: 完全退出游戏（桌面平台）

**特殊选项**:
- **新游戏+**: 如果支持，提供继承某些元素的新游戏选项
- **每日挑战**: 快速跳转到每日挑战模式
- **分享功能**: 分享胜利成就（如果平台支持）

### 4. 数据保存和清理
- **游戏记录**: 保存完整的胜利游戏记录到排行榜
- **存档清理**: 清理临时存档和游戏状态
- **统计更新**: 更新全局游戏统计数据
- **设置重置**: 可选地重置某些游戏设置

## 视觉设计

### 核心元素
- **护身符图标**: 使用 `ItemSpriteSheet.AMULET` 显示标准护身符图像
- **金色主题**: 整体使用金色和白色作为主要颜色，象征胜利和成就
- **中心布局**: 护身符居中显示，按钮在下方排列

### 动画效果
- **Flare特效**: 护身符周围的光芒闪烁效果
- **Speck粒子**: 漂浮的光点粒子系统
- **缩放动画**: 护身符的大小变化动画
- **颜色渐变**: 背景和文本的颜色渐变效果

### UI组件
- **StyledButton**: 统一风格的选项按钮
- **RenderedTextBlock**: 渲染的文本块用于标题和说明
- **Chrome.Type**: 标准的UI组件样式

## 技术实现

### 音频管理
- **音乐切换**: 停止当前游戏音乐，播放胜利主题曲 `Assets.Music.VICTORY`
- **音效同步**: 与视觉动画同步的音效播放
- **资源清理**: 场景销毁时正确停止音乐

### 粒子系统
- **Speck工厂**: 创建各种类型的粒子效果
- **Flare系统**: 处理光芒和闪光特效
- **性能优化**: 限制同时存在的粒子数量

### 数据持久化
- **GamesInProgress**: 更新游戏进度记录
- **Badges**: 保存徽章成就数据
- **Statistics**: 更新全局统计数据

## 生命周期方法

### create()
场景初始化和胜利逻辑：
1. 停止当前音乐，播放胜利主题曲
2. 创建背景和核心视觉元素
3. 设置护身符动画和粒子效果
4. 显示胜利徽章横幅
5. 创建退出选项按钮
6. 执行淡入动画

### destroy()
场景销毁时清理音乐和粒子系统资源

## 用户体验设计

### 庆祝氛围
- **视觉冲击**: 丰富的动画和特效营造庆祝感
- **音频配合**: 胜利音乐增强成就感
- **成就认可**: 立即显示相关徽章和成就

### 选项清晰
- **简单选择**: 提供明确的后续操作选项
- **默认推荐**: 推荐最常用的选项（通常返回主菜单）
- **无压力设计**: 不强制任何特定选择

### 性能考虑
- **流畅动画**: 确保在各种设备上都有流畅的动画效果
- **内存管理**: 正确清理游戏相关的临时资源
- **快速响应**: 按钮响应迅速，不阻塞用户操作

## 特殊功能

### 随机元素
- **粒子随机性**: 粒子效果包含随机位置和时间
- **动画变化**: 每次胜利的动画略有不同

### 调试支持
- **快速访问**: 开发者可以快速测试胜利场景
- **成就测试**: 支持测试各种徽章显示效果

### 平台适配
- **移动平台**: 优化触摸交互
- **桌面平台**: 支持键盘和鼠标操作
- **控制器**: 支持游戏手柄导航
