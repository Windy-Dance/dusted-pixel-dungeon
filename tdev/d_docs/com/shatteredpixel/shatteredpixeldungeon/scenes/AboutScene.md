# AboutScene 类文档

## 概述
`AboutScene` 是 Shattered Pixel Dungeon 的关于页面场景类，继承自 `PixelScene`。它显示游戏的版权信息、许可证详情、贡献者列表以及相关的法律声明。

## 类结构
```java
public class AboutScene extends PixelScene
```

## 主要功能

### 1. 版权信息展示
- **原始作者**: 显示 Pixel Dungeon 的原始作者 Oleg Dolya
- **Shattered版本**: 显示 Shattered Pixel Dungeon 的作者 Evan Debenham
- **版权声明**: 完整的版权年份和声明文本
- **版本信息**: 显示当前游戏版本号

### 2. 开源许可证
- **GPLv3许可证**: 完整显示 GNU General Public License v3.0 文本
- **许可证链接**: 提供到官方许可证文本的链接
- **使用条款**: 说明游戏的使用和分发条件
- **免责声明**: 包含标准的软件免责声明

### 3. 贡献者致谢
- **主要贡献者**: 列出核心开发团队成员
- **社区贡献**: 感谢翻译贡献者、测试人员等
- **资源作者**: 致谢音效、图像等资源的原作者
- **开源项目**: 列出使用的第三方开源库和框架

### 4. 法律合规信息
- **商标声明**: 相关商标的使用声明
- **隐私政策**: 链接到隐私政策（如果适用）
- **年龄分级**: 游戏的内容分级信息
- **地区限制**: 特定地区的法律合规说明

## UI设计特点

### 布局系统
- **响应式列布局**: 横屏显示两列，竖屏显示单列
- **滚动支持**: 使用 `ScrollPane` 支持长文本内容的滚动查看
- **安全区域适配**: 自动处理刘海屏和圆角屏的安全区域

### 视觉元素
- **标题背景**: 使用标准的 `TitleBackground` 组件
- **退出按钮**: 右上角提供返回按钮
- **文本格式化**: 使用渲染文本块支持多语言和格式化
- **颜色方案**: 使用标准的窗口颜色主题

### 交互设计
- **滚动浏览**: 支持触摸和鼠标滚轮滚动
- **简单导航**: 只有退出按钮，保持界面简洁
- **无外部依赖**: 所有内容本地存储，无需网络连接

## 技术实现

### 内容管理
- **静态文本**: 大部分内容为静态文本，通过消息系统本地化
- **动态版本**: 游戏版本号从 `Game.version` 动态获取
- **许可证文本**: GPLv3许可证文本作为静态资源包含

### 本地化支持
- **多语言文本**: 所有用户可见文本通过 `Messages` 系统本地化
- **RTL支持**: 支持从右到左的语言布局
- **字符编码**: 正确处理各种语言的特殊字符

### 内存优化
- **按需渲染**: 长文本内容只在需要时渲染
- **资源缓存**: 重复使用的UI资源进行缓存
- **高效滚动**: 使用优化的滚动组件处理长内容

## 生命周期方法

### create()
场景初始化：
1. 创建标题背景和基础UI结构
2. 设置响应式列布局（横竖屏不同）
3. 创建并填充所有关于信息内容
4. 添加退出按钮和滚动区域
5. 执行淡入动画

### destroy()
场景销毁时清理滚动组件和临时资源

## 用户体验考虑

### 信息完整性
- **法律要求**: 包含所有必要的法律和版权信息
- **透明度**: 清晰说明游戏的来源和许可
- **准确性**: 确保所有信息准确且最新

### 可读性
- **字体选择**: 使用清晰易读的字体大小
- **行间距**: 适当的行间距提高长文本可读性
- **颜色对比**: 确保文本与背景有足够的对比度

### 可访问性
- **屏幕阅读器**: 支持辅助技术访问内容
- **键盘导航**: 支持键盘操作（桌面平台）
- **放大支持**: 文本在放大时保持可读性

## 特殊注意事项

### 法律合规
- **许可证遵守**: 严格按照GPLv3要求显示许可证信息
- **版权保护**: 正确归属所有版权持有者
- **商标使用**: 正确使用相关商标和标识

### 维护要求
- **版本更新**: 需要在新版本发布时更新版本信息
- **贡献者列表**: 定期更新贡献者名单
- **许可证变更**: 如果许可证有变化需要及时更新
