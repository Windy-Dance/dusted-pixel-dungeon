# RankingsScene 类文档

## 概述
`RankingsScene` 是 Shattered Pixel Dungeon 的排行榜场景类，继承自 `PixelScene`。它显示玩家的所有游戏记录，包括完成的游戏、死亡记录以及每日挑战成绩。

## 类结构
```java
public class RankingsScene extends PixelScene
```

## 主要功能

### 1. 排行榜数据展示
- **完成记录**: 显示成功通关的游戏记录
- **死亡记录**: 显示失败的游戏记录（按死亡深度排序）
- **每日挑战**: 专门的每日挑战成绩展示
- **详细信息**: 点击记录显示完整的游戏详情窗口

### 2. 记录分类系统
**三个主要标签页**:
- `TAB_OVERALL`: 总体记录（默认）
- `TAB_CUSTOM_SEEDS`: 自定义种子记录  
- `TAB_DAILIES`: 每日挑战记录

每个标签页显示相应类型的游戏记录，支持独立的排序和筛选。

### 3. 视觉元素
- **英雄图标**: 使用对应职业的头像显示
- **胜利标识**: 成功通关的记录显示金色星星特效
- **深度指示**: 显示每个记录的最终关卡深度
- **时间戳**: 显示游戏完成或死亡的时间

### 4. 特殊成就展示
- **徽章集成**: 显示与记录相关的特殊徽章
- **胜利庆祝**: 首次查看胜利记录时显示祝贺窗口
- **动态效果**: 胜利记录的星星闪烁动画

### 5. 用户交互
- **记录详情**: 点击任意记录显示详细信息窗口 (`WndRanking`)
- **每日挑战**: 点击每日挑战标签直接跳转到每日挑战窗口
- **返回导航**: 退出按钮返回主菜单

## 内部组件

### RankingButton (排行榜按钮)
私有内部类，用于显示单个游戏记录：
- 包含英雄头像、深度数字、时间戳等元素
- 胜利记录添加特殊视觉效果
- 处理点击事件显示详情

### TabSelector (标签选择器)  
私有内部类，管理三个标签页的切换：
- 显示当前选中的标签
- 处理标签切换逻辑
- 更新显示的记录列表

## 生命周期方法

### create()
场景初始化：
1. 停止当前音乐并播放排行榜专用音乐
2. 创建标题背景和UI元素
3. 加载并显示所有游戏记录
4. 处理胜利成就检查
5. 执行淡入动画

### destroy()
场景销毁时停止音乐播放

## 数据处理

### 记录排序
- 完成记录：按通关时间倒序
- 死亡记录：按死亡深度降序，同深度按时长升序
- 每日挑战：按挑战日期倒序

### 时间格式化
- 显示相对时间（"今天"、"昨天"、"X天前"）
- 长时间记录显示具体日期

### 特殊记录标识
- `rank.gameWon`: 胜利记录（显示星星特效）
- `rank.daily`: 每日挑战记录（特殊颜色标识）
- `rank.customSeed`: 自定义种子记录（黄色标识）

## 性能优化
- **延迟加载**: 记录详情窗口按需创建
- **内存管理**: 正确清理音乐资源
- **渲染优化**: 使用缓存的英雄头像

## 用户体验
- **清晰分类**: 三种记录类型明确区分
- **重要突出**: 胜利记录获得特殊视觉强调
- **便捷访问**: 一键查看详细信息
- **响应式设计**: 适配不同屏幕尺寸
