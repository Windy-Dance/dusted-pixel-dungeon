# RaisedTerrainTilemap.java - 抬升地形悬垂系统

## 概述
`RaisedTerrainTilemap` 负责渲染抬升地形的悬垂效果，主要处理高草和犁沟草地的下悬视觉效果。该系统创建深度感和立体感，使某些地形元素看起来从地面升起，增强地牢环境的视觉层次。

## 核心功能

### 悬垂效果支持
系统专注于两种主要的抬升地形类型：

#### 高草悬垂 (HIGH_GRASS_UNDERHANG)
- 为 **HIGH_GRASS** 地形提供下悬视觉效果
- 创建草地从地面升起的立体感
- 增强环境的自然外观

#### 犁沟草悬垂 (FURROWED_UNDERHANG)  
- 为 **FURROWED_GRASS** 地形提供下悬视觉效果
- 表现耕作过的土地纹理细节
- 与普通草地形成视觉对比

### 视觉层次管理
- **渲染顺序**：位于基础地形之上，其他特征之下
- **深度感知**：通过悬垂效果创建三维错觉
- **遮挡处理**：正确处理与其他瓦片图层的叠加关系

## 技术实现

### 视觉映射逻辑
`getTileVisual()` 方法实现简单的视觉映射：

```java
@Override
protected int getTileVisual(int pos, int tile, boolean flat) {
    if (tile == Terrain.HIGH_GRASS) {
        return DungeonTileSheet.HIGH_GRASS_UNDERHANG;
    } else if (tile == Terrain.FURROWED_GRASS) {
        return DungeonTileSheet.FURROWED_UNDERHANG;
    }
    return -1; // 返回 -1 表示无瓦片
}
```

- **直接映射**：基于地形类型直接返回对应的悬垂瓦片
- **性能优化**：简单的条件判断，无复杂计算
- **扩展性**：易于添加新的抬升地形类型

### 瓦片资源集成
- 使用 `DungeonTileSheet` 中预定义的悬垂瓦片常量
- 与中央瓦片管理系统保持同步
- 共享相同的纹理图集和UV坐标

## 使用模式

### 自动集成
```java
// 在游戏场景中自动创建和添加
RaisedTerrainTilemap raisedTerrain = new RaisedTerrainTilemap();
raisedTerrain.map(Dungeon.level.map, Dungeon.level.width());
scene.add(raisedTerrain);
```

### 动态更新
```java
// 当地形发生变化时自动更新
// 例如：高草被踩踏变成普通草地
Dungeon.level.set(pos, Terrain.GRASS);
raisedTerrain.updateMapCell(pos); // 悬垂效果自动消失
```

## 扩展性设计

### 新地形支持
添加新的抬升地形类型非常简单：
1. 在 `DungeonTileSheet` 中添加新的悬垂瓦片常量
2. 在 `getTileVisual()` 方法中添加相应的映射逻辑
3. 确保新地形在地牢生成时正确放置

### 自定义效果
- 可以扩展支持不同类型的悬垂效果
- 支持动态悬垂强度（基于游戏状态）

- 支持垂直方向的深度感知增强
