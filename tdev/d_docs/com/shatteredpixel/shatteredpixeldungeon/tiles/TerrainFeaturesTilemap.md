# TerrainFeaturesTilemap.java - 地形特征渲染系统

## 概述
`TerrainFeaturesTilemap` 负责渲染覆盖在基础地形之上的特征元素，包括植物、陷阱和草地变化效果。该系统为地牢环境添加丰富的细节层次，增强视觉表现力和游戏沉浸感。

## 核心功能

### 特征类型支持
系统支持三种主要的地形特征类型：

#### 陷阱 (Traps)
- 动态显示各种陷阱的视觉图标
- 基于陷阱的颜色和形状生成独特视觉效果
- 支持陷阱激活时的特殊视觉反馈

#### 植物 (Plants)
- 渲染所有可互动植物的精灵图像
- 支持植物的生长动画和状态变化
- 处理植物被踩踏或收获时的视觉效果

#### 草地变化 (Grass Variations)
- **HIGH_GRASS** (高草) - 带有基于生长阶段的着色效果
- **FURROWED_GRASS** (犁沟草) - 特殊纹理变体
- **GRASS** (普通草地) - 基础草地纹理
- **EMBERS** (余烬) - 燃烧效果的视觉表现

### 动态效果支持
- **植物生长动画**：使用缩放补间效果模拟植物生长
- **状态着色**：根据植物生长阶段动态调整颜色
- **交互反馈**：玩家与特征互动时的视觉响应

## 数据结构

### 稀疏数组存储
```java
private SparseArray<Plant> plants;
private SparseArray<Trap> traps;
```

- **内存效率**：只存储实际存在的特征，避免空值开销
- **快速访问**：O(1) 时间复杂度的特征查找
- **动态管理**：支持特征的添加、移除和更新

### 静态工具方法
```java
public static Image getTrapVisual(Trap trap)
public static Image getPlantVisual(Plant plant)
public static void growPlant(int pos)
```

- **重用性**：可在系统任何地方获取特征视觉效果
- **一致性**：确保所有特征使用统一的视觉表示
- **动画支持**：提供便捷的动画触发接口

## 技术实现

### 视觉生成逻辑
`getTileVisual()` 方法根据单元格的地形类型返回相应的视觉效果：

1. **陷阱检测**：检查单元格是否有陷阱并返回对应图标
2. **植物检测**：检查单元格是否有植物并返回植物图像  
3. **草地处理**：根据草地类型和生长阶段应用适当着色
4. **余烬效果**：处理燃烧状态的特殊视觉效果

### 动画系统集成
- 使用游戏的补间动画系统实现植物生长效果
- 缩放动画从0到1平滑过渡，创建自然生长感
- 动画完成后自动清理临时对象

### 着色算法
草地和植物的着色基于以下因素：
- **生长阶段**：幼苗、成熟、枯萎等不同阶段
- **环境条件**：光照、水分等游戏内状态
- **特殊效果**：中毒、祝福等状态影响

## 渲染性能

### 增量更新
- 只在特征发生变化时更新相关单元格
- 支持单特征更新和批量区域更新
- 避免不必要的全图重绘

### 内存优化
- 共享植物和陷阱的图像资源
- 重用动画对象减少GC压力
- 按需加载大型特征资源

### 批处理渲染
- 与底层Tilemap渲染器集成
- 合并相似的绘制调用
- 利用GPU纹理缓存提高效率

## 游戏集成

### 关卡生成
- 在关卡生成过程中自动放置特征
- 根据区域类型调整特征分布（如洞穴更多植物）
- 支持特殊房间的特征配置

### 玩家交互
- 玩家踩踏草地时更新视觉状态
- 与陷阱互动时显示激活效果
- 收获植物时播放消失动画

### 状态同步
- 与游戏世界状态保持同步
- 处理保存/加载时的特征状态恢复
- 支持多人游戏中的特征同步

## 使用模式

### 基本使用
```java
// 创建特征图层
TerrainFeaturesTilemap features = new TerrainFeaturesTilemap();
features.map(Dungeon.level.map, Dungeon.level.width());
scene.add(features);
```

### 特征操作
```java
// 添加植物
Plant plant = new Sungrass();
plant.pos = position;
Dungeon.level.plants.put(position, plant);
features.updateMapCell(position);

// 触发陷阱
Trap trap = Dungeon.level.traps.get(position);
if (trap != null && !trap.active) {
    trap.active = true;
    features.updateMapCell(position);
}
```

### 动画触发
```java
// 播放植物生长动画
TerrainFeaturesTilemap.growPlant(plantPos);
```
