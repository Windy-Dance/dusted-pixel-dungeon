# SlimeSprite - 史莱姆精灵

## 概述
`SlimeSprite` 类是史莱姆（Slime）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有流畅的动画序列和独特的血液颜色，专门用于表现粘性凝胶状的敌人。

## 动画帧
史莱姆精灵使用史莱姆纹理贴图 (`Assets.Sprites.SLIME`)，使用 14x12 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 1, 1, 0]，3 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [0, 2, 3, 3, 2, 0]，10 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [2, 3, 4, 6, 5]，15 FPS，非循环
- **死亡动画 (die)**: 帧序列 [0, 5, 6, 7]，10 FPS，非循环

## 特殊功能

### 血液颜色
- **亮绿色血液**: 重写 `blood()` 方法返回 `0xFF88CC44`（亮绿色）
- **粘液特性**: 绿色血液符合史莱姆作为凝胶状生物的身份特征
- **视觉识别**: 独特的血液颜色帮助玩家识别敌人类型

### 动画设计
- **弹性运动**: 空闲动画通过帧0和帧1的交替营造史莱姆的弹性抖动效果
- **完整奔跑循环**: 奔跑动画包含完整的起始-中间-结束循环，表现流畅的移动
- **复杂攻击序列**: 攻击动画包含5帧，展现史莱姆伸出触手的完整攻击动作

## 主要方法

### 构造函数
- 初始化所有基本动画
- 加载纹理并配置动画帧序列
- 设置初始空闲状态

### `blood()`
- 返回自定义的亮绿色血液颜色
- 使用十六进制颜色代码 `0xFF88CC44`

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.SLIME` 纹理贴图
- **帧尺寸**: 14x12 像素的紧凑帧尺寸
- **动画节奏**: 空闲动画使用3 FPS营造缓慢抖动感，攻击动画使用15 FPS提供快速反馈
- **死亡动画**: 包含4帧的详细死亡过程

## 使用场景
此精灵专门用于史莱姆敌人，通过弹性抖动的空闲动画、亮绿色血液和完整的攻击序列，准确表现了粘性凝胶状敌人的特性，为玩家提供清晰的战术识别。
