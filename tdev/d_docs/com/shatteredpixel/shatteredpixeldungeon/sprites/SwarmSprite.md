# SwarmSprite - 虫群精灵

## 概述
`SwarmSprite` 类是虫群（Swarm）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有流畅的动画序列和独特的血液颜色，专门用于表现由多个小昆虫组成的敌人。

## 动画帧
虫群精灵使用虫群纹理贴图 (`Assets.Sprites.SWARM`)，使用 16x16 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 1, 2, 3, 4, 5]，15 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [0, 1, 2, 3, 4, 5]，15 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [6, 7, 8, 9]，20 FPS，非循环
- **死亡动画 (die)**: 帧序列 [10, 11, 12, 13, 14]，15 FPS，非循环

## 特殊功能

### 血液颜色
- **橄榄绿色血液**: 重写 `blood()` 方法返回 `0xFF8BA077`（橄榄绿色）
- **昆虫特性**: 绿色血液符合虫群作为昆虫类敌人的生物特征
- **视觉识别**: 独特的血液颜色帮助玩家识别敌人类型

### 动画设计
- **同步空闲和奔跑**: 空闲和奔跑动画使用相同的帧序列，表现虫群的持续活动状态
- **流畅运动**: 15 FPS的高帧率确保动画的流畅性
- **完整攻击序列**: 攻击动画包含4帧，展现完整的攻击动作

## 主要方法

### 构造函数
- 初始化所有基本动画
- 加载纹理并配置动画帧序列
- 设置初始空闲状态

### `blood()`
- 返回自定义的橄榄绿色血液颜色
- 使用十六进制颜色代码 `0xFF8BA077`

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.SWARM` 纹理贴图
- **帧尺寸**: 16x16 像素的标准帧尺寸
- **动画节奏**: 攻击动画使用20 FPS提供快速的动作反馈
- **死亡动画**: 包含5帧的详细死亡过程

## 使用场景
此精灵专门用于虫群敌人，通过流畅的同步动画、橄榄绿色血液和完整的动作序列，准确表现了由多个昆虫组成的群体敌人的特性，为玩家提供清晰的战术识别。
