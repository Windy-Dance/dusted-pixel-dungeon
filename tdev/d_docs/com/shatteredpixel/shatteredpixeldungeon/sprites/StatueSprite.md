# StatueSprite - 雕像精灵

## 概述
`StatueSprite` 类是雕像（Statue）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有多层级装甲系统和自定义血液颜色，专门用于表现可以装备不同等级护甲的石像敌人。

## 动画帧
雕像精灵使用雕像纹理贴图 (`Assets.Sprites.STATUE`)，使用 12x15 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 0, 0, 0, 0, 1, 1]，2 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [2, 3, 4, 5, 6, 7]，15 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [8, 9, 10]，12 FPS，非循环
- **死亡动画 (die)**: 帧序列 [11, 12, 13, 14, 15, 15]，5 FPS，非循环

## 特殊功能

### 多层级装甲系统
- **动态装甲**: 通过 `setArmor()` 方法支持6个不同等级的装甲外观（0-5级）
- **帧偏移**: 每个装甲等级对应不同的纹理偏移量：[0, 21, 32, 43, 54, 65]
- **装甲限制**: 死亡动画始终保持无装甲状态，确保一致的死亡表现
- **平滑切换**: 使用 `play(idle, true)` 强制立即切换到新装甲的空闲动画

### 血液颜色
- **石灰色血液**: 重写 `blood()` 方法返回 `0xFFcdcdb7`（石灰色）
- **石材特性**: 灰色血液符合石像作为构造物的身份特征
- **视觉一致性**: 与石质敌人的材质保持一致

## 主要方法

### `setArmor(int tier)`
- 根据装甲等级设置对应的视觉外观
- 使用 `GameMath.gate()` 限制等级在有效范围内（0-5）
- 更新所有动画帧（除死亡动画外）以反映新的装甲外观
- 强制切换到新的空闲动画状态

### 构造函数
- 初始化基础无装甲动画
- 加载纹理并配置默认动画帧序列
- 设置初始空闲状态

### `blood()`
- 返回自定义的石灰色血液颜色
- 使用十六进制颜色代码 `0xFFcdcdb7`

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.STATUE` 纹理贴图
- **层级数组**: `tierFrames` 数组存储每个装甲等级的起始帧偏移量
- **数学函数**: 使用 `GameMath.gate()` 进行安全的等级边界检查
- **动画控制**: 死亡动画独立于装甲系统，保持一致的视觉表现

## 使用场景
此精灵专门用于雕像敌人，通过多层级装甲系统、石灰色血液和完整的动画序列，准确表现了可升级护甲的构造物敌人的特性，为玩家提供清晰的难度识别和战术判断依据。
