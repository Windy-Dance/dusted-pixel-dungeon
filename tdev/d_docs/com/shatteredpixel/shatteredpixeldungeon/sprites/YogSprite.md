# YogSprite - 尤格精灵

## 概述
`YogSprite` 类是最终Boss尤格-索托斯（Yog-Doom）的视觉表现类，继承自 `MobSprite`。该精灵具有复杂的空闲动画和独特的死亡特效，专门用于表现游戏中的终极敌人。

## 动画帧
尤格精灵使用尤格纹理贴图 (`Assets.Sprites.YOG`)，使用 20x19 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 1, 2, 2, 1, 0, 3, 4, 4, 3, 0, 5, 6, 6, 5]，10 FPS，循环播放
- **奔跑动画 (run)**: 单帧 [0]，12 FPS，循环播放  
- **攻击动画 (attack)**: 单帧 [0]，12 FPS，非循环
- **死亡动画 (die)**: 帧序列 [0, 7, 8, 9]，10 FPS，非循环

## 特殊功能

### 视觉渲染特性
- **视角提升**: 设置 `perspectiveRaise = 5/16f`，将精灵在视觉上提升适当高度
- **无阴影渲染**: 在链接角色时设置 `renderShadow = false`，符合Boss级别的特殊渲染需求

### 死亡特效
- **血液飞溅**: 死亡时调用 `Splash.at()` 创建血液飞溅效果
- **中心定位**: 使用精灵中心点作为飞溅效果的源点
- **粒子数量**: 发射12个血液粒子，营造壮观的死亡场面

## 主要方法

### 构造函数
- 初始化复杂的空闲动画序列
- 设置视角提升参数
- 加载纹理并配置所有动画

### `link(Char ch)`
- 链接角色实体
- 禁用阴影渲染以获得最佳视觉效果

### `die()`
- 处理死亡逻辑
- 触发血液飞溅特效

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.YOG` 纹理贴图
- **复杂动画**: 空闲动画包含15帧的复杂序列，展现尤格的动态特性
- **特效系统**: 集成 `Splash` 系统创建死亡特效
- **渲染优化**: 通过无阴影和视角提升优化Boss视觉表现

## 使用场景
此精灵专门用于最终Boss尤格-索托斯，通过复杂的空闲动画、无阴影渲染和壮观的死亡特效，准确表现了作为游戏终极敌人的威严和恐怖，为玩家提供史诗级的战斗体验。
