# WraithSprite - 幽灵精灵

## 概述
`WraithSprite` 类是幽灵（Wraith）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有简洁而有效的动画设计，专门用于表现亡灵类敌人的飘忽特性。

## 动画帧
幽灵精灵使用幽灵纹理贴图 (`Assets.Sprites.WRAITH`)，使用 14x15 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 1]，5 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [0, 1]，10 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [0, 2, 3]，10 FPS，非循环
- **死亡动画 (die)**: 帧序列 [0, 4, 5, 6, 7]，8 FPS，非循环

## 特殊功能

### 血液颜色
- **透明黑色血液**: 重写 `blood()` 方法返回 `0x88000000`（半透明黑色）
- **亡灵特性**: 黑色血液符合幽灵作为亡灵生物的身份特征
- **视觉一致性**: 透明效果营造出非物质化的视觉感受

## 主要方法

### 构造函数
- 初始化所有基本动画
- 加载纹理并配置动画帧序列
- 设置初始空闲状态

### `blood()`
- 返回自定义的半透明黑色血液颜色
- 使用十六进制颜色代码 `0x88000000`

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.WRAITH` 纹理贴图
- **简洁设计**: 空闲和奔跑动画共享相同的帧序列，体现幽灵飘忽的移动特性
- **攻击动画**: 包含3帧，展现攻击动作的完整过程
- **死亡动画**: 包含5帧，提供详细的死亡过程表现

## 使用场景
此精灵专门用于幽灵敌人，通过简洁的动画设计和半透明黑色血液效果，准确表现了亡灵生物的非物质化特性和恐怖氛围，为玩家提供清晰的敌人识别。
