# RatKingSprite - 鼠王精灵

## 概述
`RatKingSprite` 类是鼠王（Rat King）NPC的视觉表现类，继承自 `MobSprite`。该精灵具有独特的节日主题支持和特殊能力互动功能，能够根据不同的游戏状态显示不同的外观。

## 动画帧
鼠王精灵使用鼠王纹理贴图 (`Assets.Sprites.RATKING`)，使用 16x17 像素的帧大小，支持多种变体：

- **标准模式**: 偏移量 0-7
- **愚人节模式**: 偏移量 8-15  
- **冬季节日模式**: 偏移量 16-23
- **鼠化能力模式**: 偏移量 24-31

### 动画序列
- **空闲动画 (idle)**: 根据当前模式，帧序列 [c+0, c+0, c+0, c+1]，2 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [c+2, c+3, c+4, c+5, c+6]，10 FPS，循环播放
- **攻击动画 (attack)**: 单帧 [c+0]，15 FPS，非循环
- **死亡动画 (die)**: 单帧 [c+0]，10 FPS，非循环

## 特殊功能

### 节日主题支持
- **自动检测**: 通过 `Holiday.getCurrentHoliday()` 检测当前节日
- **动态切换**: 支持愚人节和冬季节日的特殊外观
- **无缝过渡**: 通过 `resetAnims()` 方法重新加载对应节日的动画

### 特殊能力互动
- **鼠化能力检测**: 当英雄拥有鼠化能力 (`Ratmogrify`) 时自动切换到特殊外观
- **光环效果**: 在鼠化能力激活时添加黄色光环 (`aura(0xFFFF00, 5)`)
- **实时更新**: 在链接角色时检查并应用相应的视觉效果

## 主要方法

### `resetAnims()`
- 重新初始化所有动画
- 根据当前节日和英雄能力选择正确的纹理偏移量
- 设置相应的动画帧序列

### `link(Char ch)`
- 链接角色实体
- 检查英雄是否拥有鼠化能力并应用光环效果

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.RATKING` 纹理贴图
- **多模式支持**: 同一纹理包含4种不同外观（标准、愚人节、冬季节日、鼠化）
- **动态检测**: 实时检测节日状态和英雄能力
- **光环系统**: 利用 `aura()` 方法创建视觉光环效果

## 使用场景
此精灵专门用于鼠王NPC，通过节日主题支持和特殊能力互动，为玩家提供丰富的视觉变化和个性化的游戏体验，体现了Shattered Pixel Dungeon注重细节和玩家互动的设计理念。
