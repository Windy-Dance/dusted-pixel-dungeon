# RotHeartSprite - 腐烂核心精灵

## 概述
`RotHeartSprite` 类是腐烂核心（Rot Heart）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有独特的毒性云雾粒子效果和静态外观设计，专门用于表现作为腐烂生态系统核心的植物敌人。

## 动画帧
腐烂核心精灵使用腐烂核心纹理贴图 (`Assets.Sprites.ROT_HEART`)，使用 16x16 像素的帧大小：

- **空闲动画 (idle)**: 单帧 [0]，1 FPS，静态显示
- **奔跑动画 (run)**: 单帧 [0]，1 FPS，静态显示  
- **攻击动画 (attack)**: 单帧 [0]，1 FPS，静态显示
- **死亡动画 (die)**: 帧序列 [1, 2, 3, 4, 5, 6, 7, 7, 7]，8 FPS，非循环

## 特殊功能

### 毒性云雾粒子
- **持续发射**: 在链接角色时创建毒性粒子发射器 (`Speck.TOXIC`)
- **可见性控制**: 粒子效果的可见性与精灵本身同步
- **资源管理**: 在角色死亡时正确关闭粒子发射器以释放资源
- **无阴影渲染**: 设置 `renderShadow = false`，符合植物核心的特性

### 静态外观设计
- **固定姿态**: 所有基本状态（空闲、奔跑、攻击）都使用单帧静态图像
- **视角提升**: 设置 `perspectiveRaise = 0.2f`，将核心在视觉上提升一定高度
- **方向固定**: 重写 `turnTo()` 方法为空实现，确保核心始终保持固定朝向

### 死亡动画
- **详细分解**: 包含9帧的详细死亡序列，展现核心的逐步分解过程
- **重复结束**: 最后3帧重复显示，营造核心完全消散的视觉效果

## 主要方法

### `link(Char ch)`
- 链接角色实体
- 初始化毒性云雾粒子发射器
- 确保无阴影渲染设置

### `update()`
- 更新精灵状态
- 同步粒子效果的可见性

### `die()`
- 处理死亡逻辑
- 关闭毒性粒子发射器以释放资源

### `blood()`
- 返回自定义的亮绿色血液颜色 (`0xFF88CC44`)
- 符合腐烂植物的生物特征

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.ROT_HEART` 纹理贴图
- **粒子系统**: 集成 `Speck.factory(Speck.TOXIC)` 创建毒性云雾效果
- **内存管理**: 通过正确的资源清理避免内存泄漏
- **渲染优化**: 静态外观减少性能开销，适合作为环境核心元素

## 使用场景
此精灵专门用于腐烂核心敌人，通过毒性云雾粒子、静态外观和详细的死亡动画，准确表现了作为腐烂生态系统核心的植物敌人的特性，为玩家提供清晰的战术识别和环境沉浸感。
