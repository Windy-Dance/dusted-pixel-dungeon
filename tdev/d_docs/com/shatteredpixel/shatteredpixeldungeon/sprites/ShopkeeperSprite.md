# ShopkeeperSprite - 商店老板精灵

## 概述
`ShopkeeperSprite` 类是商店老板（Shopkeeper）NPC的视觉表现类，继承自 `MobSprite`。该精灵具有独特的金币粒子效果和简洁的动画设计，专门用于表现游戏中的重要商业NPC。

## 动画帧
商店老板精灵使用守护者纹理贴图 (`Assets.Sprites.KEEPER`)，使用 14x14 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [1, 1, 1, 1, 1, 0, 0, 0, 0]，10 FPS，循环播放
- **奔跑动画 (run)**: 复制空闲动画  
- **攻击动画 (attack)**: 复制空闲动画
- **死亡动画 (die)**: 单帧 [0]，20 FPS，非循环

## 特殊功能

### 金币粒子效果
- **动态金币**: 在空闲动画完成时创建金色粒子 (`0xFFFF00`) 模拟金币
- **物理模拟**: 金币具有向上的初速度 (-40) 和向下的加速度 (+160)
- **位置定位**: 金币从精灵的右上角位置发射（考虑水平翻转）
- **可见性控制**: 仅在精灵可见且执行空闲动画时触发效果

### 粒子系统
- **像素粒子**: 使用 `PixelParticle` 创建简单的金币视觉效果
- **生命周期管理**: 通过 `reset()` 方法重用粒子对象
- **性能优化**: 仅在需要时创建粒子，避免不必要的资源消耗

## 主要方法

### `onComplete(Animation anim)`
- 监控空闲动画完成事件
- 创建或重用金币粒子
- 设置粒子的位置、颜色、大小和物理属性
- 触发金币抛射动画

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.KEEPER` 纹理贴图
- **粒子引擎**: 集成 `PixelParticle` 系统创建简单但有效的视觉效果
- **物理系统**: 利用速度和加速度属性创建逼真的抛物线运动
- **内存管理**: 通过对象重用减少内存分配
- **视觉反馈**: 金色粒子强化商店老板的商业身份特征

## 使用场景
此精灵专门用于商店老板NPC，通过金币粒子效果、简洁的动画设计和金色视觉反馈，准确表现了商业NPC的身份特征，为玩家提供直观的商店识别和沉浸式的交易体验。
