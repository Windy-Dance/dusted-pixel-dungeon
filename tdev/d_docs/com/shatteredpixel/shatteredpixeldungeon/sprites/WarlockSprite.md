# WarlockSprite - 术士精灵

## 概述
`WarlockSprite` 类是术士（Warlock）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有魔法施法特效和完整的动画序列，专门用于表现使用暗影魔法的敌人。

## 动画帧
术士精灵使用术士纹理贴图 (`Assets.Sprites.WARLOCK`)，使用 12x15 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 0, 0, 1, 0, 0, 1, 1]，2 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [0, 2, 3, 4]，15 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [0, 5, 6]，12 FPS，非循环
- **施法动画 (zap)**: 复制攻击动画
- **死亡动画 (die)**: 帧序列 [0, 7, 8, 8, 9, 10]，15 FPS，非循环

## 特殊功能

### 魔法导弹特效
- **暗影魔法**: 使用 `MagicMissile.SHADOW` 创建暗影魔法导弹
- **音效播放**: 施法时播放 `Assets.Sounds.ZAP` 音效
- **回调机制**: 通过回调函数通知角色施法完成

### 动画状态管理
- **自动切换**: 施法完成后自动切换回空闲状态
- **流畅过渡**: 通过 `onComplete()` 方法处理动画完成后的状态转换

## 主要方法

### `zap(int cell)`
- 执行施法动画
- 创建从角色到目标位置的魔法导弹
- 播放施法音效
- 通过回调通知角色施法完成

### `onComplete(Animation anim)`
- 处理施法动画完成后的状态切换
- 自动返回空闲状态

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.WARLOCK` 纹理贴图
- **魔法系统**: 集成 `MagicMissile` 系统创建视觉特效
- **音频系统**: 使用 `Sample.INSTANCE.play()` 播放音效
- **回调模式**: 利用 `Callback` 接口实现异步操作完成通知

## 使用场景
此精灵专门用于术士敌人，通过复杂的空闲动画、魔法施法特效和完整的死亡序列，准确表现了暗影魔法师的身份特征，为玩家提供清晰的魔法攻击预警和战术识别。
