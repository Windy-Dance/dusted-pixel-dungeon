# PiranhaSprite - 食人鱼精灵

## 概述
`PiranhaSprite` 类是普通食人鱼（Piranha）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有水生生物的典型特征，包括无阴影渲染、升高的视角和水波纹攻击效果。

## 动画帧
食人鱼精灵使用食人鱼纹理贴图 (`Assets.Sprites.PIRANHA`) 的起始区域，使用 12x16 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 1, 2, 1]，8 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [0, 1, 2, 1]，20 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [3, 4, 5, 6, 7, 8, 9, 10, 11]，20 FPS，非循环
- **死亡动画 (die)**: 帧序列 [12, 13, 14]，4 FPS，非循环

## 特殊功能

### 视觉渲染特性
- **无阴影**: 设置 `renderShadow = false`，使精灵不投射阴影，符合水生生物的特性
- **视角提升**: 设置 `perspectiveRaise = 0.2f`，将精灵在视觉上提升一定高度

### 水波纹效果
- **攻击反馈**: 在攻击动画完成时调用 `GameScene.ripple(ch.pos)` 创建水波纹效果
- **环境互动**: 通过水波纹增强攻击动作的视觉冲击力，营造水下战斗氛围

## 主要方法

### `link(Char ch)`
- 链接角色实体
- 确保无阴影渲染设置正确

### `onComplete(Animation anim)`
- 处理动画完成回调
- 在攻击动画结束时触发水波纹效果

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.PIRANHA` 纹理贴图的前21帧
- **帧尺寸**: 12x16 像素的动画帧尺寸
- **场景交互**: 通过 `GameScene.ripple()` 与游戏场景进行视觉互动

## 使用场景
此精灵专门用于普通食人鱼敌人，通过无阴影渲染、升高的视角和水波纹效果，准确表现了水生掠食者的特性，为玩家提供直观的水下敌人识别。
