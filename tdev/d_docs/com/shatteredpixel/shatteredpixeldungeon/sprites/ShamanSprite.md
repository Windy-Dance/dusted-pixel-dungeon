# ShamanSprite - 萨满精灵

## 概述
`ShamanSprite` 类是萨满（Shaman）敌人的抽象基类，继承自 `MobSprite`。该精灵具有三种不同颜色变体（红色、蓝色、紫色），每种变体使用相同的纹理贴图但不同的帧偏移量和魔法导弹类型，专门用于表现使用不同元素魔法的萨满敌人。

## 动画帧
萨满精灵使用萨满纹理贴图 (`Assets.Sprites.SHAMAN`)，使用 12x15 像素的帧大小，通过 `texOffset()` 方法确定起始帧偏移量：

### 红色萨满 (Red)
- **纹理偏移**: 0
- **魔法类型**: `MagicMissile.SHAMAN_RED`

### 蓝色萨满 (Blue)  
- **纹理偏移**: 21
- **魔法类型**: `MagicMissile.SHAMAN_BLUE`

### 紫色萨满 (Purple)
- **纹理偏移**: 42
- **魔法类型**: `MagicMissile.SHAMAN_PURPLE`

#### 通用动画序列
- **空闲动画 (idle)**: 帧序列 [c+0, c+0, c+0, c+1, c+0, c+0, c+1, c+1]，2 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [c+4, c+5, c+6, c+7]，12 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [c+2, c+3, c+0]，12 FPS，非循环
- **施法动画 (zap)**: 复制攻击动画
- **死亡动画 (die)**: 帧序列 [c+8, c+9, c+10]，12 FPS，非循环

## 特殊功能

### 抽象设计模式
- **模板方法**: `texOffset()` 作为抽象方法由子类实现
- **统一接口**: 所有变体共享相同的魔法施法逻辑
- **扩展性**: 易于添加新的萨满变体

### 魔法导弹系统
- **类型差异化**: 每种颜色萨满使用不同的魔法导弹类型
- **音效反馈**: 施法时播放 `Assets.Sounds.ZAP` 音效
- **回调机制**: 通过回调函数通知角色施法完成

### 动画状态管理
- **自动切换**: 施法完成后自动切换回空闲状态
- **流畅过渡**: 通过 `onComplete()` 方法处理动画完成后的状态转换

## 主要方法

### `zap(int cell)`
- 执行施法动画
- 创建对应颜色的魔法导弹特效
- 播放施法音效
- 通过回调通知角色施法完成

### `onComplete(Animation anim)`
- 处理施法动画完成后的状态切换
- 自动返回空闲状态

### `texOffset()`
- 抽象方法，由子类实现以确定纹理偏移量

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.SHAMAN` 纹理贴图
- **内存优化**: 三种变体共享同一纹理，通过偏移量区分
- **魔法系统**: 集成 `MagicMissile` 系统创建视觉特效
- **音频系统**: 使用 `Sample.INSTANCE.play()` 播放音效
- **回调模式**: 利用 `Callback` 接口实现异步操作完成通知
- **面向对象**: 采用抽象基类和静态内部类的设计模式

## 使用场景
此精灵专门用于三种颜色的萨满敌人，通过抽象基类设计、颜色差异化魔法效果和统一的施法逻辑，准确表现了使用不同元素魔法的萨满敌人的特性，为玩家提供清晰的魔法类型识别和战术应对策略。
