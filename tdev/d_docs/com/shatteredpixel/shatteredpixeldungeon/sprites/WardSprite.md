# WardSprite - 护卫精灵

## 概述
`WardSprite` 类是魔杖护卫（Ward）的视觉表现类，继承自 `MobSprite`。该精灵具有多层级外观、动态视觉效果和独特的死亡动画，专门用于表现由魔杖召唤的守护单位。

## 动画帧
护卫精灵使用护卫纹理贴图 (`Assets.Sprites.WARDS`)，支持7个不同的层级，每个层级具有独特的UV矩形区域：

- **1级**: UV矩形 (0, 0, 9, 10)
- **2级**: UV矩形 (10, 0, 21, 12)  
- **3级**: UV矩形 (22, 0, 37, 16)
- **4级**: UV矩形 (38, 0, 44, 13)
- **5级**: UV矩形 (45, 0, 51, 15)
- **6级**: UV矩形 (52, 0, 60, 15)

所有层级都使用静态动画（1 FPS，单帧）。

## 特殊功能

### 多层级系统
- **动态切换**: 通过 `updateTier()` 方法根据护卫等级切换外观
- **视觉差异**: 不同等级的护卫具有不同的大小和外观
- **层级渲染**: 高等级护卫（4-6级）具有动态浮动效果

### 死亡特效
- **淡出动画**: 死亡时不播放传统死亡动画，而是立即淡出
- **粒子效果**: 发射10个向上飘散的护卫粒子
- **平滑过渡**: 使用 `AlphaTweener` 实现2秒的淡出效果

### 施法特效
- **死亡射线**: 施法时创建从护卫到目标的死亡射线光束
- **粒子爆发**: 施法时发射2个向上的护卫粒子
- **音效反馈**: 播放射线音效 (`Assets.Sounds.RAY`)

### 动态效果
- **亮度调节**: 低等级护卫（1-3级）根据使用次数调整亮度
- **浮动动画**: 高等级护卫（4-6级）具有正弦波浮动效果
- **阴影变化**: 根据等级调整阴影大小和高度

## 主要方法

### `zap(int pos)`
- 执行施法逻辑
- 创建死亡射线光束
- 触发粒子效果和音效

### `die()`
- 处理特殊死亡逻辑
- 触发淡出动画和粒子效果

### `updateTier(int tier)`
- 更新护卫外观等级
- 调整渲染参数和视觉效果
- 设置正确的阴影和视角提升

### `resetColor()`
- 根据使用次数调整低等级护卫的亮度

### `update()`
- 处理高等级护卫的浮动动画
- 更新Y坐标和阴影偏移

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.WARDS` 纹理贴图
- **UV映射**: 使用 `texture.uvRect()` 精确控制每个等级的纹理区域
- **粒子系统**: 集成 `MagicMissile.WardParticle.UP` 创建独特粒子效果
- **光束系统**: 使用 `Beam.DeathRay` 创建视觉冲击力强的施法效果
- **动画系统**: 利用 `AlphaTweener` 实现平滑的淡出效果
- **数学函数**: 使用正弦函数创建自然的浮动动画

## 使用场景
此精灵专门用于魔杖护卫单位，通过多层级外观、动态浮动效果、死亡射线施法和淡出死亡动画，准确表现了魔法召唤物的特性，为玩家提供清晰的战术识别和视觉反馈。
