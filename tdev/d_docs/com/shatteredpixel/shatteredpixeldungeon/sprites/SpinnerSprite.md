# SpinnerSprite - 蜘蛛精灵

## 概述
`SpinnerSprite` 类是蜘蛛（Spinner）敌人的视觉表现类，继承自 `MobSprite`。该精灵具有独特的层级渲染、魔法导弹特效和特殊的血液颜色，专门用于表现能够投射蛛网的蜘蛛敌人。

## 动画帧
蜘蛛精灵使用蜘蛛纹理贴图 (`Assets.Sprites.SPINNER`)，使用 16x16 像素的帧大小：

- **空闲动画 (idle)**: 帧序列 [0, 0, 0, 0, 0, 1, 0, 1]，10 FPS，循环播放
- **奔跑动画 (run)**: 帧序列 [0, 2, 0, 3]，15 FPS，循环播放  
- **攻击动画 (attack)**: 帧序列 [0, 4, 5, 0]，12 FPS，非循环
- **施法动画 (zap)**: 复制攻击动画
- **死亡动画 (die)**: 帧序列 [6, 7, 8, 9]，12 FPS，非循环

## 特殊功能

### 层级渲染
- **背景渲染**: 在链接角色时将精灵置于父容器的最底层 (`parent.sendToBack()`)
- **光环处理**: 同时将光环效果也置于底层，确保正确的视觉层次
- **无阴影**: 设置 `renderShadow = false`，符合蜘蛛的生物特性

### 魔法导弹特效
- **标准导弹**: 使用 `MagicMissile.MAGIC_MISSILE` 创建魔法导弹
- **音效反馈**: 施法时播放 `Assets.Sounds.MISS` 音效
- **回调机制**: 通过回调函数触发蜘蛛的 `shootWeb()` 方法

### 血液颜色
- **淡绿色血液**: 重写 `blood()` 方法返回 `0xFFBFE5B8`（淡绿色）
- **蜘蛛特性**: 绿色血液符合蜘蛛作为节肢动物的生物特征

## 主要方法

### `link(Char ch)`
- 处理层级渲染逻辑
- 确保无阴影设置
- 正确放置精灵和光环在渲染层级中

### `zap(int cell)`
- 执行施法动画
- 创建魔法导弹特效
- 触发蛛网投射回调

### `onComplete(Animation anim)`
- 处理施法完成后的状态切换
- 自动切换到奔跑动画

## 技术细节
- **纹理资源**: 使用 `Assets.Sprites.SPINNER` 纹理贴图
- **视角设置**: `perspectiveRaise = 0f`，贴近地面的渲染效果
- **魔法系统**: 集成 `MagicMissile` 系统创建视觉特效
- **音频系统**: 使用 `Sample.INSTANCE.play()` 播放音效
- **回调模式**: 利用 `Callback` 接口实现异步操作完成通知

## 使用场景
此精灵专门用于蜘蛛敌人，通过底层渲染、魔法导弹特效和淡绿色血液，准确表现了能够投射蛛网的蜘蛛敌人的特性，为玩家提供清晰的战术识别和视觉反馈。
