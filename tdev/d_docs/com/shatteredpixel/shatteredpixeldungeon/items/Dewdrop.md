# Dewdrop.java - 露珠

## 概述
`Dewdrop` 类代表游戏中的露珠道具，是基础的治疗资源。露珠可以被收集到水袋（Waterskin）中，也可以直接消耗为英雄恢复生命值，并与特定天赋和饰品产生交互。

## 核心功能

### 双重使用机制
- **水袋收集**: 当英雄拥有未满的水袋时，露珠会自动被收集到水袋中
- **直接消耗**: 当没有水袋或水袋已满时，露珠会立即为英雄恢复生命值
- **强制消耗**: 在特殊地形（入口/出口）即使生命值满也会强制消耗露珠

### 治疗计算
- **基础治疗量**: 每颗露珠恢复5%最大生命值（HT * 0.05f）
- **堆叠效果**: 多颗露珠同时拾取时按总量计算治疗效果
- **上限控制**: 治疗量不会超过英雄缺失的生命值

### 天赋集成
- **SHIELDING_DEW**: 盾化露珠天赋
  - 将多余的治疗量转换为护盾（Barrier）
  - 护盾上限：最大生命值的20% × 天赋等级（1-2级）
  - 护盾可叠加，但有上限限制

### 饰品协同
- **VialOfBlood**: 血瓶饰品
  - 启用延迟能量爆发治疗（delayBurstHealing）
  - 应用总治疗倍率（totalHealMultiplier）
  - 限制每回合最大治疗量（maxHealPerTurn）

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.DEWDROP` 图像
- **stackable = true**: 支持堆叠
- **dropsDownHeap = true**: 拾取时产生掉落动画
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）

### 堆叠限制
- **特殊合并逻辑**: 虽然标记为可堆叠，但实际限制每堆最多1颗
- **quantity(int value)**: 重写数量设置方法，确保不超过1

## 视觉和音效反馈

### UI表现
- **治疗显示**: 显示绿色治疗数字（FloatingText.HEALING）
- **护盾显示**: 显示蓝色护盾数字（FloatingText.SHIELDING）
- **状态提示**: 当英雄生命值满时显示"already_full"提示

### 音效系统
- 播放露珠音效（Assets.Sounds.DEWDROP）
- 拾取动画通过GameScene.pickUp处理

## 使用场景
- **地图生成**: 地牢中随机生成的露珠
- **植物互动**: 某些植物（如Sungrass）会产生露珠
- **天赋奖励**: 特定天赋或技能产生的露珠
- **任务物品**: 某些任务或事件中获得的露珠

## 注意事项
1. 露珠的实际使用取决于英雄当前装备状态（是否有水袋）
2. Shielding Dew天赋将治疗溢出转换为护盾，提高生存能力
3. VialOfBlood饰品改变了治疗的应用方式，启用延迟治疗机制
4. 露珠在入口/出口地形会强制消耗，即使生命值已满
5. 统计数据会记录露珠的使用次数（Catalog.countUse）
