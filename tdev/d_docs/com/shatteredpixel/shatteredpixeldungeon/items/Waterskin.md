# Waterskin.java - 水袋

## 概述
`Waterskin` 类代表水袋道具，用于收集和存储露珠（Dewdrop），并在需要时为英雄提供治疗。水袋是唯一物品，具有固定的最大容量，并支持与天赋和饰品的深度交互。

## 核心功能

### 容量管理系统
- **MAX_VOLUME = 20**: 最多可存储20颗露珠
- **volume**: 当前存储的露珠数量
- **collectDew(Dewdrop dew)**: 收集露珠到水袋中
- **empty()**: 清空水袋内容
- **fill()**: 将水袋填满至最大容量
- **isFull()**: 检查水袋是否已满

### 饮用机制
- **AC_DRINK**: 饮用动作常量
- **智能消耗**: 根据英雄当前状态自动计算所需露珠数量
  - 计算缺失生命值百分比
  - 转换为所需露珠数量（每颗露珠=5%最大生命值）
  - 考虑护盾需求（Shielding Dew天赋）

### 天赋和饰品集成
- **SHIELDING_DEW**: 计算额外的护盾需求并相应增加消耗量
- **VialOfBlood**: 应用治疗倍率调整，减少实际消耗量
- 通过 `Dewdrop.consumeDew()` 处理实际的治疗和护盾效果

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.WATERSKIN` 图像
- **defaultAction = AC_DRINK**: 默认动作为饮用
- **unique = true**: 作为唯一物品，死亡后保留
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）

### 状态显示
- **status()**: 显示当前容量状态（"X/20"格式）
- **info()**: 根据容量状态显示不同的描述文本
  - 空水袋：显示收集露珠的说明
  - 有内容：显示治疗功能说明  
  - 已满：显示容量已满的提示

## 交互逻辑

### 动作系统
- **actions(Hero hero)**: 仅当水袋非空时显示饮用选项
- **execute(Hero hero, String action)**: 处理饮用动作
  - 计算最优消耗量（考虑生命值和护盾需求）
  - 限制消耗量不超过可用容量
  - 更新统计数据（Catalog.countUses）

### 视觉和音效反馈
- **操作动画**: 显示英雄使用水袋的动作
- **音效**: 播放喝水音效（Assets.Sounds.DRINK）
- **状态更新**: 自动更新快捷栏显示

## 使用场景
- **露珠收集**: 自动收集地图上的露珠
- **紧急治疗**: 在战斗中快速恢复生命值
- **天赋配合**: 与Shielding Dew天赋配合获得护盾
- **饰品协同**: 与VialOfBlood饰品产生特殊治疗效果

## 注意事项
1. 水袋的容量限制确保了资源管理的策略性
2. 智能消耗系统避免了不必要的资源浪费
3. 与Dewdrop类紧密耦合，共享治疗计算逻辑
4. 统计数据会记录露珠的实际使用次数
5. 快捷栏显示会实时反映水袋的容量状态
6. 空水袋时仍可执行饮用动作，但会显示警告消息
