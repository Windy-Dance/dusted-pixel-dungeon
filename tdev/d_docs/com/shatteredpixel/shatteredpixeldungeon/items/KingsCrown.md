# KingsCrown.java - 国王之冠

## 概述
`KingsCrown` 类代表国王之冠道具，是战士职业的专属物品，用于将普通护甲升级为职业护甲（ClassArmor）并选择特殊能力。这个唯一物品在英雄达到特定条件后获得，是战士职业进阶的重要机制。

## 核心功能

### 护甲升级系统
- **upgradeArmor()**: 主要升级方法，执行完整的升级流程
- **ClassArmor.upgrade()**: 调用护甲工厂方法创建职业护甲
- **能力绑定**: 将选择的ArmorAbility绑定到英雄身上
- **天赋初始化**: 调用Talent.initArmorTalents()初始化相关天赋

### 能力选择界面
- **WndChooseAbility**: 显示能力选择窗口
- **前提条件**: 英雄必须装备了护甲才能使用国王之冠
- **错误处理**: 未装备护甲时显示"naked"提示

### 视觉和音效反馈
- **粒子效果**: 显示12个皇冠粒子效果（Speck.CROWN）
- **操作动画**: 显示英雄佩戴皇冠的动作
- **音效**: 播放精通音效（Assets.Sounds.MASTERY）
- **UI更新**: 更新英雄护甲外观和快捷栏

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.CROWN` 图像
- **defaultAction = AC_WEAR**: 默认动作为佩戴
- **unique = true**: 作为唯一物品，确保不会重复获得
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）

### 经济价值
- **统计追踪**: Catalog系统记录使用次数
- **不可交易**: 作为关键剧情物品不具有经济价值

## 交互逻辑

### 动作系统
- **AC_WEAR**: 佩戴动作，触发能力选择界面
- **actions()**: 只显示WEAR动作（无丢弃、投掷等其他选项）

### 升级流程
1. **验证条件**: 检查英雄是否装备了护甲
2. **显示界面**: 打开能力选择窗口
3. **执行升级**: 
   - 移除国王之冠
   - 创建职业护甲
   - 替换现有护甲或添加到背包
   - 激活新护甲的能力
4. **状态更新**: 
   - 绑定选择的能力到英雄
   - 初始化相关天赋
   - 更新视觉表现

### 特殊能力处理
- **Ratmogrify**: 鼠化能力显示特殊的"ratgraded"消息
- **其他能力**: 显示标准的"upgraded"消息
- **能力持久化**: 通过hero.armorAbility保存选择的能力

## 使用场景

### 获取条件
- **战士职业**: 通常作为战士职业的专属奖励
- **进度要求**: 在达到特定游戏进度后获得
- **任务奖励**: 可能作为特殊任务或挑战的奖励

### 战略意义
- **职业进阶**: 标志着战士职业的正式进阶
- **能力定制**: 允许玩家根据游戏风格选择不同的能力
- **护甲强化**: 提供比普通护甲更强的防护和特殊效果

## 注意事项
1. 国王之冠只能使用一次，使用后会被消耗
2. 必须装备护甲才能使用，裸体状态下无法使用
3. 升级后的职业护甲会继承原护甲的等级和附魔
4. 选择的能力会影响后续的天赋发展和游戏策略
5. 视觉和音效反馈强调了这是一个重要的里程碑事件
6. 与Talent系统深度集成，影响英雄的整体能力构建
