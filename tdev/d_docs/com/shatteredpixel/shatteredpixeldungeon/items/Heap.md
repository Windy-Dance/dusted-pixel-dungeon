# Heap.java - 物品堆

## 概述
`Heap` 类代表游戏中地面的物品堆，用于管理地面上的物品集合。它处理物品的拾取、放置、交互以及特殊容器（如宝箱、墓穴等）的逻辑，是物品系统与地牢环境交互的核心组件。

## 核心功能

### 堆类型系统 (Type)
- **HEAP**: 普通物品堆
- **FOR_SALE**: 商店销售物品
- **CHEST**: 普通宝箱
- **LOCKED_CHEST**: 锁定宝箱
- **CRYSTAL_CHEST**: 水晶宝箱（包含神器、魔杖或戒指）
- **TOMB**: 墓穴（打开时召唤幽灵）
- **SKELETON**: 骨架（包含遗物）
- **REMAINS**: 英雄遗骸

### 物品管理
- **LinkedList<Item> items**: 使用链表存储物品，保持拾取顺序
- **drop(Item item)**: 添加物品到堆中，处理堆叠和特殊放置规则
- **pickUp()**: 拾取最上面的物品
- **peek()**: 查看最上面的物品而不移除
- **replace(Item a, Item b)**: 替换指定物品
- **remove(Item a)**: 移除指定物品

### 特殊交互

#### 开启机制 (open)
- **墓穴(TOMB)**: 召唤周围的幽灵(Wraith)
- **骨架/遗骸(SKELETON/REMAINS)**: 播放骨骼摇晃音效
- **诅咒检查(haunted)**: 如果包含被诅咒物品，会触发诅咒效果
- **财富戒指(RingOfWealth)**: 有机会额外掉落物品

#### 环境影响
- **burn()**: 火焰效果处理
  - 烧毁卷轴(Scroll)
  - 蒸发露珠(Dewdrop)  
  - 烹饪肉类(MysteryMeat → ChargrilledMeat)
  - 冻肉解冻(FrozenCarpaccio → MysteryMeat)
  - 引爆炸弹(Bomb)
- **explode()**: 爆炸效果处理
  - 破坏普通容器
  - 打碎药水(Potion)
  - 销毁非独特物品
  - 保留装备和独特物品
- **freeze()**: 冰冻效果处理
  - 冷冻肉类(MysteryMeat → FrozenCarpaccio)
  - 打碎药水
  - 冻结炸弹引信

### 视觉和状态管理
- **seen**: 是否已被玩家看到
- **haunted**: 是否被诅咒（包含被诅咒物品）
- **autoExplored**: 是否计入探索奖励
- **hidden**: 是否隐藏（15%透明度）
- **sprite**: 关联的视觉精灵

## 属性和方法

### 核心属性
- **pos**: 在地牢中的位置坐标
- **type**: 堆的类型（影响交互行为）
- **items**: 物品列表
- **sprite**: 视觉表现精灵

### 重要方法
- **setHauntedIfCursed()**: 检查物品是否包含被诅咒物品并设置haunted标志
- **size()**: 返回物品数量
- **isEmpty()**: 检查是否为空
- **destroy()**: 销毁物品堆（从地牢中移除）
- **title()/info()**: 返回显示标题和描述信息

### 特殊规则
- **LostBackpack优先**: 失落背包总是位于堆顶
- **dropsDownHeap**: 特定物品（如露珠）会放在堆底
- **商店物品**: FOR_SALE类型的堆有特殊的定价显示逻辑

## 数据持久化

### Bundle集成
- **restoreFromBundle()**: 从存档恢复物品堆状态
  - 清理无效的文档页面
  - 移除已获得的指南书
  - 保持物品顺序和状态
- **storeInBundle()**: 保存物品堆状态到存档
  - 包含位置、类型、可见性等所有属性

### 文档系统集成
- **DocumentPage清理**: 自动移除玩家已拥有的文档页面
- **Guidebook处理**: 如果玩家已阅读冒险者指南第一页，则移除指南书

## 使用场景

### 游戏机制
- **物品掉落**: 敌人死亡后在地面创建物品堆
- **宝箱系统**: 各种类型宝箱的开启和奖励机制
- **环境交互**: 火焰、冰冻、爆炸对地面物品的影响
- **商店系统**: 商店物品的展示和购买机制

### 视觉表现
- **堆叠显示**: 多个物品时显示堆叠图标
- **特殊动画**: 宝箱开启、诅咒触发等特殊效果
- **粒子效果**: 燃烧、蒸发、冻结等环境影响的视觉反馈

## 注意事项
1. 物品堆使用链表而不是数组，确保高效的头部操作
2. 特殊容器（宝箱、墓穴等）在开启后会转换为普通堆
3. 被诅咒的物品堆会自动标记haunted标志并触发相应效果  
4. 环境效果（燃烧、爆炸、冰冻）只影响特定类型的物品
5. 装备和独特物品在爆炸中不会被破坏
6. 商店物品的价格计算和显示有特殊逻辑
7. 文档系统的集成确保游戏进度的一致性
