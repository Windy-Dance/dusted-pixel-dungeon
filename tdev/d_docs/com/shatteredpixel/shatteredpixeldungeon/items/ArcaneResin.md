# ArcaneResin.java - 秘法树脂

## 概述
`ArcaneResin` 类代表秘法树脂道具，用于临时增强魔杖（Wand）的充能能力。树脂可以增加魔杖的当前充能数和最大充能数，并与法师职业天赋产生深度交互。

## 核心功能

### 魔杖增强系统
- **resinBonus**: 魔杖的树脂加成计数器
- **充能增加**: 应用树脂后立即增加1点当前充能
- **等级要求**: 只能对等级3以下的魔杖使用
- **消耗数量**: 使用数量 = 魔杖等级 + 1

### 使用机制
- **目标选择**: 必须选择已识别的魔杖
- **资源检查**: 确保有足够的树脂数量
- **自动消耗**: 使用后自动从背包中扣除相应数量的树脂
- **效果应用**: 立即更新魔杖的充能状态

### 制作系统集成
- **Recipe内部类**: 提供从魔杖制作树脂的配方
- **制作条件**: 
  - 输入：单个未被诅咒的魔杖
  - 成本：5点能量水晶
- **产量计算**: 
  - 基础产量：2 × (魔杖等级 + 1)
  - Wand Preservation天赋：额外增加产量

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.ARCANE_RESIN` 图像
- **stackable = true**: 支持堆叠
- **defaultAction = AC_APPLY**: 默认动作为应用
- **bones = true**: 可以出现在英雄遗骸中
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）

### 经济价值
- **value()**: 每单位价值30金币
- **统计追踪**: Catalog系统记录使用次数

## 交互逻辑

### 动作系统
- **AC_APPLY**: 应用动作，打开物品选择器
- **itemSelector**: 特殊的物品选择器
  - 首选手套包（MagicalHolster）
  - 只允许选择已识别的魔杖

### 视觉和音效反馈
- **操作动画**: 显示英雄使用树脂的动作
- **粒子效果**: 向上飘散的粒子效果（Speck.UP）
- **音效**: 传送音效（Assets.Sounds.TELEPORT）
- **状态提示**: 显示成功应用的消息

## 天赋集成

### Wand Preservation
- **法师职业专属**: 法师使用时有额外效果
- **产量提升**: 制作树脂时产量增加
- **非法师职业**: 也可以使用天赋提升产量

### 效果限制
- **等级上限**: 魔杖等级不能超过3
- **诅咒检查**: 不能对被诅咒的魔杖使用
- **识别要求**: 魔杖必须已识别

## 使用场景
- **紧急充能**: 战斗中快速为魔杖补充充能
- **战术准备**: 在重要战斗前预先增强魔杖
- **资源管理**: 通过制作系统将不需要的魔杖转换为树脂
- **天赋配合**: 与Wand Preservation天赋配合最大化收益

## 注意事项
1. 树脂效果是永久性的，会持续到魔杖被丢弃或销毁
2. 制作树脂需要消耗完整的魔杖（魔杖数量变为0）
3. 对于未识别的魔杖，制作出的树脂数量固定为1
4. 树脂的视觉效果使用传送音效和向上粒子，暗示能量注入的概念
5. 与Broken Seal类似，也具有独特的制作配方系统
