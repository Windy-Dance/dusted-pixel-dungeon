# EnergyCrystal.java - 能量水晶

## 概述
`EnergyCrystal` 类代表游戏中的能量水晶道具，是用于制作和升级的特殊货币。与金币不同，能量水晶专门用于高级制作系统（Recipes）和特殊物品的合成。

## 核心功能

### 能量货币系统
- **自动累加**: 拾取时自动增加 `Dungeon.energy` 全局能量值
- **制作资源**: 用于Recipe（配方）系统中的物品制作
- **不可操作**: 没有任何可用动作，仅作为资源存在

### 可堆叠系统
- **stackable = true**: 支持堆叠多个能量水晶为单个物品
- **quantity**: 表示能量水晶的数量值
- 拾取时会自动合并到现有的能量水晶堆栈中

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.ENERGY` 图像
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）
- **actions()**: 返回空列表（无法进行任何操作）

### 视觉反馈
- **拾取动画**: 显示能量水晶拾取特效 (`GameScene.pickUp`)
- **状态显示**: 在英雄头顶显示获得的能量数量 (`FloatingText.ENERGY`)，使用青色(0x44CCFF)显示
- **快捷栏更新**: 自动更新快捷栏显示

### 音效系统
- 播放通用物品音效 (`Assets.Sounds.ITEM`)

## 使用场景
- **制作系统**: Recipe系统中需要消耗能量水晶来制作高级物品
- **任务奖励**: 完成特定任务或挑战后获得的能量奖励
- **敌人掉落**: 击败特定敌人后掉落的能量水晶
- **宝箱奖励**: 开启特殊宝箱获得的能量水晶

## 与金币的区别
| 特性 | 金币 (Gold) | 能量水晶 (EnergyCrystal) |
|------|-------------|-------------------------|
| 颜色 | 金色 | 青色 |
| 用途 | 商店购买、基础经济 | 高级制作、特殊合成 |
| 统计追踪 | `Statistics.goldCollected` | 未单独追踪（通过Recipes统计） |
| 获取方式 | 敌人掉落、宝箱、随机生成 | 任务奖励、特殊掉落、制作系统 |

## 注意事项
1. 能量水晶拾取后立即从地面消失并添加到玩家总能量中
2. 能量水晶不占用背包空间，直接转换为全局能量值
3. 能量水晶的Catalog记录会在首次拾取时自动标记为已见
4. 目前没有单独追踪收集的能量水晶数量，但会追踪制作的配方数量
5. 能量水晶主要用于后期游戏的高级制作系统
