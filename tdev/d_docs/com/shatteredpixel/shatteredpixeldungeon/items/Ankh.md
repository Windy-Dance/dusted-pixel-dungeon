# Ankh.java - 安克护身符

## 概述
`Ankh` 类代表复活道具安克护身符，允许英雄在死亡后复活。通过与 Waterskin（水袋）交互可以对其进行祝福，提升复活效果。

## 核心功能

### 复活机制
- 作为基础复活道具，死亡时自动触发复活效果
- **bones = true**: 可以出现在英雄遗骸中，供后续角色使用

### 祝福系统
- **AC_BLESS**: 祝福动作常量
- **bless()**: 将安克护身符标记为已祝福状态
- **isBlessed()**: 检查是否已被祝福
- 需要完整的 Waterskin（水袋）才能进行祝福

### 视觉和音效反馈
- **未祝福状态**: 正常显示
- **已祝福状态**: 显示白色发光效果 (`Glowing WHITE`)
- 祝福时播放喝水音效并显示光粒子效果

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.ANKH` 图像
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）
- **value()**: 返回50金币价值（可堆叠）

### 状态持久化
- **blessed**: 通过Bundle系统保存祝福状态
- 支持游戏保存和加载时保持状态

## 交互逻辑

### 动作系统
- **actions(Hero hero)**: 仅当拥有完整水袋且未祝福时显示祝福选项
- **execute(Hero hero, String action)**: 处理祝福动作，消耗水袋中的水

### 描述文本
- **未祝福**: 显示基础描述
- **已祝福**: 显示祝福后的特殊描述（通过 `Messages.get(this, "desc_blessed")`）

## 使用场景
- **死亡复活**: 英雄死亡时自动使用安克护身符复活
- **主动祝福**: 在安全环境下使用水袋对安克护身符进行祝福
- **遗骸回收**: 从之前死亡英雄的遗骸中获得安克护身符

## 注意事项
1. 祝福操作会消耗水袋中的全部水量
2. 已祝福的安克护身符在视觉上会有明显的发光效果
3. 安克护身符的价值随数量线性增长（50 * quantity）
4. 祝福后的安克护身符可能提供更好的复活效果（具体效果在其他系统中实现）
