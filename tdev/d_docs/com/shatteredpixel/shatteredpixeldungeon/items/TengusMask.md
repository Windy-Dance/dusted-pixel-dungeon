# TengusMask.java - 天狗面具

## 概述
`TengusMask` 类代表天狗面具道具，是英雄选择子职业（HeroSubClass）的关键物品。这个唯一物品在英雄达到特定条件后获得，通过佩戴面具可以选择四种不同的子职业路径，每个路径提供独特的技能和天赋。

## 核心功能

### 子职业选择系统
- **choose(HeroSubClass way)**: 主要选择方法，执行完整的子职业选择流程
- **WndChooseSubclass**: 显示子职业选择窗口
- **天赋初始化**: 调用Talent.initSubclassTalents()初始化对应子职业的天赋
- **成就验证**: 拾取时自动验证Master成就

### 子职业特殊处理
- **ASSASSIN**: 如果英雄处于隐身状态，自动应用Preparation Buff
- **其他子职业**: 直接应用子职业效果，无特殊处理

### 视觉和音效反馈
- **粒子效果**: 20个面具粒子效果（Speck.MASK），定位在英雄头部位置
- **操作动画**: 显示英雄佩戴面具的动作
- **音效**: 播放精通音效（Assets.Sounds.MASTERY）
- **成功提示**: 显示"used"提示表示子职业选择成功

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.MASK` 图像
- **stackable = false**: 不可堆叠（唯一物品）
- **defaultAction = AC_WEAR**: 默认动作为佩戴
- **unique = true**: 作为唯一物品，确保不会重复获得
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）

### 成就集成
- **Badges.validateMastery()**: 拾取时验证Master成就
- **Catalog追踪**: 记录使用次数用于统计

## 交互逻辑

### 动作系统
- **AC_WEAR**: 佩戴动作，触发子职业选择界面
- **actions()**: 只显示WEAR动作（无丢弃、投掷等其他选项）

### 选择流程
1. **显示界面**: 打开子职业选择窗口
2. **执行选择**: 
   - 移除天狗面具
   - 设置英雄的subClass属性
   - 初始化对应的子职业天赋
   - 处理子职业特殊效果（如Assassin的Preparation）
3. **状态更新**: 
   - 更新英雄的视觉表现
   - 播放完成音效和粒子效果

### 安全机制
- **一次性使用**: 使用后立即从背包中移除
- **状态保护**: 确保子职业选择的原子性操作

## 使用场景

### 获取条件
- **等级要求**: 通常在英雄达到特定等级后获得
- **进度奖励**: 作为游戏进度的重要里程碑奖励
- **挑战完成**: 可能作为完成特定挑战的奖励

### 战略意义
- **职业进阶**: 标志着英雄职业的正式分化
- **玩法定制**: 允许玩家根据游戏风格选择不同的发展路径
- **天赋影响**: 子职业选择会显著影响后续的天赋发展方向

## 注意事项
1. 天狗面具只能使用一次，使用后会被消耗
2. 子职业选择是永久性的，无法更改或重置
3. Assassin子职业与隐身状态有特殊交互，提供额外的战斗优势
4. 视觉和音效反馈强调了这是一个重要的角色发展里程碑
5. 与Talent系统的深度集成确保了子职业能力的正确初始化
6. 成就系统的集成鼓励玩家积极探索不同的子职业路径
