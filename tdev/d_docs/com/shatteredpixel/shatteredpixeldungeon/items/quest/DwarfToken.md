## 2. 文件职责说明

### 核心职责
矮人徽记是一个简单的任务物品，用于在特定任务中收集。它仅作为任务进度的计数物品，没有特殊的游戏逻辑。

### 系统定位
该类属于任务物品系统的基础部分，代表一种可以在矮人相关任务中收集的标识物品。

### 不负责什么
- 不处理任何特殊的游戏逻辑
- 不管理任务进度（由外部系统管理）
- 不提供任何战斗或交互功能

## 6. 构造与初始化机制

### 构造器
使用默认构造器，通过实例初始化块进行初始化。

### 初始化块
```java
{
    image = ItemSpriteSheet.TOKEN;
    stackable = true;
    unique = true;
}
```
- 设置物品图像为徽记图标
- 标记为可堆叠（允许多个矮人徽记堆叠在一起）
- 标记为唯一物品（与其他物品类型区分）

### 初始化注意事项
- 作为简单任务物品，不需要复杂的初始化逻辑
