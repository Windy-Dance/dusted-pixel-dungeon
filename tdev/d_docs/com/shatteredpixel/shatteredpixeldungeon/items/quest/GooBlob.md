## 2. 文件职责说明

### 核心职责
粘咕球是一个任务物品，可以从粘咕（Goo）怪物的尸体上获得。它具有经济价值和炼金能量值，可用于商店出售或炼金合成。

### 系统定位
该类属于任务物品系统的一部分，代表一种可以从特定敌人掉落的资源物品。

### 不负责什么
- 不负责生成粘咕球（由敌人掉落系统生成）
- 不处理任何特殊的游戏逻辑
- 不提供战斗或交互功能

## 7. 方法详解

### isUpgradable()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：确定物品是否可升级

**参数**：无

**返回值**：boolean，始终返回false

**前置条件**：无


### isIdentified()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：确定物品是否已鉴定

**参数**：无

**返回值**：boolean，始终返回true

**前置条件**：无

**副作用**：无

**核心实现逻辑**：
```java
return true;
```

**边界情况**：无

### value()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：返回物品的金币价值

**参数**：无

**返回值**：int，每个粘咕球价值30金币，总价为数量×30

**前置条件**：无

**副作用**：无

**核心实现逻辑**：
```java
return quantity * 30;
```

**边界情况**：quantity为0时返回0

### energyVal()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：返回物品的炼金能量值

**参数**：无

**返回值**：int，每个粘咕球提供3点炼金能量，总能量为数量×3

**前置条件**：无

**副作用**：无

**核心实现逻辑**：
```java
return quantity * 3;
```

**边界情况**：quantity为0时返回0

## 11. 使用示例

### 基本用法
```java

// 查询炼金能量值
int energyValue = blob.energyVal(); // 返回 6
```
