## 2. 文件职责说明

### 核心职责
元素余烬是一个任务物品，用于召唤元素生物。它具有特殊的发光效果，表示其中蕴含的魔法能量。

### 系统定位
该类属于任务物品系统的一部分，专门用于处理与元素余烬相关的视觉效果和任务逻辑。

### 不负责什么
- 不负责生成余烬（由任务系统生成）
- 不处理召唤元素的具体逻辑（由其他系统处理）
- 不管理任务进度

使用默认构造器，通过实例初始化块进行初始化。

### 初始化块
```java

    image = ItemSpriteSheet.EMBER;
    unique = true;
}
```
- 设置物品图像为余烬图标
- 标记为唯一物品（不会与其他物品堆叠）

### 初始化注意事项
- 余烬不可堆叠，每次只能持有一个
- 具有特殊的红色发光效果

## 7. 方法详解

### isUpgradable()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：确定物品是否可升级

**参数**：无

**返回值**：boolean，始终返回false

**前置条件**：无

**副作用**：无

**核心实现逻辑**：
```java
return false;
```

**边界情况**：无

### isIdentified()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：确定物品是否已鉴定

**参数**：无

**返回值**：boolean，始终返回true

**前置条件**：无

**副作用**：无

**核心实现逻辑**：
```java
return true;
```

**边界情况**：无

### glowing()
**可见性**：public

**是否覆写**：是，覆写自 Item

**方法职责**：返回物品的发光效果

**参数**：无

**返回值**：ItemSprite.Glowing，红色发光效果对象

**前置条件**：物品正在被渲染

**副作用**：无

**核心实现逻辑**：
```java
return new ItemSprite.Glowing(0x660000, 3f);
```
- 创建深红色（0x660000）发光效果
- 发光强度为3f

**边界情况**：无
