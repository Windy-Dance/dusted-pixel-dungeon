# Gold.java - 金币

## 概述
`Gold` 类代表游戏中的货币系统，用于购买物品、升级装备等经济活动。金币具有可堆叠特性，并在拾取时自动添加到玩家的总金币数中。

## 核心特性

### 货币管理
- **自动累加**: 拾取时自动增加 `Dungeon.gold` 全局金币数量
- **统计数据**: 更新 `Statistics.goldCollected` 统计数据
- **成就验证**: 触发金币收集相关的成就 (`Badges.validateGoldCollected()`)

### 可堆叠系统
- **stackable = true**: 支持堆叠多个金币为单个物品
- **quantity**: 表示金币的数量值
- 拾取时会自动合并到现有的金币堆栈中

## 物品属性

### 基础设置
- 使用 `ItemSpriteSheet.GOLD` 图像
- **isUpgradable()**: 返回false（不可升级）
- **isIdentified()**: 返回true（始终已识别）
- **actions()**: 返回空列表（无法进行任何操作）

### 随机生成
- **random()**: 根据当前地牢深度生成随机数量的金币
  - 数量范围：30 + depth×10 到 60 + depth×20
  - 确保随着游戏进度金币奖励逐渐增加

## 视觉和音效反馈

### UI表现
- **拾取动画**: 显示金币拾取特效 (`GameScene.pickUp`)
- **状态显示**: 在英雄头顶显示获得的金币数量 (`FloatingText.GOLD`)
- **快捷栏更新**: 自动更新快捷栏显示

### 音效系统
- 播放金币音效 (`Assets.Sounds.GOLD`)
- 音调随机变化 (0.9f 到 1.1f) 增加听觉多样性

## 使用场景
- **敌人掉落**: 击败敌人后掉落金币
- **宝箱奖励**: 开启宝箱获得金币
- **商店交易**: 向商店出售物品获得金币
- **随机生成**: 地牢中的随机金币堆

## 注意事项
1. 金币拾取后立即从地面消失并添加到玩家总金币中
2. 金币不占用背包空间，直接转换为全局金币数量
3. 金币的Catalog记录会在首次拾取时自动标记为已见
4. 金币的随机生成数量与地牢深度成正比，确保游戏平衡性
