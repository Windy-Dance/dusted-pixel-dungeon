# RainbowParticle 类

## 概述
`RainbowParticle` 类是 Shattered Pixel Dungeon 游戏中的彩虹粒子效果类。它继承自 `PixelParticle`，专门用于表现彩虹、幻彩、庆祝或特殊魔法的多彩视觉效果。

## 功能描述
该类实现了彩虹粒子的核心特性，具有以下特点：
- **随机颜色**: 每个粒子都有完全随机的颜色（1600万色中随机选择）
- **尺寸变化**: 粒子大小从1像素逐渐增大到5像素
- **双重模式**: 支持普通随机运动和爆发放射状运动两种效果
- **爆发工厂**: 提供专门的爆发工厂实例用于高速放射状效果
- **光照混合**: 爆发模式使用光照混合模式增强视觉效果

## 核心字段

### 工厂实例
- **BURST**: 爆发工厂实例，用于彩虹爆发或庆祝效果
  - `emit()`: 调用 `resetBurst()` 方法创建高速放射状粒子
  - `lightMode()`: 返回 true，使用光照混合模式增强发光效果

### 视觉属性
- **color**: 在构造函数中随机设置（0到0x1000000范围内的任意颜色）
- **lifespan**: 生命周期固定为0.5秒

## 构造函数

### RainbowParticle()
**功能**: 创建基础彩虹粒子实例

**实现细节**:
- 调用父类构造函数初始化基础 particle
- **随机颜色**: 使用 `Random.Int(0x1000000)` 设置完全随机的颜色
- 配置生命周期为0.5秒

## 核心方法

### reset(float x, float y)
**功能**: 重置并配置普通彩虹粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.5秒)
4. **速度配置**: 
   - 水平速度: 在-5到+5像素/秒范围内随机选择
   - 垂直速度: 在-5到+5像素/秒范围内随机选择
   - 创造缓慢的随机飘动效果

### resetBurst(float x, float y)
**功能**: 重置并配置爆发彩虹粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.5秒)
4. **速度配置**: 
   - 随机角度: 在完整圆周范围内 (0到2π) 随机选择
   - 随机速度: 在16-32像素/秒范围内随机选择（高速放射状）

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的运动逻辑
- **尺寸控制**: 
  - 公式: `size(5 - am * 4)`
  - 粒子从1像素逐渐增大到5像素
- **颜色保持**: 颜色在生命周期内保持不变（在构造时随机设定）

## 技术特点

### 视觉多样性
- **完全随机**: 每个粒子都有独特的颜色，创造丰富的视觉效果
- **色彩丰富**: 1600万色的全色域支持确保视觉冲击力
- **尺寸增强**: 增大的尺寸配合随机颜色创造绚丽的视觉效果

### 双重行为模式
- **普通模式**: 缓慢、随机的运动适合表现持续的彩虹效果
- **爆发模式**: 快速、放射状的运动适合表现庆祝或魔法爆炸效果

### 运动特性
- **随机飘动**: 普通模式的低速随机运动创造自然的漂浮感
- **爆炸扩散**: 爆发模式的高速放射状运动创造强烈的视觉冲击
- **简单高效**: 不进行颜色插值，保持性能优化

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **一次性颜色**: 颜色在构造时设定，避免运行时计算开销
- **自动销毁**: 粒子生命周期结束后自动失效
