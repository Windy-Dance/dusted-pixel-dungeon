# Chains 类

## 概述
`Chains` 类是 Shattered Pixel Dungeon 游戏中用于创建链条连接效果的视觉组件类。它继承自 `Group`，能够在两个指定点之间动态显示一系列链条图像，形成从起点向终点延伸的动画效果。

## 功能描述
该类实现了链条效果的完整动画系统，包括链条的创建、定位、旋转和动画播放。常用于表现角色之间的连接、束缚效果或魔法链等视觉表现。

## 技术原理
- **数学计算**: 使用三角函数计算链条的方向和角度
- **线性插值**: 通过时间进度控制链条的延伸动画
- **动态数量**: 根据距离自动计算需要的链条数量
- **回调机制**: 动画完成后触发回调函数

## 核心常量和字段

### 数学常量
- **A**: 弧度到角度的转换系数 (`180 / Math.PI`)

### 时间控制
- **spent**: 已经过的时间（秒）
- **duration**: 动画总持续时间（秒）

### 链条配置
- **callback**: 动画完成后的回调函数
- **chains**: 链条图像数组
- **numChains**: 链条数量
- **distance**: 起点到终点的距离（像素）
- **rotation**: 链条的旋转角度（度）
- **from/to**: 起点和终点的世界坐标

## 构造函数

### Chains(int from, int to, Effects.Type type, Callback callback)
**功能**: 从地图格子索引创建链条效果

**参数说明**:
- `from`: 起始格子索引
- `to`: 目标格子索引  
- `type`: 链条类型（来自 `Effects.Type` 枚举）
- `callback`: 动画完成回调

**实现细节**:
- 使用 `DungeonTilemap.tileCenterToWorld()` 将格子索引转换为中心世界坐标
- 调用另一个构造函数进行实际初始化

### Chains(PointF from, PointF to, Effects.Type type, Callback callback)
**功能**: 从世界坐标创建链条效果

**实现细节**:
1. **距离计算**: 使用 `Math.hypot()` 计算欧几里得距离
2. **持续时间**: 基础 200ms + 每格子 50ms (`distance/320f + 0.2f`)
3. **角度计算**: 使用 `Math.atan2()` 计算方向，转换为角度并加 90° 调整
4. **链条数量**: 每 6 像素一个链条 (`Math.round(distance/6f)+1`)
5. **图像创建**: 为每个链条创建 `Image` 对象，设置角度和原点

## 核心方法

### update()
**功能**: 每帧更新链条动画状态

**实现逻辑**:
1. **时间检查**: 如果已过时间超过持续时间：
   - 调用 `killAndErase()` 销毁对象
   - 执行回调函数（如果存在）
2. **动画更新**: 否则计算每个链条的位置：
   - 使用线性插值：`(起点 + (方向 * 链条比例) * 时间进度)`
   - 调用 `center()` 方法设置每个链条的中心位置

**位置计算公式**:
```
x = from.x + ((dx * (i / chains.length)) * (spent/duration))
y = from.y + ((dy * (i / chains.length)) * (spent/duration))
```

其中：
- `dx/dy`: 方向向量分量
- `i / chains.length`: 当前链条在链条序列中的比例位置
- `spent/duration`: 动画当前进度

## 视觉特性
- **自动旋转**: 链条会根据连接方向自动旋转对齐
- **平滑动画**: 线性插值确保动画流畅自然
- **可变密度**: 距离越长，链条数量越多，保持视觉一致性
- **精确对齐**: 原点设置在底部中心，确保链条正确连接

## 支持的链条类型
通过 `Effects.Type` 参数支持以下类型：
- **CHAIN**: 普通链条效果
- **ETHEREAL_CHAIN**: 灵体链条效果（透明或发光）

## 使用场景
该类主要用于以下游戏效果：
- **角色束缚**: 敌人被魔法链条束缚的效果
- **技能连接**: 连接两个目标的魔法技能视觉表现  
- **环境互动**: 表现机关或陷阱的链条机制
- **BOSS 技能**: BOSS 的连锁攻击或控制技能

## 性能特点
- **自动清理**: 动画完成后自动销毁，避免内存泄漏
- **高效渲染**: 使用 `Group` 容器管理多个图像，优化渲染批次
- **精确计时**: 基于游戏帧率的时间控制，确保跨设备一致性

## 相关类
- **Effects**: 提供链条纹理资源
- **DungeonTilemap**: 提供坐标转换功能  
- **Group**: 容器基类，提供批量管理功能
- **Image**: 图像基类，处理纹理和变换
- **PointF**: 二维坐标类，存储位置信息
- **Callback**: 回调接口，处理动画完成事件
