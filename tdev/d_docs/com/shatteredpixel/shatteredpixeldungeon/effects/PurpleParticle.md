# PurpleParticle 类

## 概述
`PurpleParticle` 类是 Shattered Pixel Dungeon 游戏中的紫色粒子效果类。它继承自 `PixelParticle`，专门用于表现诅咒、暗影魔法、传送或神秘仪式等紫色系的视觉效果。

## 功能描述
该类实现了紫色粒子的两种不同行为模式，具有以下核心特性：
- **颜色渐变**: 粒子颜色从红色 (0xFF0044) 逐渐变为深紫色 (0x220066)
- **尺寸变化**: 粒子大小从1像素逐渐增大到5像素
- **双重模式**: 支持导弹轨迹和爆发扩散两种效果
- **随机运动**: 导弹模式使用随机低速运动，爆发模式使用高速放射状运动
- **工厂模式**: 提供两种不同的工厂实例适应不同使用场景

## 核心字段

### 工厂实例
- **MISSILE**: 导弹工厂实例，用于紫色魔法投射物的轨迹效果
  - `emit()`: 调用 `reset()` 方法创建随机低速运动的粒子
  - `lightMode()`: 默认false，使用正常混合模式
  
- **BURST**: 爆发工厂实例，用于紫色魔法爆发效果  
  - `emit()`: 调用 `resetBurst()` 方法创建高速放射状粒子
  - `lightMode()`: 返回 true，使用光照混合模式增强视觉效果

### 视觉属性
- **lifespan**: 生命周期固定为0.5秒

## 构造函数

### PurpleParticle()
**功能**: 创建基础紫色粒子实例

**实现细节**:
- 调用父类构造函数初始化基础 particle
- 配置生命周期为0.5秒

## 核心方法

### reset(float x, float y)
**功能**: 重置并配置导弹轨迹粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.5秒)
4. **速度配置**: 
   - 水平速度: 在-5到+5像素/秒范围内随机选择
   - 垂直速度: 在-5到+5像素/秒范围内随机选择
   - 创造缓慢的随机飘动效果

### resetBurst(float x, float y)
**功能**: 重置并配置爆发扩散粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.5秒)
4. **速度配置**: 
   - 随机角度: 在完整圆周范围内 (0到2π) 随机选择
   - 随机速度: 在16-32像素/秒范围内随机选择（高速放射状）

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的运动逻辑
- **尺寸控制**: 
  - 公式: `size(5 - am * 4)`
  - 粒子从1像素逐渐增大到5像素
- **颜色插值**: 
  - 使用 `ColorMath.interpolate()` 在两种颜色间渐变
  - 起始颜色: 深紫色 (0x220066)
  - 结束颜色: 红色 (0xFF0044)
  - 插值因子: `am = left / lifespan`

## 技术特点

### 视觉渐变
- **反直觉设计**: 粒子在消失前变得更大更亮，强化神秘效果
- **颜色语义**: 从红色到深紫色的渐变表现魔法能量的凝聚过程
- **尺寸增强**: 增大的尺寸配合光照混合创造强烈的视觉冲击力

### 双重行为模式
- **导弹模式**: 缓慢、随机的运动适合表现持续的魔法轨迹
- **爆发模式**: 快速、放射状的运动适合表现魔法爆炸或传送效果

### 运动特性
- **随机飘动**: 导弹模式的低速随机运动创造神秘的漂浮感
- **爆炸扩散**: 爆发模式的高速放射状运动创造强烈的视觉冲击
- **速度差异**: 不同模式使用截然不同的速度范围以匹配使用场景

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **高效插值**: 使用预计算的颜色插值避免实时计算开销
- **自动销毁**: 粒子生命周期结束后自动失效
