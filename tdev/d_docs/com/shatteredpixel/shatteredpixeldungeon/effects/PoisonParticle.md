# PoisonParticle 类

## 概述
`PoisonParticle` 类是 Shattered Pixel Dungeon 游戏中的毒素粒子效果类。它继承自 `PixelParticle`，专门用于表现毒素、毒液或腐蚀性液体的视觉效果，具有独特的颜色渐变和尺寸变化特性。

## 功能描述
该类实现了毒素粒子的两种不同行为模式，具有以下核心特性：
- **颜色渐变**: 粒子颜色从紫色 (0x8844FF) 逐渐变为绿色 (0x00FF00)
- **尺寸变化**: 粒子大小从1像素逐渐增大到4像素
- **双重模式**: 支持导弹轨迹和飞溅扩散两种效果
- **重力效果**: 应用向下的加速度模拟液体下落
- **工厂模式**: 提供两种不同的工厂实例适应不同使用场景

## 核心字段

### 工厂实例
- **MISSILE**: 导弹工厂实例，用于毒素投射物的轨迹效果
  - `emit()`: 调用 `resetMissile()` 方法创建缓慢移动的粒子
  - `lightMode()`: 返回 true，使用光照混合模式
  
- **SPLASH**: 飞溅工厂实例，用于毒素液体飞溅效果  
  - `emit()`: 调用 `resetSplash()` 方法创建快速扩散的粒子
  - `lightMode()`: 返回 true，使用光照混合模式

### 视觉属性
- **lifespan**: 生命周期固定为0.6秒
- **acc**: 加速度设置为 (0, +30)，模拟向下的重力效果

## 构造函数

### PoisonParticle()
**功能**: 创建基础毒素粒子实例

**实现细节**:
- 调用父类构造函数初始化基础粒子
- 配置生命周期为0.6秒
- 设置向下的加速度 (+30) 模拟液体重力

## 核心方法

### resetMissile(float x, float y)
**功能**: 重置并配置导弹轨迹粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.6秒)
4. **速度配置**: 
   - 随机角度: 在下半圆范围内 (-π到0) 随机选择（确保向下运动）
   - 随机速度: 在0-6像素/秒范围内随机选择（较慢的速度）

### resetSplash(float x, float y)
**功能**: 重置并配置飞溅扩散粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.6秒)
4. **速度配置**: 
   - 随机角度: 在上半圆范围内 (0到π) 随机选择
   - 随机速度: 在10-20像素/秒范围内随机选择（较快的速度）

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的运动逻辑
- **尺寸控制**: 
  - 公式: `size(4 - am * 3)`
  - 粒子从1像素逐渐增大到4像素
- **颜色插值**: 
  - 使用 `ColorMath.interpolate()` 在两种颜色间渐变
  - 起始颜色: 绿色 (0x00FF00)
  - 结束颜色: 紫色 (0x8844FF)
  - 插值因子: `am = left / lifespan`

## 技术特点

### 视觉渐变
- **反直觉设计**: 与大多数粒子相反，毒素粒子在消失前变得更大更亮
- **颜色语义**: 从紫色到绿色的渐变表现毒素的化学反应过程
- **尺寸增强**: 增大的尺寸强化了毒素效果的视觉冲击力

### 双重行为模式
- **导弹模式**: 缓慢、受控的运动适合表现持续的毒素轨迹
- **飞溅模式**: 快速、扩散的运动适合表现爆炸性的毒素溅射

### 物理模拟
- **重力影响**: 向下的加速度使粒子轨迹更加真实
- **速度差异**: 不同模式使用不同的速度范围以匹配视觉需求
- **角度限制**: 导弹模式限制在下半圆，确保效果的合理性

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **高效插值**: 使用预计算的颜色插值避免实时计算开销
- **自动销毁**: 粒子生命周期结束后自动失效

## 使用场景

### 战斗系统
- **毒素伤害**: 毒液攻击、毒素魔法等技能的视觉表现
- **持续伤害**: 中毒debuff的持续视觉反馈
- **环境伤害**: 毒素地形或陷阱的效果表现

### 技能特效
- **投掷技能**: 毒素药水或毒液瓶的飞行轨迹
- **爆炸技能**: 毒素炸弹或魔法的飞溅效果
- **范围技能**: 毒雾或毒素领域的粒子表现

### UI系统
- **状态指示**: 中毒状态的持续视觉提醒
- **伤害数字**: 毒素伤害数字的伴随效果
- **警告提示**: 毒素危险区域的标记效果

### 特殊事件
- **Boss技能**: 强大敌人的毒素攻击特效
- **环境互动**: 角色接触毒素物质时的反应效果
- **物品效果**: 毒素装备或消耗品的使用效果
