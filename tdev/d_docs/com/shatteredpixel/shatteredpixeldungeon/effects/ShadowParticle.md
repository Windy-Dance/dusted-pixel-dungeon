# ShadowParticle 类

## 概述
`ShadowParticle` 类是 Shattered Pixel Dungeon 游戏中的暗影粒子效果类。它继承自 `PixelParticle.Shrinking`，专门用于表现诅咒、黑暗魔法、死亡或邪恶仪式等暗影系的视觉效果。

## 功能描述
该类实现了暗影粒子的三种不同行为模式，具有以下核心特性：
- **颜色渐变**: 粒子颜色从深紫色 (0x440044) 逐渐变为纯黑色 (0x000000)
- **脉冲透明度**: 透明度先快速增加后线性减少，创造闪烁效果
- **三重模式**: 支持导弹轨迹、诅咒爆发和上升扩散三种效果
- **随机运动**: 不同模式使用不同的速度和方向配置
- **工厂模式**: 提供三种不同的工厂实例适应不同使用场景

## 核心字段

### 工厂实例
- **MISSILE**: 导弹工厂实例，用于暗影魔法投射物的轨迹效果
  - `emit()`: 调用 `reset()` 方法创建随机低速运动的粒子
  
- **CURSE**: 诅咒工厂实例，用于诅咒爆发效果  
  - `emit()`: 调用 `resetCurse()` 方法创建高速放射状粒子
  
- **UP**: 上升工厂实例，用于暗影能量上升效果
  - `emit()`: 调用 `resetUp()` 方法创建向上快速运动的粒子

### 视觉属性
- **颜色渐变**: 从深紫色 (0x440044) 到黑色 (0x000000)
- **透明度**: 脉冲式变化（先加速增加后线性减少）
- **尺寸**: 不同模式使用不同初始大小

## 构造函数

### ShadowParticle()
**功能**: 暗影粒子没有显式的构造函数，继承自父类

**实现细节**:
- 继承 `PixelParticle.Shrinking` 的收缩动画特性
- 颜色和透明度在 `update()` 方法中动态计算

## 核心方法

### reset(float x, float y)
**功能**: 重置并配置导弹轨迹粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **速度配置**: 
   - 水平速度: 在-5到+5像素/秒范围内随机选择
   - 垂直速度: 在-5到+5像素/秒范围内随机选择
   - 创造缓慢的随机飘动效果
4. **尺寸初始化**: 设置初始大小为6像素
5. **生命周期**: 设置生命周期为0.5秒

### resetCurse(float x, float y)
**功能**: 重置并配置诅咒爆发粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **尺寸初始化**: 设置初始大小为8像素（比普通模式更大）
3. **生命周期**: 设置生命周期为0.5秒
4. **速度配置**: 
   - 随机角度: 在完整圆周范围内 (0到2π) 随机选择
   - 随机速度: 在16-32像素/秒范围内随机选择（高速放射状）
5. **位置计算**: 
   - 计算起点位置确保粒子在生命周期结束时到达目标点
   - 公式: `x = targetX - speed.x * lifespan`

### resetUp(float x, float y)
**功能**: 重置并配置上升扩散粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **速度配置**: 
   - 水平速度: 在-8到+8像素/秒范围内随机选择
   - 垂直速度: 在-32到-48像素/秒范围内随机选择（强烈向上）
3. **位置设置**: 设置粒子的初始坐标
4. **尺寸初始化**: 设置初始大小为6像素
5. **生命周期**: 设置较长的生命周期为1秒

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的收缩和运动逻辑
- **颜色插值**: 
  - 使用 `ColorMath.interpolate()` 在深紫和黑色间渐变
  - 起始颜色: 黑色 (0x000000)
  - 结束颜色: 深紫色 (0x440044)
  - 插值因子: `p = left / lifespan`
- **透明度控制**: 
  - 前半段 (p < 0.5f): `am = p * p * 4` (二次加速增加)
  - 后半段: `am = (1 - p) * 2` (线性减少)
  - 创造脉冲式的闪烁效果

## 技术特点

### 视觉渐变
- **颜色语义**: 深紫色到黑色的渐变表现暗影能量的凝聚和消散
- **脉冲透明度**: 复杂的透明度曲线创造生动的闪烁效果
- **尺寸差异**: 不同模式使用不同尺寸以匹配视觉需求

### 三重行为模式
- **导弹模式**: 缓慢、随机的运动适合表现持续的暗影轨迹
- **诅咒模式**: 快速、放射状的运动适合表现诅咒爆发效果
- **上升模式**: 强烈向上的运动适合表现暗影能量升腾效果

### 运动特性
- **随机飘动**: 导弹模式的低速随机运动创造神秘的漂浮感
- **爆炸扩散**: 诅咒模式的高速放射状运动创造强烈的视觉冲击
- **能量升腾**: 上升模式的强烈向上运动表现暗影能量的升腾

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **高效插值**: 使用预计算的颜色插值避免实时计算开销
- **自动销毁**: 粒子生命周期结束后自动失效

## 使用场景

### 暗影魔法
- **诅咒效果**: 诅咒魔法或debuff的视觉表现

- **死亡魔法**: 死亡相关的魔法视觉表现
- **邪恶仪式**: 各种邪恶仪式的视觉表现

### 战斗系统
- **魔法攻击**: 暗影系魔法技能的视觉反馈
- **范围技能**: 暗影领域或结界的粒子表现
- **Boss技能**: 强大暗影Boss的特殊技能效果

### UI系统
- **状态指示**: 诅咒或暗影状态的持续视觉提醒
- **技能冷却**: 暗影技能冷却时的视觉表现
- **警告提示**: 危险区域或邪恶存在的标记效果

### 特殊事件
- **剧情演出**: 关键暗影时刻的视觉强化
- **隐藏要素**: 暗影相关的谜题或机关提示效果
- **环境互动**: 角色与暗影环境的互动效果

- **Emitter**: 粒子发射器，管理粒子池和发射逻辑  
- **ColorMath**: 颜色数学工具，提供颜色插值功能
- **Random**: 随机数生成器，用于行为的随机化
- **PointF**: 2D点坐标类，提供数学常数 PI2 和向量操作
