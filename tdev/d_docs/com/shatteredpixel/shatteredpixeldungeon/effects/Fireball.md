# Fireball 类

## 概述
`Fireball` 类是 Shattered Pixel Dungeon 游戏中用于显示火球动画效果的电影剪辑类。它继承自 `MovieClip`，根据设备方向自动选择合适的火球动画纹理，并支持创建两个略有差异的火球实例以增加视觉多样性。

## 功能描述
该类实现了火球爆炸效果的完整动画系统，包括：
- **方向适配**: 根据横屏/竖屏模式选择不同尺寸的纹理
- **视觉变化**: 创建交替的火球效果（翻转和动画偏移）
- **预设动画**: 内置24帧的循环火球动画

## 核心特性

### 纹理适配
- **横屏模式**: 使用 `effects/fireball-tall.png` (61x61 像素/帧)
- **竖屏模式**: 使用 `effects/fireball-short.png` (47x47 像素/帧)
- **自动检测**: 通过 `PixelScene.landscape()` 判断当前设备方向

### 视觉多样性
- **交替实例**: 静态变量 `second` 跟踪创建顺序
- **第二火球**: 
  - 水平翻转 (`flipHorizontal = true`)
  - 动画偏移12帧 (`curFrame = 12`)
  - 提供视觉变化避免重复感

## 构造函数

### Fireball()
**功能**: 默认构造函数，自动处理交替逻辑

**实现细节**:
- 调用重载构造函数并传入当前的 `second` 状态
- 切换 `second` 状态为下次调用准备

### Fireball(boolean second)
**功能**: 基础构造函数，根据参数创建特定类型的火球

**实现逻辑**:
1. **纹理选择**: 根据屏幕方向选择合适的火球纹理
2. **帧设置**: 创建 `TextureFilm` 定义每帧的尺寸
3. **动画配置**: 创建24帧的循环动画
4. **特殊处理**: 如果是第二个火球，应用翻转和帧偏移

## 技术特点
- **资源优化**: 为不同屏幕方向提供优化的纹理尺寸
- **内存效率**: 复用现有的 `MovieClip` 动画系统
- **视觉增强**: 通过简单的变换创造丰富的视觉效果
- **自动管理**: 动画播放和循环由父类自动处理

## 使用场景
该类主要用于以下游戏效果：
- **爆炸技能**: 火球术、爆裂魔法等技能的视觉表现
- **环境效果**: 爆炸陷阱、火焰机关的动画
- **敌人攻击**: 火属性敌人的远程攻击效果
- **UI反馈**: 某些界面元素的动态装饰

## 相关类
- **MovieClip**: 动画基类，提供帧动画功能
- **TextureFilm**: 纹理胶片类，管理纹理帧
- **PixelScene**: 场景工具类，提供方向检测
- **Animation**: 动画配置类，定义播放参数

## 设计考量
- **性能优先**: 预渲染的精灵表比实时粒子更高效
- **适应性**: 自动适配不同设备方向
- **简洁性**: 最小化代码复杂度，专注于核心功能
- **可扩展**: 易于添加新的火球变体或动画效果
