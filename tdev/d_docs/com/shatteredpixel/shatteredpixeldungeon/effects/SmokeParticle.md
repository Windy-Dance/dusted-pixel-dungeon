# SmokeParticle 类

## 概述
`SmokeParticle` 类是 Shattered Pixel Dungeon 游戏中的烟雾粒子效果类。它继承自 `PixelParticle`，专门用于表现烟雾、蒸汽、爆炸烟尘或毒气云等各种烟雾相关的视觉效果。

## 功能描述
该类实现了烟雾粒子的两种不同行为模式，具有以下核心特性：
- **双重视觉**: 支持普通烟雾和喷射烟雾两种不同的视觉效果
- **深灰色调**: 使用固定的深灰色 (0x222222) 表现真实的烟雾颜色
- **反重力效果**: 应用向上的加速度模拟烟雾上升的物理特性
- **随机生命周期**: 每个粒子的生命周期在0.6-1秒之间随机变化
- **工厂模式**: 提供两种不同的工厂实例适应不同使用场景

## 核心字段

### 工厂实例
- **FACTORY**: 普通烟雾工厂实例
  - `emit()`: 调用 `reset()` 方法创建随机飘动的烟雾粒子
  
- **SPEW**: 喷射烟雾工厂实例  
  - `emit()`: 调用 `resetSpew()` 方法创建侧向喷射的烟雾粒子
  - 适用于定向喷射或爆炸烟尘效果

### 视觉属性
- **color**: 固定为深灰色 (0x222222)，表现真实烟雾的颜色
- **acc**: 加速度设置为 (0, -40)，模拟向上的烟雾上升效果

## 构造函数

### SmokeParticle()
**功能**: 创建基础烟雾粒子实例

**实现细节**:
- 调用父类构造函数初始化基础 particle
- 设置固定的颜色为深灰色 (0x222222)
- 配置向上的加速度 (-40) 模拟烟雾自然上升

## 核心方法

### reset(float x, float y)
**功能**: 重置并配置普通烟雾粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 
   - 随机设置生命周期 (0.6-1秒之间的随机值)
   - 创造自然的烟雾持续时间变化
4. **速度配置**: 
   - 水平速度: 在-4到+4像素/秒范围内随机选择
   - 垂直速度: 在-8到+8像素/秒范围内随机选择
   - 创造缓慢的随机飘动效果

### resetSpew(float x, float y)
**功能**: 重置并配置喷射烟雾粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **位置设置**: 设置粒子的初始坐标
3. **加速度调整**: 
   - X轴加速度: -40 (向左加速)
   - Y轴加速度: +40 (向下加速)
   - 创造侧向喷射的抛物线轨迹
4. **生命周期**: 随机设置生命周期 (0.6-1秒)
5. **速度配置**: 
   - 角度范围: 在306°-324°范围内 (5.4π-5.6π弧度，右下方向)
   - 速度范围: 在30-60像素/秒范围内 (高速喷射)

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的运动逻辑
- **透明度控制**: 
  - 前80%时间: `am = p * 0.5f` (线性淡入，最大0.5透明度)
  - 后20%时间: `am = 2 - 2*p` (线性淡出到0)
  - 创造自然的烟雾出现和消散效果
- **尺寸控制**: 
  - 公式: `size(16 - p * 8)`
  - 粒子从16像素逐渐缩小到8像素

## 技术特点

### 物理模拟
- **烟雾上升**: 向上的加速度使粒子自然向上飘动
- **喷射轨迹**: SPEW模式的特殊加速度创造真实的喷射抛物线
- **随机性**: 生命周期和速度的随机性增加视觉自然感

### 视觉设计
- **颜色语义**: 深灰色准确表现各种类型的烟雾效果
- **透明度曲线**: 特殊的透明度曲线模拟烟雾的扩散和消散过程
- **尺寸变化**: 缩小的尺寸配合透明度创造烟雾扩散效果

### 双重模式
- **普通模式**: 适合持续的烟雾效果，如篝火烟雾、蒸汽等
- **喷射模式**: 适合爆炸、喷射或定向释放的烟雾效果

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **简单计算**: 使用高效的随机数生成和线性插值
- **自动销毁**: 粒子生命周期结束后自动失效

通过双重模式精确适配不同的游戏场景需求，从持续烟雾到爆发喷射。

### 视觉层次
