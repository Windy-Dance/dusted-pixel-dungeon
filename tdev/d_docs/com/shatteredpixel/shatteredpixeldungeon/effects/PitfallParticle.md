# PitfallParticle 类

## 概述
`PitfallParticle` 类是 Shattered Pixel Dungeon 游戏中的陷阱粒子效果类。它继承自 `PixelParticle.Shrinking`，专门用于表现陷阱触发、地面塌陷或黑洞相关的黑色粒子视觉效果。

## 功能描述
该类实现了陷阱粒子的核心行为，具有以下特点：
- **收缩动画**: 粒子在生命周期内逐渐缩小
- **纯黑颜色**: 使用固定的纯黑色 (0x000000) 表现陷阱或虚空效果
- **随机角度**: 每个粒子都有轻微的随机旋转角度增加视觉变化
- **双尺寸模式**: 支持4像素和8像素两种不同大小的粒子效果

## 核心字段

### 工厂实例
- **FACTORY4**: 小型陷阱粒子工厂，创建4像素大小的粒子
- **FACTORY8**: 大型陷阱粒子工厂，创建8像素大小的粒子

### 视觉属性
- **color**: 固定为纯黑色 (0x000000)
- **angle**: 随机角度在-30到+30度范围内
- **lifespan**: 生命周期固定为1秒

## 构造函数

### PitfallParticle()
创建基础陷阱粒子实例，设置黑色和随机角度。

## 核心方法

### reset(float x, float y, int size)
重置粒子状态，设置位置、1秒生命周期和指定尺寸。

## 技术特点

### 视觉设计
- 纯黑色表示危险、陷阱或虚空
- 两种尺寸适应不同视觉需求
- 随机角度避免粒子完全对齐

### 性能优化
- 对象池复用 (`emitter.recycle()`)
- 简单配置通过工厂模式
- 自动销毁生命周期结束后

## 使用场景

### 陷阱系统
- 地面陷阱触发效果
- 坑洞或地面塌陷视觉表现
- 虚空或黑洞陷阱效果

### 战斗系统
- 暗影攻击视觉表现
- 吞噬或吸收技能效果
- 黑暗区域伤害反馈

### 特殊事件
- 暗影系Boss特殊技能
- 关键陷阱场景视觉强化
- 隐藏陷阱提示效果
