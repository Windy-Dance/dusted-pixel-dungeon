# ShaftParticle 类

## 概述
`ShaftParticle` 类是 Shattered Pixel Dungeon 游戏中的光柱粒子效果类。它继承自 `PixelParticle`，专门用于表现从上而下的神圣光线、天降光束或启示效果的视觉表现。

## 功能描述
该类实现了光柱粒子的核心行为，具有以下特点：
- **垂直运动**: 粒子以恒定速度向上缓慢运动
- **随机偏移**: 每个粒子都有随机的时间偏移，创造错落有致的效果
- **缩放动画**: X轴宽度和Y轴高度都随时间变化，创造动态光柱效果
- **淡入淡出**: 透明度先增加后减少，创造自然的出现和消失效果
- **光照混合**: 使用光照混合模式增强发光效果

## 核心字段

### 工厂实例
- **FACTORY**: 静态工厂实例，负责粒子的创建和回收
  - `emit()`: 从发射器回收并重置粒子
  - `lightMode()`: 返回 true，使用光照混合模式增强发光效果

### 视觉属性  
- **lifespan**: 生命周期固定为1.2秒（较长的持续时间）
- **speed**: 固定向上速度 (0, -6)，创造缓慢上升效果
- **offs**: 随机时间偏移，用于错开粒子的生命周期

## 构造函数

### ShaftParticle()
**功能**: 创建基础光柱粒子实例

**实现细节**:
- 调用父类构造函数初始化基础 particle
- 配置较长的生命周期为1.2秒
- 设置固定的向上速度 (-6像素/秒)，创造缓慢上升效果

## 核心方法

### reset(float x, float y)
**功能**: 重置并配置粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **时间偏移**: 
   - 生成随机偏移值 `offs = -Random.Float(lifespan)`
   - 范围: 0到-1.2秒之间的随机值
4. **生命周期调整**: 
   - 实际生命周期: `left = lifespan - offs`
   - 确保粒子在不同时间点开始和结束

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父 class 的 `update()` 处理基础的运动逻辑
- **透明度控制**: 
  - 前半段 (p < 0.5f): `am = p` (线性增加)
  - 后半段: `am = 1 - p` (线性减少)
  - 创造淡入淡出的自然效果
- **X轴缩放**: 
  - 公式: `scale.x = (1 - p) * 4`
  - 宽度从4倍逐渐缩小到0倍
- **Y轴缩放**: 
  - 公式: `scale.y = 16 + (1 - p) * 16`
  - 高度从32像素逐渐缩小到16像素

## 技术特点

### 运动设计
- **恒定速度**: 固定的向上速度确保所有光柱运动一致
- **时间错位**: 随机时间偏移创造自然的错落效果
- **缓慢上升**: 较慢的速度配合较长的生命周期创造优雅的光柱效果

### 视觉效果
- **动态缩放**: X轴和Y轴的不同缩放策略创造真实的光柱形态
- **淡入淡出**: 平滑的透明度变化避免粒子突然出现或消失
- **光照混合**: 使用特殊的混合模式使光柱在暗处更明亮，在亮处更融合

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **简单计算**: 使用高效的线性插值和基本运算
- **自动销毁**: 粒子生命周期结束后自动失效

## 使用场景

### 神圣效果
- **神圣光线**: 从天而降的神圣光束效果
- **启示时刻**: 关键剧情中的天启或启示效果
- **传送光柱**: 传送或召唤时的光柱表现

### 环境效果
- **天窗效果**: 地牢中天窗透入的光线
- **魔法光源**: 替代普通光源的神圣照明效果
- **特殊区域**: 神圣或神秘区域的氛围营造

### UI系统
- **神圣提示**: 神圣相关操作的交互反馈
- **状态指示**: 神圣增益或特殊状态的持续视觉提醒
- **成就解锁**: 神圣类成就达成时的庆祝效果

### 特殊事件
- **Boss战**: 神圣系Boss的特殊技能效果
- **剧情演出**: 关键神圣时刻的视觉强化
- **隐藏要素**: 神圣隐藏房间或秘密发现时的提示效果

## 相关类
- **PixelParticle**: 基础粒子类，提供基础的渲染和运动功能
- **Emitter**: 粒子发射器，管理粒子池和发射逻辑
- **Random**: 随机数生成器，用于时间偏移的随机化
- **PointF**: 2D点坐标类，用于表示位置和速度向量

## 设计哲学
`ShaftParticle` 类体现了游戏特效设计的几个重要原则：

### 自然模拟
通过时间错位和动态缩放创造出逼真的光柱效果，避免机械重复感。

### 视觉层次
X轴和Y轴的不同缩放策略创造丰富的视觉层次，使光柱看起来更加立体。

### 性能平衡
在保证视觉质量的同时，通过简单的计算和对象复用确保性能稳定。

### 氛围营造
作为神圣氛围的重要组成部分，与其他视觉元素协调统一，增强游戏沉浸感.
