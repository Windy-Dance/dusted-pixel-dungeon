# TorchHalo 类

## 概述
`TorchHalo` 类是 Shattered Pixel Dungeon 游戏中的火炬光环效果类。它继承自 `Halo`，专门用于表现火把、光源或温暖魔法在角色周围的光环视觉效果。

## 功能描述
该类实现了火炬光环的核心行为，具有以下特点：
- **动态缩放**: 环光从0逐渐放大到完整尺寸（淡入）或从完整尺寸逐渐缩小到0（淡出）
- **亮度控制**: 环光亮度从20%基础亮度逐渐增加到100%或逐渐减少到0
- **光照绘制**: 使用光照混合模式增强发光效果
- **跟随定位**: 自动跟随目标角色精灵的位置
- **双阶段动画**: 支持淡入显示和淡出熄灭两种不同的动画阶段

## 核心字段

### 配置参数
- **target**: 目标角色精灵对象，用于位置跟随
- **phase**: 动画阶段值，控制光环的状态 (-1到1的范围)
  - **phase < 0**: 淡出熄灭阶段
  - **phase = 0**: 初始状态  
  - **phase < 1**: 淡入显示阶段
  - **phase = 1**: 完全显示状态

### 视觉属性  
- **半径**: 固定为20像素，提供合适的光环大小
- **颜色**: 固定为橙黄色 (0xFFDDCC)，表现火把或温暖光源的颜色
- **基础亮度**: 20% (0.2f)，提供基础的微弱光晕效果

## 构造函数

### TorchHalo(CharSprite sprite)
**功能**: 创建基础火炬光环实例

**实现细节**:
- 调用父类 `Halo` 构造函数，设置半径20、颜色0xFFDDCC、亮度0.2f
- 保存目标角色精灵引用
- 设置初始透明度为0 (`am = 0`)，确保淡入开始时不可见

## 核心方法

### update()
**功能**: 每帧更新火炬光环状态

**实现逻辑**:
1. **淡出处理** (`phase < 0`):
   - 递增 phase 值（向0靠近）
   - 超过0时调用 `killAndErase()` 销毁光环
   - 否则动态调整缩放和透明度:
     - 缩放: `scale.set((2 + phase) * radius / RADIUS)`
     - 透明度: `am = -phase * brightness`
   
2. **淡入处理** (`phase < 1`):
   - 以双倍速度递增 phase 值（更快的淡入）
   - 限制最大值为1
   - 动态调整缩放和透明度:
     - 缩放: `scale.set(phase * radius / RADIUS)`
     - 透明度: `am = phase * brightness`

3. **位置同步**: 
   - 计算角色精灵中心点位置
   - 设置光环位置到角色中心

### draw()
**功能**: 渲染火炬光环效果

**实现逻辑**:
- **光照模式**: 切换到光照混合模式 (`Blending.setLightMode()`)
- **父类绘制**: 调用父类 `draw()` 方法进行实际渲染
- **恢复正常**: 切换回正常混合模式 (`Blending.setNormalMode()`)

### putOut()
**功能**: 开始火炬光环淡出熄灭效果

**实现逻辑**:
- 设置 phase 为 -1，触发淡出动画逻辑
- 下一帧开始执行淡出处理

## 技术特点

### 动画设计
- **非对称速度**: 淡入速度是淡出速度的两倍，创造更自然的点亮/熄灭效果
- **线性插值**: 使用简单的线性插值确保动画流畅
- **双阶段控制**: 精确区分淡入和淡出的不同处理逻辑

### 视觉效果
- **颜色语义**: 橙黄色准确表现火把或温暖光源的视觉特征
- **光照混合**: 使用特殊的混合模式使光环在暗处更明亮，在亮处更融合
- **基础亮度**: 20%的基础亮度确保即使在淡入前也有微弱可见性

### 性能优化
- **简单计算**: 使用高效的数学运算避免复杂开销
- **自动销毁**: 效果结束后自动释放资源
- **可见性同步**: 只在角色可见时进行位置同步和渲染

## 使用场景

### 光源系统
- **火把效果**: 角色持有火把时的光源光环
- **照明魔法**: 各种照明或光源魔法的视觉表现
- **温暖光环**: 表现温暖或保护性魔法的效果

### 战斗系统
- **增益状态**: 光源相关增益状态的持续视觉反馈
- **技能效果**: 光源系技能激活时的视觉表现
- **Boss技能**: Boss光源技能的光环效果

### UI系统
- **状态指示**: 光源状态的持续视觉提醒
- **技能冷却**: 光源技能冷却时的视觉表现
- **环境提示**: 光照相关环境变化的视觉反馈

### 特殊事件
- **剧情演出**: 关键光源时刻的视觉强化
- **成就解锁**: 光源相关成就达成时的庆祝效果
- **隐藏要素**: 光源相关的谜题或机关提示效果

## 相关类
- **Halo**: 光环基类，提供基础的光环渲染功能
- **CharSprite**: 角色精灵类，作为光环的目标对象
- **Blending**: OpenGL混合模式管理器，控制渲染混合方式
- **Game**: 游戏主类，提供时间流逝信息 (`Game.elapsed`)

## 设计哲学
`TorchHalo` 类体现了游戏特效设计的几个重要原则：

### 物理隐喻
通过淡入淡出的动画模拟真实火把点燃和熄灭的过程。

### 视觉层次
光照混合模式确保光环在各种背景下都能提供合适的视觉效果。

### 性能平衡
在保证视觉质量的同时，通过简单的计算和自动管理确保性能稳定。

### 用户体验
非对称的动画速度（快速点亮，缓慢熄灭）符合玩家对光源的直观认知。

## 扩展建议

### 视觉变体
- **多色支持**: 支持不同类型的光源颜色（蓝色冷光、红色警告等）
- **脉冲效果**: 增加亮度的周期性变化模拟火焰跳动
- **纹理光环**: 支持带纹理的光环效果而非纯色

### 行为增强
- **强度控制**: 根据光源强度动态调整光环大小和亮度
- **声音同步**: 与火把音效的精确同步
- **环境交互**: 光环颜色随周围环境光线强度变化

### 用户定制
- **亮度调节**: 允许玩家调节光环的基础亮度
- **动画速度**: 支持自定义淡入淡出速度
- **大小控制**: 允许调整光环半径
