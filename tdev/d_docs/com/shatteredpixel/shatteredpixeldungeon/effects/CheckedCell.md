# CheckedCell 类

## 概述
`CheckedCell` 类是 Shattered Pixel Dungeon 游戏中用于显示格子检查效果的视觉组件类。它继承自 `Image`，在指定地图格子位置显示一个淡紫色的半透明方块，并带有淡出动画效果。

## 功能描述
该类主要用于可视化显示角色或技能"检查"过的地图格子，常见于搜索、探测或范围指示等场景。它提供了两种构造方式：立即显示和延迟显示（基于距离的渐进效果）。

## 视觉特性
- **颜色**: 淡紫色 (#55AAFF)，具有较高的可见性但不会过于刺眼
- **透明度**: 初始透明度为 0.8（80% 不透明）
- **尺寸**: 与地图格子大小相同 (`DungeonTilemap.SIZE`)
- **动画**: 线性淡出并同时缩小，直到完全消失

## 核心字段

### 动画控制
- **alpha**: 当前透明度值，同时也是动画进度控制器
- **delay**: 延迟时间（秒），用于基于距离的渐进显示效果

## 构造函数

### CheckedCell(int pos)
**功能**: 创建立即显示的检查效果

**参数说明**:
- `pos`: 地图格子索引

**实现细节**:
- 使用 `TextureCache.createSolid(0xFF55AAFF)` 创建纯色纹理
  - 颜色值解析：`0xFF55AAFF`
  - Alpha: FF (255, 完全不透明纹理)
  - Red: 55 (85)
  - Green: AA (170)  
  - Blue: FF (255)
- 设置原点为中心点 (`origin.set(0.5f)`)
- 计算世界坐标并偏移到格子中心
- 初始透明度设为 0.8

### CheckedCell(int pos, int visSource)
**功能**: 创建基于距离延迟显示的检查效果

**参数说明**:
- `pos`: 目标格子索引
- `visSource`: 视觉源点格子索引（用于计算距离）

**实现细节**:
- 调用第一个构造函数进行基础初始化
- 计算延迟时间：`trueDistance(pos, visSource) - 1`
- 应用非线性加速：`Math.pow(delay, 0.67f) / 10f`
- 如果有延迟，则初始透明度设为 0（隐藏状态）

**距离计算说明**:
- 使用 `Dungeon.level.trueDistance()` 计算真实路径距离（考虑障碍物）
- 减去 1 是因为相邻格子应该几乎没有延迟
- 幂函数 `0.67f` (约 2/3) 提供平滑的加速效果

## 核心方法

### update()
**功能**: 每帧更新动画状态

**实现逻辑**:
1. **延迟阶段**: 如果还有延迟时间剩余
   - 保持透明度为 0（不可见）
   - 继续倒计时
2. **淡出阶段**: 如果延迟结束但 alpha > 0
   - 线性减少透明度 (`alpha -= Game.elapsed`)
   - 同时按比例缩小尺寸 (`scale.set(SIZE * alpha)`)
   - 实现"收缩淡出"效果
3. **销毁阶段**: 如果 alpha ≤ 0
   - 调用 `killAndErase()` 销毁对象

## 技术特点
- **智能延迟**: 基于实际路径距离而非直线距离，更符合游戏逻辑
- **非线性加速**: 使用幂函数使远距离效果更快出现，避免等待过久
- **内存优化**: 自动清理机制防止内存泄漏
- **视觉一致性**: 尺寸和位置精确匹配地图格子系统

## 使用场景
该类主要用于以下游戏功能：
- **搜索效果**: 显示角色搜索过的区域
- **探测范围**: 魔法探测或感知技能的影响范围
- **路径指示**: 显示AI或技能选择的路径格子
- **范围预览**: 技能施放前的范围预览效果
- **发现反馈**: 显示新发现的物品或陷阱位置

## 性能优化
- **纹理缓存**: 使用 `TextureCache.createSolid()` 复用纯色纹理
- **自动生命周期**: 无需手动管理对象销毁
- **精确计时**: 基于游戏帧率的时间控制确保跨平台一致性

## 相关类
- **DungeonTilemap**: 提供坐标转换和格子尺寸信息
- **TextureCache**: 纹理缓存系统，优化内存使用
- **Dungeon.level**: 地图数据，提供距离计算功能
- **Image**: 图像基类，处理渲染和变换

## 扩展性
通过修改构造函数中的颜色值或动画参数，可以轻松创建不同视觉效果的格子指示器：
- 改变颜色值可创建不同用途的指示效果（如红色表示危险，绿色表示安全）
- 调整 alpha 衰减速率可控制动画持续时间
- 修改缩放逻辑可创建不同的消失动画（如旋转、闪烁等）
