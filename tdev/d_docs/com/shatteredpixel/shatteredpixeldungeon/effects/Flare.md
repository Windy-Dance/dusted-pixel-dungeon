# Flare 类

## 概述
`Flare` 类是 Shattered Pixel Dungeon 游戏中用于创建放射状光芒效果的高级视觉组件类。它继承自 `Visual`，通过 OpenGL 直接绘制多个三角形射线来实现动态的光芒绽放和旋转效果。

## 功能描述
该类实现了基于三角剖分的放射状光芒系统，支持动态的数量、半径、颜色和动画效果。常用于表现魔法爆发、神圣光环、能量释放等需要强烈视觉冲击的效果。

## 技术原理
- **射线生成**: 将圆形分解为多个三角形射线进行渲染
- **OpenGL 直接绘制**: 使用底层 OpenGL API 进行高效渲染
- **渐变纹理**: 使用预定义的颜色渐变创建平滑的光线效果
- **旋转动画**: 内置角速度系统实现自动旋转

## 核心字段

### 时间控制
- **duration**: 动画总持续时间
- **lifespan**: 剩余生命周期时间

### 视觉参数  
- **lightMode**: 是否使用光照混合模式
- **nRays**: 射线数量（决定光芒密度）
- **angle/angularSpeed**: 当前角度和角速度（实现旋转效果）

### 图形数据
- **texture**: 渐变纹理（白色到透明的径向渐变）
- **vertices**: 顶点缓冲区（FloatBuffer）
- **indices**: 索引缓冲区（ShortBuffer）

## 构造函数

### Flare(int nRays, float radius)
**功能**: 创建光芒效果实例

**参数说明**:
- `nRays`: 射线数量，越多密度越高但性能开销越大
- `radius`: 光芒半径（像素）

**实现细节**:
1. **纹理创建**: 使用 `TextureCache.createGradient()` 创建5级渐变：
   - `0xFFFFFFFF` (完全不透明白色)
   - `0xBBFFFFFF` (75% 不透明白色)  
   - `0x88FFFFFF` (50% 不透明白色)
   - `0x00FFFF` (完全透明青色，实际可能有误)
   - `0x00FFFFFF` (完全透明白色)
2. **初始旋转**: 设置初始角度45度，角速度180度/秒
3. **缓冲区分配**: 
   - 顶点缓冲区: `(nRays * 2 + 1) * 4 * sizeof(float)` 字节
   - 索引缓冲区: `nRays * 3 * sizeof(short)` 字节
4. **几何生成**: 
   - 中心顶点设为圆心 (0, 0)
   - 每个射线由两个边缘顶点定义
   - 边缘顶点均匀分布在圆周上

## 核心方法

### color(int color, boolean lightMode)
**功能**: 设置光芒颜色和混合模式

**实现细节**:
- 调用 `hardlight(color)` 应用颜色
- 设置 `lightMode` 标志位
- 支持链式调用

### show(Visual visual, float duration)
**功能**: 在指定视觉对象中心显示光芒动画

**特殊处理**:
- 对 `CharSprite` 使用 `destinationCenter()` 获取目标中心
- 对其他对象使用 `center()` 获取中心点
- 自动添加到父容器后部（背景层）
- 如果有持续时间，初始缩放设为0（淡入效果）

### show(Group parent, PointF pos, float duration)  
**功能**: 在指定位置和容器中显示光芒动画

**实现细节**:
- 设置绝对位置
- 添加到指定父容器
- 初始缩放设为0（如果有持续时间）

### update()
**功能**: 每帧更新动画状态

**动画逻辑**:
- 计算进度比例: `p = 1 - lifespan/duration`
- 应用非线性缩放: 
  - 前25%: `p * 4` (快速淡入)
  - 后75%: `(1 - p) * 1.333f` (缓慢淡出)
- 同时设置缩放和透明度
- 生命周期结束后自动销毁

### draw()
**功能**: 渲染光芒效果

**渲染流程**:
1. **混合模式**: 根据 `lightMode` 设置 OpenGL 混合模式
2. **调用绘制**: 调用私有 `drawRays()` 方法
3. **模式恢复**: 如果使用了光照模式，恢复为正常混合

### drawRays()
**功能**: 实际的OpenGL绘制调用

**实现细节**:
- 绑定渐变纹理
- 设置着色器参数（模型矩阵、光照等）
- 调用 `drawElements()` 绘制所有三角形射线

## 视觉特性
- **渐变效果**: 从中心到边缘的平滑颜色过渡
- **旋转动画**: 自动旋转增加动态感
- **脉动效果**: 非线性的淡入淡出创造呼吸感
- **高可定制**: 支持任意颜色、射线数量和半径

## 使用场景
该类主要用于以下游戏效果：
- **魔法爆发**: 强力法术释放时的视觉冲击
- **神圣效果**: 祝福、净化等正面状态的光环
- **能量场**: BOSS技能或环境能量场的视觉表现
- **技能充能**: 充能完成时的爆发效果
- **UI强调**: 重要界面元素的高亮效果

## 性能考虑
- **射线数量**: 平衡视觉效果和性能，通常8-16条射线足够
- **缓冲区管理**: 使用直接字节缓冲区提高OpenGL性能
- **自动清理**: 动画完成后自动销毁，避免内存泄漏
- **智能更新**: 只在必要时重新计算几何数据

## 相关类和技术
- **Visual**: 视觉组件基类，提供基础变换和渲染功能
- **SmartTexture/TextureCache**: 纹理管理系统
- **OpenGL 缓冲区**: FloatBuffer 和 ShortBuffer 用于 GPU 数据传输
- **NoosaScript**: 游戏引擎的 OpenGL 封装层
- **Blending**: 混合模式管理类

## 扩展应用
通过调整参数可以创建多种变体效果：
- **太阳效果**: 大量射线 + 黄色渐变
- **冰霜效果**: 蓝色渐变 + 较慢旋转
- **毒性效果**: 绿色渐变 + 脉动动画
- **复合效果**: 结合多个 Flare 实例创建层次感
