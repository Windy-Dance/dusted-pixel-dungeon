# FlowParticle 类

## 概述
`FlowParticle` 类是 Shattered Pixel Dungeon 游戏中的水流/液体粒子效果类。它继承自 `PixelParticle`，专门用于表现液体流动、水流或类似流体的视觉效果。

## 功能描述
该类实现了流体粒子的核心行为，具有以下特点：
- **流体运动**: 应用向下的加速度模拟液体流动特性
- **旋转动画**: 随机角速度使粒子产生自然的旋转效果
- **脉动透明度**: 透明度先增加后减少，创造流动的脉动感
- **收缩尺寸**: 粒子大小随时间逐渐缩小
- **工厂模式**: 提供标准的粒子工厂用于高效创建和复用

## 内部类 Flow
- **持续发射器**: 在指定格子位置创建持续的流体效果
- **视野优化**: 只在英雄视野内时才更新和渲染
- **自动定位**: 自动将发射器定位到格子底部边缘

## 核心字段

### 工厂实例
- **FACTORY**: 静态工厂实例，负责粒子的创建和回收
  - `emit()`: 从发射器回收并重置粒子

### 视觉属性  
- **lifespan**: 生命周期固定为0.6秒
- **acc**: 加速度设置为 (0, 32)，模拟向下的流体重力
- **angularSpeed**: 随机角速度在-360到+360度/秒范围内

## 构造函数

### FlowParticle()
**功能**: 创建基础流体粒子实例

**实现细节**:
- 调用父类构造函数初始化基础粒子
- 配置生命周期为0.6秒
- 设置向下的加速度 (+32) 模拟液体流动
- 设置随机角速度创造自然的旋转效果

## 核心方法

### reset(float x, float y)
**功能**: 重置并配置粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **生命周期**: 设置完整生命周期 (0.6秒)
3. **位置设置**: 设置粒子的初始坐标
4. **初始状态**: 
   - 透明度设置为0 (`am = 0`)
   - 尺寸设置为0 (`size(0)`)
   - 速度设置为零 (`speed.set(0)`)

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的运动和旋转逻辑
- **透明度控制**: 
  - 前半段 (p < 0.5f): `am = p * 0.6f` (线性增加)
  - 后半段: `am = (1 - p) * 0.6f` (线性减少)
  - 最大透明度为0.6，创造柔和的流动效果
- **尺寸控制**: 
  - 公式: `size((1 - p) * 4)`
  - 粒子从4像素逐渐缩小到0像素

## 内部类 Flow

### Flow(int pos)
**功能**: 创建持续流体效果发射器

**实现逻辑**:
1. **位置计算**: 使用 `DungeonTilemap.tileToWorld()` 转换格子坐标到世界坐标
2. **精确定位**: 定位到格子底部边缘 (`y + SIZE - 1`)
3. **发射配置**: 设置发射区域为整个格子宽度，高度为0（底部线）
4. **持续发射**: 以0.05秒间隔持续发射流体粒子

### update()
**功能**: 更新持续流体效果

**实现逻辑**:
- **视野检查**: 只有当格子在英雄视野内时才更新 (`Dungeon.level.heroFOV[pos]`)
- **性能优化**: 视野外自动暂停更新，节省性能开销
- **调用父类**: 在视野内时调用父类更新逻辑

## 技术特点

### 流体模拟
- **重力影响**: 向下的加速度使粒子自然向下流动
- **旋转效果**: 随机角速度增加视觉动态性和真实感
- **脉动透明度**: 创造液体流动的自然节奏感

### 视觉设计
- **柔和透明度**: 最大0.6的透明度避免过于突兀
- **渐进尺寸**: 从大到小的尺寸变化模拟液体扩散
- **底部发射**: 从格子底部发射更符合真实的水流源头

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **视野裁剪**: 只在必要时更新，大幅减少性能开销
- **简单计算**: 使用高效的线性插值和基本运算

## 使用场景

### 环境效果
- **水流效果**: 地下水流、泉水、瀑布等
- **液体泄漏**: 药水瓶破碎后的液体扩散
- **潮湿地面**: 湿润区域的持续水滴效果
- **雨水效果**: 雨水落到地面的溅射效果

### 战斗系统
- **水系魔法**: 水流冲击、水柱喷射等技能效果
- **液体攻击**: 酸液、毒液等液体攻击的视觉表现
- **环境互动**: 角色踩入水中的涟漪和水流效果

### UI系统
- **液体提示**: 可交互液体的视觉提示
- **状态指示**: 湿润状态的持续视觉反馈
- **技能预览**: 水系技能的预览效果

### 特殊事件
- **机关触发**: 水闸、喷泉等机关激活效果
- **剧情演出**: 关键剧情中的水流场景
- **隐藏要素**: 隐藏通道或秘密房间的水流提示

## 相关类
- **PixelParticle**: 基础粒子类，提供基础的渲染、运动和旋转功能
- **Emitter**: 粒子发射器基类，提供持续发射功能
- **DungeonTilemap**: 地牢瓦片地图，提供坐标转换功能
- **Random**: 随机数生成器，用于角速度的随机化
- **PointF**: 2D点坐标类，用于表示位置和向量

## 设计哲学
`FlowParticle` 类体现了游戏特效设计的几个重要原则：

### 自然模拟
通过重力、旋转和透明度的精心组合，创造出逼真的流体效果。

### 性能意识
通过视野裁剪和对象复用，在保证视觉质量的同时确保性能稳定。

### 实用导向
内部Flow类提供了开箱即用的持续流体效果，简化了上层代码的使用。

### 视觉和谐
柔和的透明度和自然的动画曲线与游戏整体美术风格保持一致。

## 扩展建议

### 视觉变体
- **不同液体**: 根据液体类型调整颜色和透明度（水=蓝色，酸液=绿色等）
- **粘度效果**: 不同粘度的液体使用不同的加速度和生命周期
- **温度表现**: 热水使用上升加速度，冷水使用下降加速度

### 行为增强
- **碰撞检测**: 粒子与墙壁碰撞后的反弹或沿壁流动效果
- **声音同步**: 与水流音效的精确同步
- **蒸发效果**: 粒子到达终点后的蒸发或消失效果

### 环境交互
- **材质影响**: 不同地面材质对水流效果的影响
- **坡度适应**: 根据地形坡度调整水流方向
- **容量限制**: 根据格子容量限制水流强度
