# CorrosionParticle 类

## 概述
`CorrosionParticle` 类是 Shattered Pixel Dungeon 游戏中的腐蚀粒子效果类。它继承自 `PixelParticle`，专门用于表现腐蚀、酸液或化学伤害的视觉效果，具有独特的颜色渐变和尺寸变化特性。

## 功能描述
该类实现了腐蚀粒子的两种不同行为模式，具有以下核心特性：
- **颜色渐变**: 粒子颜色从灰白色 (0xAAAAAA) 逐渐变为橙黄色 (0xFF8800)
- **尺寸变化**: 粒子大小从1像素逐渐增大到4像素
- **双重模式**: 支持导弹轨迹和飞溅扩散两种效果
- **重力效果**: 应用向下的加速度模拟液体下落
- **工厂模式**: 提供两种不同的工厂实例适应不同使用场景

## 核心字段

### 工厂实例
- **MISSILE**: 导弹工厂实例，用于腐蚀性投射物的轨迹效果
  - `emit()`: 调用 `resetMissile()` 方法创建缓慢移动的粒子
  - `lightMode()`: 返回 false，使用正常混合模式
  
- **SPLASH**: 飞溅工厂实例，用于腐蚀液体飞溅效果  
  - `emit()`: 调用 `resetSplash()` 方法创建快速扩散的粒子
  - `lightMode()`: 返回 false，使用正常混合模式

### 视觉属性
- **lifespan**: 生命周期固定为0.6秒
- **acc**: 加速度设置为 (0, +30)，模拟向下的重力效果

## 构造函数

### CorrosionParticle()
**功能**: 创建基础腐蚀粒子实例

**实现细节**:
- 调用父类构造函数初始化基础粒子
- 配置生命周期为0.6秒
- 设置向下的加速度 (+30) 模拟液体重力

## 核心方法

### resetMissile(float x, float y)
**功能**: 重置并配置导弹轨迹粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子使其可见
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.6秒)
4. **速度配置**: 
   - 随机角度: 在下半圆范围内 (-π到0) 随机选择（确保向下运动）
   - 随机速度: 在0-6像素/秒范围内随机选择（较慢的速度）

### resetSplash(float x, float y)
**功能**: 重置并配置飞溅扩散粒子的初始状态

**实现逻辑**:
1. **激活粒子**: 调用 `revive()` 激活粒子
2. **位置设置**: 设置粒子的初始坐标
3. **生命周期**: 设置完整生命周期 (0.6秒)
4. **速度配置**: 
   - 随机角度: 在上半圆范围内 (0到π) 随机选择
   - 随机速度: 在10-20像素/秒范围内随机选择（较快的速度）

### update()
**功能**: 每帧更新粒子状态

**实现逻辑**:
- 调用父类的 `update()` 处理基础的运动逻辑
- **尺寸控制**: 
  - 公式: `size(4 - am * 3)`
  - 粒子从1像素逐渐增大到4像素
- **颜色插值**: 
  - 使用 `ColorMath.interpolate()` 在两种颜色间渐变
  - 起始颜色: 橙黄色 (0xFF8800)
  - 结束颜色: 灰白色 (0xAAAAAA)
  - 插值因子: `am = left / lifespan`

## 技术特点

### 视觉渐变
- **反直觉设计**: 与大多数粒子相反，腐蚀粒子在消失前变得更大更亮
- **颜色语义**: 从炽热的橙黄到冷却的灰白，表现腐蚀过程的化学反应
- **尺寸增强**: 增大的尺寸强化了腐蚀效果的视觉冲击力

### 双重行为模式
- **导弹模式**: 缓慢、受控的运动适合表现持续的腐蚀轨迹
- **飞溅模式**: 快速、扩散的运动适合表现爆炸性的腐蚀溅射

### 物理模拟
- **重力影响**: 向下的加速度使粒子轨迹更加真实
- **速度差异**: 不同模式使用不同的速度范围以匹配视觉需求
- **角度限制**: 导弹模式限制在下半圆，确保效果的合理性

### 性能优化
- **对象池**: 通过 `emitter.recycle()` 复用粒子对象
- **高效插值**: 使用预计算的颜色插值避免实时计算开销
- **自动销毁**: 粒子生命周期结束后自动失效

- **物品效果**: 腐蚀性装备或消耗品的使用效果

## 设计哲学
`CorrosionParticle` 类体现了游戏特效设计的几个重要原则：

### 过程可视化
通过颜色和尺寸的变化直观地表现腐蚀这一化学过程，而不仅仅是静态的视觉效果。

### 行为差异化
针对不同的游戏场景提供专门优化的行为模式，确保每种使用情况都有最佳表现。

### 视觉记忆
独特的反常规设计（增大而非缩小）使腐蚀效果在游戏中更容易被识别和记忆。

### 技术整合
巧妙地结合了颜色插值、物理模拟和粒子系统，创造出丰富的视觉层次。
