# BadgeBanner 类

## 概述
`BadgeBanner` 类是 Shattered Pixel Dungeon 游戏中徽章横幅的动画显示类，用于在玩家获得成就徽章时展示动态的徽章弹出效果。

## 功能描述
该类继承自 `Image`，实现了一个完整的徽章显示动画系统，包括淡入、静态显示和淡出三个阶段。当玩家获得新徽章时，该类会创建一个带有音效和高亮效果的动态徽章展示。

## 状态机设计
`BadgeBanner` 使用内部枚举 `State` 实现状态机：

- **FADE_IN**: 淡入状态，徽章从透明逐渐显现并缩小到正常尺寸
- **STATIC**: 静态显示状态，徽章完全可见并保持一段时间  
- **FADE_OUT**: 淡出状态，徽章逐渐透明直至消失

## 核心常量

### 显示参数
- **DEFAULT_SCALE**: 默认缩放比例 (3.0)
- **SIZE**: 徽章纹理的标准尺寸 (16x16 像素)

### 时间控制
- **FADE_IN_TIME**: 淡入持续时间 (0.25 秒)
- **STATIC_TIME**: 静态显示时间 (1.0 秒)  
- **FADE_OUT_TIME**: 淡出持续时间 (1.75 秒)

## 主要字段
- **state**: 当前动画状态
- **index**: 徽章在纹理图集中的索引位置
- **time**: 当前状态剩余时间
- **atlas**: 纹理胶片，用于管理徽章纹理帧
- **showing**: 静态列表，跟踪当前正在显示的所有徽章横幅

## 核心方法

### 构造函数 BadgeBanner(int index)
**功能**: 创建新的徽章横幅实例

**实现细节**:
- 加载 `Assets.Interfaces.BADGES` 纹理资源
- 初始化纹理胶片（如果尚未创建）
- 调用 `setup()` 方法进行初始配置

### setup(int index)
**功能**: 配置徽章显示参数

**实现细节**:
- 设置纹理帧为指定索引的徽章
- 设置原点为中心点以便缩放动画
- 初始化透明度为 0，缩放为 2 倍默认值
- 设置初始状态为 FADE_IN
- 播放徽章获得音效 (`Assets.Sounds.BADGE`)

### update()
**功能**: 每帧更新动画状态

**实现细节**:
- 根据当前状态计算透明度和缩放
- 在状态切换时执行相应操作：
  - FADE_IN → STATIC: 设置正常缩放和完全不透明，触发高亮效果
  - STATIC → FADE_OUT: 开始淡出动画  
  - FADE_OUT → 结束: 调用 `killAndErase()` 移除对象

### highlight(Image image, int index)
**功能**: 在徽章上添加闪光高亮效果

**实现细节**:
- 使用缓存机制避免重复计算高亮点位置
- 分析纹理像素找到合适的高亮点（避开背景色）
- 创建 `Speck.DISCOVER` 特效在指定位置显示闪光
- 特殊徽章（如连击大师）使用硬编码的高亮点位置

### show(int image)
**功能**: 静态工厂方法，创建并显示徽章横幅

**返回值**: 新创建的 `BadgeBanner` 实例

### isShowingBadges()
**功能**: 检查是否有徽章正在显示

**返回值**: 布尔值，true 表示有徽章正在显示

### image(int index)
**功能**: 创建静态徽章图像（无动画）

**用途**: 用于不需要动画效果的场景，如徽章收集界面

## 技术特点
- **智能缓存**: 高亮点位置计算结果被缓存，避免重复分析纹理
- **内存管理**: 通过 `kill()` 和 `destroy()` 方法正确清理显示列表
- **音效集成**: 自动播放徽章获得音效，增强用户体验
- **动态效果**: 结合缩放、透明度和粒子特效，创造丰富的视觉体验
- **线程安全**: 静态显示列表确保多个徽章可以同时显示

## 使用场景
该类主要在以下场景中使用：
- 玩家首次完成特定成就时
- 游戏结算界面展示新获得的徽章
- 徽章收集系统的动态预览
- 任何需要突出显示徽章获得的场合

## 相关类
- **Badges**: 徽章管理类，定义所有徽章类型
- **Speck**: 粒子特效类，用于高亮闪光效果
- **TextureFilm**: 纹理管理类，用于处理纹理图集
- **Sample**: 音频播放类，用于播放徽章音效
