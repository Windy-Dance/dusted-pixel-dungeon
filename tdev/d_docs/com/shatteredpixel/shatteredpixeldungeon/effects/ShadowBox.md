# ShadowBox 类

## 概述
`ShadowBox` 类是 Shattered Pixel Dungeon 游戏中的阴影框UI组件类。它继承自 `NinePatch`，专门用于在界面元素周围添加阴影效果，提升UI的立体感和视觉层次。

## 功能描述
该类实现了UI阴影效果的核心功能，具有以下特点：
- **九宫格缩放**: 使用九宫格技术确保阴影在任意尺寸下都能正确显示
- **固定边距**: 阴影边距固定为16像素，提供一致的视觉效果
- **线性过滤**: 启用线性纹理过滤确保阴影边缘平滑
- **便捷定位**: 提供专用方法简化阴影框的定位和尺寸设置
- **性能优化**: 仅在首次创建时设置纹理过滤，避免重复操作

## 核心字段

### 配置参数
- **SIZE**: 阴影边距常量，固定为16像素
- **texture**: 阴影纹理资源，来自 `Assets.Interfaces.SHADOW`

## 构造函数

### ShadowBox()
**功能**: 创建基础阴影框实例

**实现细节**:
- 调用父类 `NinePatch` 构造函数，加载阴影纹理
- **纹理过滤**: 仅在首次创建时 (`texture.id == -1`) 设置线性过滤
  - 使用 `SmartTexture.LINEAR` 确保阴影边缘平滑
  - 避免重复设置提高性能
- **初始缩放**: 设置默认缩放为 (16, 16) 像素

## 核心方法

### size(float width, float height)
**功能**: 设置阴影框的实际内容尺寸

**实现逻辑**:
- **尺寸转换**: 将实际内容尺寸转换为九宫格缩放比例
- 公式: `super.size(width / SIZE, height / SIZE)`
- 确保阴影边距保持16像素不变

### boxRect(float x, float y, float width, float height)
**功能**: 便捷方法，同时设置阴影框位置和覆盖范围

**实现逻辑**:
1. **位置调整**: 
   - X坐标向左偏移16像素 (`x - SIZE`)
   - Y坐标向上偏移16像素 (`y - SIZE`)
2. **尺寸扩展**: 
   - 宽度增加32像素 (`width + SIZE * 2`)
   - 高度增加32像素 (`height + SIZE * 2`)
3. **调用size**: 自动调用 `size()` 方法设置最终尺寸

## 技术特点

### 九宫格技术
- **边缘保护**: 角落像素不会被拉伸，保持清晰
- **中心拉伸**: 边缘和中心区域可以任意拉伸适应不同尺寸
- **一致性**: 在各种尺寸下都能保持相同的视觉效果

### 性能优化
- **纹理复用**: 利用NinePatch的纹理复用机制
- **过滤优化**: 仅在必要时设置纹理过滤
- **简单计算**: 尺寸计算使用简单的加减乘除运算

### 易用性设计
- **直观接口**: `boxRect()` 方法提供符合直觉的使用方式
- **自动计算**: 自动处理阴影边距的偏移和扩展
- **标准集成**: 与其他UI组件无缝集成

## 使用场景

### UI系统
- **窗口阴影**: 对话框、菜单等窗口组件的阴影效果
- **按钮阴影**: 重要按钮或交互元素的立体效果
- **面板背景**: 各种信息面板的背景阴影
- **HUD元素**: 头顶显示、状态栏等HUD元素的视觉增强

### 界面设计
- **层次分离**: 通过阴影效果区分不同UI层级
- **焦点引导**: 重要UI元素通过阴影突出显示
- **视觉统一**: 为整个游戏UI提供一致的阴影风格

### 特殊效果
- **动态阴影**: 配合动画创建动态的阴影变化效果
- **交互反馈**: 按钮悬停或点击时的阴影变化反馈
- **深度表现**: 通过阴影强度表现UI元素的深度层次

## 相关类
- **NinePatch**: 九宫格图像基类，提供核心的可拉伸图像功能
- **Assets.Interfaces.SHADOW**: 阴影纹理资源
- **SmartTexture**: 智能纹理类，提供纹理过滤功能
- **PointF**: 2D点坐标类，用于表示位置和尺寸

## 设计哲学
`ShadowBox` 类体现了UI组件设计的几个重要原则：

### 简洁高效
专注于单一目的（添加阴影效果），避免不必要的复杂性。

### 一致性保证
通过固定的16像素边距确保整个游戏UI的视觉一致性。

### 开发友好
提供直观的API接口，降低上层代码的使用复杂度。

### 性能意识
通过智能的纹理管理和简单的计算确保高性能。
