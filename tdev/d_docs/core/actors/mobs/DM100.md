## 2. 类职责说明
DM100 是一种机械敌人，可以发射电击攻击远程目标。它是监狱层的基础机械单位，具有远程攻击能力。DM100 掉落卷轴。

## 注意事项
1. **电击属性**: 属于 ELECTRIC 类型
2. **无机属性**: 属于 INORGANIC 类型
3. **远程攻击**: 可以远程电击
4. **卷轴掉落**: 25%概率掉落卷轴
5. **低伤害**: 近战伤害较低

## 最佳实践
1. 保持移动避开电击
2. 近战更有效
3. 注意电击的视觉提示
4. 使用电击抗性装备
