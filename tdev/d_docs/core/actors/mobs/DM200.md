## 2. 类职责说明
DM200 是一种机械敌人，可以喷射毒气攻击远处的目标。它是大型无机单位，能从远处释放有毒气体。DM200 掉落武器或护甲，掉落概率随已掉落数量递减。

## 注意事项
1. **无机属性**: 属于 INORGANIC 类型
2. **大型单位**: 属于 LARGE 属性
3. **毒气喷射**: 30回合冷却
4. **装备掉落**: 掉落 T4 武器或护甲
5. **毒气免疫**: 免疫毒气
