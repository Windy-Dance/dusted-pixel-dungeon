## 类职责

DM300 是游戏中第三关（洞穴层）的Boss机器人，是一个具有多阶段战斗机制的强大敌人。它的核心职责包括：

1. **Boss战斗管理**：作为关卡Boss，拥有高额生命值和多种攻击技能
2. **充能机制**：生命值降低到阈值时进入充能状态，激活电塔
3. **技能系统**：毒气喷射、岩石坠落两种主要攻击技能
4. **环境互动**：踩踏能量地砖获得护盾，充能时可破坏墙壁
5. **阶段转换**：与电塔(Pylon)联动的多阶段战斗

**核心设计模式**：
- **状态模式**：充能/非充能状态切换
- **策略模式**：技能选择逻辑
- **观察者模式**：与电塔和关卡的联动

## 7. 方法详解

### act() - AI行为主逻辑

```java
@Override
protected boolean act() {
    // 麻痹状态下不执行AI逻辑
    if (paralysed > 0) {
        return super.act();
    }

    // 非充能状态下的AI逻辑
    if (!supercharged) {
        if (turnsSinceLastAbility >= 0) turnsSinceLastAbility++;

        // 更新视野
        if (fieldOfView == null || fieldOfView.length != Dungeon.level.length()) {
            fieldOfView = new boolean[Dungeon.level.length()];
            Dungeon.level.updateFieldOfView(this, fieldOfView);
        }

        // 检查是否能到达敌人
        boolean canReach = checkCanReachEnemy();

        if (state != HUNTING) {
            // 非追击状态下，吸引英雄位置
            if (Dungeon.hero.invisible <= 0 && canReach) {
                beckon(Dungeon.hero.pos);
            }
        } else {
            // 追击状态下的技能使用逻辑
            // 1. 无法到达敌人时使用技能
            // 2. 正常冷却后使用技能
        }
    } else {
        // 充能状态：强制追击英雄
        if (!chargeAnnounced) {
            yell(Messages.get(this, "supercharged"));
            chargeAnnounced = true;
        }
        if (Dungeon.hero.invisible <= 0) {
            beckon(Dungeon.hero.pos);
            state = HUNTING;
            enemy = Dungeon.hero;
        }
    }

    return super.act();
}
```

**方法作用**：执行DM300每回合的AI行为。

**执行流程**：
1. 检查麻痹状态
2. 非充能状态：
   - 更新回合计数器
   - 检查是否可达敌人
   - 根据情况使用技能
3. 充能状态：
   - 宣告充能
   - 强制追击英雄

### ventGas(Char target) - 毒气喷射

```java
public void ventGas(Char target) {
    Dungeon.hero.interrupt();

    Ballistica trajectory = new Ballistica(pos, target.pos, Ballistica.STOP_TARGET);

    int gasMulti = Dungeon.isChallenged(Challenges.STRONGER_BOSSES) ? 2 : 1;

    // 延迟生成毒气，在目标行动前生效
    Actor.addDelayed(new Actor() {
        { actPriority = VFX_PRIO; }
        @Override
        protected boolean act() {
            int gasVented = 0;
            // 在碰撞点生成大量毒气
            GameScene.add(Blob.seed(trajectory.collisionPos, 100 * gasMulti, ToxicGas.class));
            
            // 沿轨迹生成毒气
            for (int i : trajectory.subPath(0, trajectory.dist)) {
                GameScene.add(Blob.seed(i, 20 * gasMulti, ToxicGas.class));
                gasVented += 20 * gasMulti;
            }
            
            // 如果毒气量不足，在自身周围补充
            if (gasVented < 250 * gasMulti) {
                int toVentAround = (int) Math.ceil(((250 * gasMulti) - gasVented) / 8f);
                for (int i : PathFinder.NEIGHBOURS8) {
                    GameScene.add(Blob.seed(pos + i, toVentAround, ToxicGas.class));
                }
            }
            Actor.remove(this);
            return true;
        }
    }, target.cooldown());
}
```

**方法作用**：向目标方向喷射毒气。

**机制详解**：
- 使用弹道计算毒气轨迹
- 碰撞点生成100单位毒气
- 轨迹路径每格20单位毒气
- 总量不足250时在自身周围补充
- 强Boss模式下毒气量翻倍

### dropRocks(Char target) - 岩石坠落

```java
public void dropRocks(Char target) {
    Dungeon.hero.interrupt();
    final int rockCenter;

    // 相邻时击退2格
    if (Dungeon.level.adjacent(pos, target.pos)) {
        int oppositeAdjacent = target.pos + (target.pos - pos);
        Ballistica trajectory = new Ballistica(target.pos, oppositeAdjacent, Ballistica.MAGIC_BOLT);
        WandOfBlastWave.throwChar(target, trajectory, 2, false, false, this);
        rockCenter = trajectory.path.get(Math.min(trajectory.dist, 2));
    }
    // 距离2格时击退1格
    else if (fieldOfView[target.pos] && Dungeon.level.distance(pos, target.pos) == 2) {
        int oppositeAdjacent = target.pos + (target.pos - pos);
        Ballistica trajectory = new Ballistica(target.pos, oppositeAdjacent, Ballistica.MAGIC_BOLT);
        WandOfBlastWave.throwChar(target, trajectory, 1, false, false, this);
        rockCenter = trajectory.path.get(Math.min(trajectory.dist, 1));
    }
    // 远距离无击退
    else {
        rockCenter = target.pos;
    }

    // 选择一个安全格
    int safeCell = selectSafeCell(rockCenter);

    // 生成岩石区域（7x7范围）
    ArrayList<Integer> rockCells = generateRockCells(rockCenter, safeCell);

    // 显示预警
    for (int i : rockCells) {
        sprite.parent.add(new TargetedCell(i, 0xFF0000));
    }

    // 延迟落下岩石
    Buff.append(this, FallingRockBuff.class, 
        GameMath.gate(TICK, (int) Math.ceil(target.cooldown()), 3 * TICK))
        .setRockPositions(rockCells);
}
```

**方法作用**：在目标位置降下岩石。

**机制详解**：
1. **击退效果**：相邻击退2格，距离2格击退1格
2. **安全格**：总有一个格子不会落下岩石
3. **预警系统**：红色标记显示即将落下的位置
4. **延迟机制**：根据目标行动速度调整延迟

public void loseSupercharge() {

### damage(int dmg, Object src) - 伤害处理

```java
@Override
public void damage(int dmg, Object src) {
    if (!BossHealthBar.isAssigned()) {
        notice();
    }

    int preHP = HP;
    super.damage(dmg, src);
    if (isInvulnerable(src.getClass())) {
        return;
    }

    int dmgTaken = preHP - HP;
    if (dmgTaken > 0) {
        // 更新楼层锁定时间
        LockedFloor lock = Dungeon.hero.buff(LockedFloor.class);
        if (lock != null && !isImmune(src.getClass()) && !isInvulnerable(src.getClass())) {
            if (Dungeon.isChallenged(Challenges.STRONGER_BOSSES)) {
                lock.addTime(dmgTaken / 2f);
            } else {
                lock.addTime(dmgTaken);
            }
        }
    }

    // 计算充能阈值
    int threshold;
    if (Dungeon.isChallenged(Challenges.STRONGER_BOSSES)) {
        // 强Boss：4次充能（75%, 50%, 25%, 0%）
        threshold = HT / 4 * (3 - pylonsActivated);
    } else {
        // 普通：2次充能（66%, 33%）
        threshold = HT / 3 * (2 - pylonsActivated);
    }

    // 生命值降到阈值时充能
    if (HP <= threshold && threshold > 0) {
        HP = threshold;
        supercharge();
    }
}
```

**方法作用**：处理受到的伤害，触发充能阶段。

**阈值计算**：
- 普通模式：66%和33%生命值触发充能
- 强Boss模式：75%、50%、25%生命值触发充能

### getCloser(int target) - 移动逻辑

```java
@Override
protected boolean getCloser(int target) {
    if (super.getCloser(target)) {
        return true;
    } else {
        // 充能状态下的穿墙移动
        if (!supercharged || state != HUNTING || rooted || target == pos 
            || Dungeon.level.adjacent(pos, target)) {
            return false;
        }

        // 寻找最佳移动位置
        int bestpos = pos;
        for (int i : PathFinder.NEIGHBOURS8) {
            if (Actor.findChar(pos + i) == null &&
                Dungeon.level.trueDistance(bestpos, target) > 
                Dungeon.level.trueDistance(pos + i, target)) {
                bestpos = pos + i;
            }
        }

        if (bestpos != pos) {
            Sample.INSTANCE.play(Assets.Sounds.ROCKS);

            // 破坏周围墙壁
            Rect gate = CavesBossLevel.gate;
            for (int i : PathFinder.NEIGHBOURS9) {
                if (Dungeon.level.map[pos + i] == Terrain.WALL 
                    || Dungeon.level.map[pos + i] == Terrain.WALL_DECO) {
                    Point p = Dungeon.level.cellToPoint(pos + i);
                    // 不破坏大门区域
                    if (p.y < gate.bottom && p.x >= gate.left - 2 && p.x < gate.right + 2) {
                        continue;
                    }
                    // 不破坏竞技场外的墙
                    if (!CavesBossLevel.diggableArea.inside(p)) {
                        continue;
                    }
                    Level.set(pos + i, Terrain.EMPTY_DECO);
                    GameScene.updateMap(pos + i);
                }
            }
            
            Dungeon.level.cleanWalls();
            Dungeon.observe();
            spend(Dungeon.isChallenged(Challenges.STRONGER_BOSSES) ? 2f : 3f);

            // 执行移动
            if (bestpos != pos) {
                move(bestpos);
            }
            PixelScene.shake(5, 1f);
            return true;
        }

        return false;
    }
}
```

**方法作用**：向目标移动，充能状态下可穿墙。

**穿墙限制**：
- 不能破坏大门区域
- 只能破坏竞技场内的墙壁

### move(int step, boolean travelling) - 移动效果

```java
@Override
public void move(int step, boolean travelling) {
    super.move(step, travelling);

    // 移动时震动屏幕
    if (travelling) {
        PixelScene.shake(supercharged ? 3 : 1, 0.25f);
    }

    // 踩到能量地砖时获得护盾
    if (!flying && Dungeon.level.map[pos] == Terrain.INACTIVE_TRAP && state == HUNTING) {
        // 已被充能的区域不获得护盾
        if (CavesBossLevel.PylonEnergy.volumeAt(pos, CavesBossLevel.PylonEnergy.class) > 0) {
            return;
        }

        if (Dungeon.level.heroFOV[pos]) {
            if (buff(Barrier.class) == null) {
                GLog.w(Messages.get(this, "shield"));
            }
            Sample.INSTANCE.play(Assets.Sounds.LIGHTNING);
            sprite.emitter().start(SparkParticle.STATIC, 0.05f, 20);
            sprite.showStatusWithIcon(CharSprite.POSITIVE, 
                Integer.toString(30 + (HT - HP) / 10), FloatingText.SHIELDING);
        }

        // 护盾量 = 30 + (已损失生命/10)
        Buff.affect(this, Barrier.class).setShield(30 + (HT - HP) / 10);
    }
}
```

**方法作用**：处理移动效果，踩地砖获得护盾。

**护盾计算**：基础30 + 已损失生命值的10%

```java
// 技能选择逻辑（非充能状态）
if (turnsSinceLastAbility > abilityCooldown) {
    if (lastAbility == NONE) {
        // 初始：50%概率选择任一技能
        lastAbility = Random.Int(2) == 0 ? GAS : ROCKS;
    } else if (lastAbility == GAS) {
        // 上次是毒气：75%概率使用岩石
        lastAbility = Random.Int(4) == 0 ? GAS : ROCKS;

        // 上次是岩石：75%概率使用毒气
        lastAbility = Random.Int(4) != 0 ? GAS : ROCKS;
    }
    
    // 无机物敌人强制使用岩石
    if (Char.hasProp(enemy, Property.INORGANIC)) {
        lastAbility = ROCKS;
    }
}
```

public static class FallingRockBuff extends DelayedRockFall {
    @Override
    public void affectChar(Char ch) {
        if (!(ch instanceof DM300 || ch instanceof Pylon)) {

            }
        }
    }
}

## 注意事项

### 充能机制陷阱

1. **无敌检测**：充能状态下 `isInvulnerable()` 返回 true
2. **生命值锁定**：充能时HP会被设为阈值
3. **冷却重置**：失去充能后技能冷却部分重置

### 技能使用限制

1. **无机物敌人**：无法使用毒气攻击
2. **已眩晕敌人**：不会重复使用岩石眩晕
3. **距离检测**：相邻使用技能会消耗回合

### 穿墙限制

1. **大门保护**：不会破坏大门区域
2. **竞技场边界**：只能破坏 `diggableArea` 内的墙
3. **花费时间**：穿墙消耗2-3回合

### 护盾机制

1. **触发条件**：必须在HUNTING状态踩地砖
2. **叠加规则**：护盾可叠加
3. **能量区域**：已被充能的地砖不再提供护盾

## 最佳实践

### 设计自定义Boss

1. **继承DM300**：重写关键方法实现自定义行为
2. **调整阈值**：修改充能触发点
3. **自定义技能**：扩展 `ventGas()` 和 `dropRocks()`

### 性能优化

1. **视野缓存**：`fieldOfView` 在必要时才重新计算
2. **延迟技能**：毒气使用 `Actor.addDelayed()` 避免即时伤害
3. **Bundle优化**：只存储必要的状态数据

### 调试技巧

1. **日志输出**：使用 `yell()` 和 `GLog` 追踪Boss状态
2. **状态检查**：在 `act()` 开头打印当前状态
3. **技能调试**：修改冷却常量测试技能逻辑
